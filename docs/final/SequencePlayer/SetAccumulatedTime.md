# SetAccumulatedTime

设置当前序列播放器的累积时间

目标是序列播放器库

## 图示

![](/uploads/projects/ue-bluprint/20221218-20481524.png)

## Inputs

在。执行。

序列播放器。序列播放器参考结构（通过参考）。

时间。浮点数（单精度）。

## Outputs

出：执行。

返回值。序列播放器参考结构。设置序列播放器的当前累积时间。
