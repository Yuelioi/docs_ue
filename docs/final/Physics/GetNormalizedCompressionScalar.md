# GetNormalizedCompressionScalar

返回弹簧压缩量为沿弹簧方向的归一化标量。0 意味着弹簧处于静止状态。1 意味着完全压缩

目标是物理学弹簧组件

## Inputs

目标。物理弹簧组件对象参考。

## Outputs

返回值。浮点数（单精度）。返回弹簧压缩量为沿弹簧方向的归一化标量。0 意味着弹簧处于静止状态。1 意味着完全压缩。
