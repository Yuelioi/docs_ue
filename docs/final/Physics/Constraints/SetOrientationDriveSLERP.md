# SetOrientationDriveSLERP

启用/禁用角向斜拉驱动。只有当角度驱动模式被设置为SLERP时才相关。

目标是约束实例蓝图库

## Inputs

在。执行。

访问器。限制性实例访问器结构（通过引用）。要改变的约束访问器。

启用SLERP。布尔值。表示是否应该启用SLERP驱动。只有在AngularDriveMode被设置为SLERP时才相关。  

## Outputs

出：执行。

Enables/Disables the angular orientation slerp drive. Only relevant if the AngularDriveMode is set to SLERP
