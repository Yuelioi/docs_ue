# SetAngularVelocityDriveTwistandS-

启用/禁用角速度扭转和摆动驱动。只有在角速度驱动模式被设置为扭转和摆动时才相关。

目标是约束实例蓝图库

## 图示

![]($-20221218-20271723.png)

## Inputs

在。执行。

访问器。限制性实例访问器结构（通过引用）。要改变的约束访问器。

启用扭转驱动。布尔值。指示是否应该启用扭曲轴的驱动。只有在AngularDriveMode被设置为Twist和Swing时才相关。

启用摇摆驱动。布尔值。指示是否应该启用摆动轴的驱动。只有在AngularDriveMode被设置为Twist和Swing时才相关。  

## Outputs

出：执行。

Enables/Disables angular velocity twist and swing drive. Only relevant if the AngularDriveMode is set to Twist and Swing

Target is Constraint Instance Blueprint Library

## 图示

![]($-20221218-20271723.png)

## Inputs

In: Exec.

Accessor: Constraint Instance Accessor Structure (by ref). Constraint accessor to change.

Enable Twist Drive: Boolean. Indicates whether the drive for the twist axis should be enabled. Only relevant if the AngularDriveMode is set to Twist and Swing.

Enable Swing Drive: Boolean. Indicates whether the drive for the swing axis should be enabled. Only relevant if the AngularDriveMode is set to Twist and Swing.  

## Outputs

Out: Exec.
