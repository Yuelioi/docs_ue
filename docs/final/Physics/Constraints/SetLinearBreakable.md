# SetLinearBreakable

设置线性可破的属性

目标是约束实例蓝图库

## 图示

![]($-20221218-20272185.png)

## Inputs

在。执行。

访问器。限制性实例访问器结构（通过引用）。要改变的约束访问器。

线性可破。布尔型。是否有可能用线性力破坏关节。

线性中断阈值。Float（单精度）。打破关节所需的力。  

## Outputs

出：执行。

Sets the Linear Breakable properties

Target is Constraint Instance Blueprint Library

## 图示

![]($-20221218-20272185.png)

## Inputs

In: Exec.

Accessor: Constraint Instance Accessor Structure (by ref). Constraint accessor to change.

Linear Breakable: Boolean. Whether it is possible to break the joint with linear force.

Linear Break Threshold: Float (single-precision). Force needed to break the joint.  

## Outputs

Out: Exec.
