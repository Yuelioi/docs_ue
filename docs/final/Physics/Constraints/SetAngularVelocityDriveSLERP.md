# SetAngularVelocityDriveSLERP

启用/禁用角速度斜率驱动。只有在角速度驱动模式被设置为 SLERP 时才相关。

目标是约束实例蓝图库

## 图示

![](/uploads/projects/ue-bluprint/20221218-20271607.png)

## Inputs

在。执行。

访问器。限制性实例访问器结构（通过引用）。要改变的约束访问器。

启用 SLERP。布尔值。表示是否应该启用 SLERP 驱动。只有在 AngularDriveMode 被设置为 SLERP 时才相关。

## Outputs

出：执行。

<hr>

Enables/Disables the angular velocity slerp drive. Only relevant if the AngularDriveMode is set to SLERP

Target is Constraint Instance Blueprint Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-20271607.png)

## Inputs

In: Exec.

Accessor: Constraint Instance Accessor Structure (by ref). Constraint accessor to change.

Enable SLERP: Boolean. Indicates whether the SLERP drive should be enabled. Only relevant if the AngularDriveMode is set to SLERP.

## Outputs

Out: Exec.
