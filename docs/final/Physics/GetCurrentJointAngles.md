# GetCurrentJointAngles

获取一个命名的骨骼约束的当前角度状态

目标是骨架网组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-20274682.png)

## Inputs

在。执行。

目标。骨骼网组件对象参考。

在骨子里的名字。名称。要获得约束范围的骨骼名称。

## Outputs

出：执行。

Swing 1Angle:Float（单精度）。约束的当前角度状态。

扭转角度。Float（单精度）。约束的当前角度状态。

Swing 2Angle:Float（单精度）。约束的当前角度状态。

Gets the current Angular state for a named bone constraint

Target is Skeletal Mesh Component

## 图示

![](/uploads/projects/ue-bluprint/20221218-20274682.png)

## Inputs

In: Exec.

Target: Skeletal Mesh Component Object Reference.

In Bone Name: Name. Name of bone to get constraint ranges for.

## Outputs

Out: Exec.

Swing 1Angle: Float (single-precision). current angular state of the constraint.

Twist Angle: Float (single-precision). current angular state of the constraint.

Swing 2Angle: Float (single-precision). current angular state of the constraint.
