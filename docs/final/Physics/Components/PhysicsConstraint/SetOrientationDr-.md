Enables/Disables angular orientation drive. Only relevant if the AngularDriveMode is set to Twist and Swing

Target is Physics Constraint Component

![](/uploads/projects/ue-bluprint/20221218-20213099.png)

## Inputs

In: Exec.

Target: Physics Constraint Component Object Reference.

Enable Twist Drive: Boolean. Indicates whether the drive for the twist axis should be enabled. Only relevant if the AngularDriveMode is set to Twist and Swing.

Enable Swing Drive: Boolean. Indicates whether the drive for the swing axis should be enabled. Only relevant if the AngularDriveMode is set to Twist and Swing.

## Outputs

Out: Exec.
