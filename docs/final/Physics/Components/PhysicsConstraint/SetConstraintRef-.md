# SetConstraintRef-

传入参考方向（保持参考位置）。如果该约束当前处于活动状态，这将设置其活动的局部姿势。否则，该变化将在 InitConstraint 中生效。

目标是物理约束组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-20211342.png)

## Inputs

在。执行。

目标。物理约束组件对象参考。

框架。EConstraintFrame Enum.

Pri 轴：向量（通过参考）。

第二轴：矢量（通过参考）。

## Outputs

出：执行。
