# SetLinearVelocityDrive

启用/停用线性位置驱动

目标是物理约束组件

## Inputs

在。执行。

目标。物理约束组件对象参考。

启用X驱动：布尔值。表示是否应启用X轴的驱动。

启用Y轴驱动：布尔值。表示是否应启用Y轴的驱动器。

启用Z轴驱动：布尔值。表示是否应启用Z轴的驱动。  

## Outputs

出：执行。

Enables/Disables linear position drive
