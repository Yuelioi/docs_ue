# GetDefaultValue

返回 Pin 的默认值为一个字符串。注意，这个值是基于针脚的子针脚计算的。例如，对于一个 FVector 类型的 Pin，默认值实际上是由 X、Y 和 Z SubPins 的默认值组成的。

目标是 Rig VMPin

## Outputs

返回值。字符串。返回 Pin 的默认值为一个字符串。注意，这个值是根据针脚的子针脚计算的。例如，对于一个 FVector 类型的 Pin，默认值实际上是由 X、Y 和 Z SubPins 的默认值组成的。

<hr>

Returns the default value of the Pin as a string.. Note that this value is computed based on the Pin's. SubPins - so for example for a FVector typed Pin. the default value is actually composed out of the. default values of the X, Y and Z SubPins.

Target is Rig VMPin
