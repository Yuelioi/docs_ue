Returns the enum of the data type of the Pin,. or nullptr otherwise.

Target is Rig VMPin

## 图示

![](/uploads/projects/ue-bluprint/20221218-20455540.png)

## Inputs

Target: Rig VMPin Object Reference.

## Outputs

Return Value: Enum Object Reference. Returns the enum of the data type of the Pin,. or nullptr otherwise..
