# GetLinkedAnimInstance

获取链接的实例是由这个节点主持的。如果该节点没有承载一个实例，那么 HasLinkedAnimInstance 将返回 false。

目标是链接 Anim 图库

## 图示

![](/uploads/projects/ue-bluprint/20221218-19440779.png)

## Inputs

节点。链接的 Anim 图参考结构（通过参考）。

## Outputs

返回值。Anim 实例对象参考。获取链接的实例是由这个节点主持的。如果该节点没有承载实例，那么 HasLinkedAnimInstance 将返回 false。

<hr>

Get the linked instance is hosted by this node. If the node does not host an instance then HasLinkedAnimInstance will return false

Target is Linked Anim Graph Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-19440779.png)

## Inputs

Node: Linked Anim Graph Reference Structure (by ref).

## Outputs

Return Value: Anim Instance Object Reference. Get the linked instance is hosted by this node. If the node does not host an instance then HasLinkedAnimInstance will return false.
