# FindTakes

查找所有现有的用指定石板录制的片段

目标是 "核心蓝图库"。

## 图示

![](/uploads/projects/ue-bluprint/20221218-21100743.png)

## Inputs

在。执行。

石板。绳子。要过滤的板块。

取数：整数。要过滤的取数。<=0 表示全部取数。

## Outputs

出：执行。

返回值。资产数据结构的数组。

Find all the existing takes that were recorded with the specified slate

Target is Takes Core Blueprint Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-21100743.png)

## Inputs

In: Exec.

Slate: String. The slate to filter by.

Take Number: Integer. The take number to filter by. <=0 denotes all takes.

## Outputs

Out: Exec.

Return Value: Array of Asset Data Structures.
