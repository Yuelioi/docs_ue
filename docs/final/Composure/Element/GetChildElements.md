# GetChildElements

获取当前元素的子合成元素的第一层。

目标是空的组合镜头

## 图示

![](/uploads/projects/ue-bluprint/20221218-18285935.png)

## Inputs

目标。Empty Comp Shot 对象参考。

## Outputs

返回值。空的 Comp Shot 对象引用的数组。TArray<ACompositingElement\*> 包含所有第一层子元素的数组，没有任何孙子。

<hr>

Get the first level of current element's child composure elements.

Target is Empty Comp Shot

## 图示

![](/uploads/projects/ue-bluprint/20221218-18285935.png)

## Inputs

Target: Empty Comp Shot Object Reference.

## Outputs

Return Value: Array of Empty Comp Shot Object References. TArray<ACompositingElement\*> The array containing all the first level children without any grandchildren..
