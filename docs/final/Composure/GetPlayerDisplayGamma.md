# GetPlayerDisplayGamma

返回给定玩家相机管理器的显示伽马，如果没有连接场景视口，则返回 0。

目标是 Composure 蓝图库

![](/uploads/projects/ue-bluprint/20221218-18291823.png)

## Inputs

播放器相机管理器。播放器相机管理器对象参考。

## Outputs

显示伽玛。Float（单精度）。
