Changes the active level sequence to a new level sequence.

Target is Level Sequence Playback Controller

## 图示

![](/uploads/projects/ue-bluprint/20221218-21265473.png)

## Inputs

In: Exec.

Target: Level Sequence Playback Controller Object Reference.

## Outputs

Out: Exec.

Return Value: Boolean. true if a valid LevelSequence was passed and sequencer was successfully found..
