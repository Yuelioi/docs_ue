# GetAnimationCurveNames

检索给定动画序列的各个浮动曲线的名称。

目标是动画蓝图库

## 图示

![](/uploads/projects/ue-bluprint/20221218-17511811.png)

## Inputs

动画序列。Anim Sequence 对象参考。

曲线类型。ERawCurveTrackTypes 枚举。

## Outputs

曲线名称。名称的数组。
