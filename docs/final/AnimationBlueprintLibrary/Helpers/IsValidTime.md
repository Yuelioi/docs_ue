# IsValidTime

检查给定的时间值是否在给定的动画序列的长度之内。

目标是动画蓝图库

## Inputs

动画序列基础。Anim Sequence Base对象参考。

时间。Float（单精度）。 

## Outputs

是否有效。Boolean（布尔）。

Checks whether or not the given Time Value lies within the given Animation Sequence's Length

Target is Animation Blueprint Library
