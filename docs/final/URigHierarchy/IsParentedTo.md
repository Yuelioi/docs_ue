# IsParentedTo

如果一个元素是另一个元素的父本，则返回真。

目标是钻机等级制度

## 图示

![](/uploads/projects/ue-bluprint/20221218-21194230.png)

## Inputs

目标。Rig Hierarchy 对象参考。

在儿童。里格元素键结构。子元素的键来检查父元素。

在父母。里格元素键结构。要检查的父元素的键。

## Outputs

返回值。布尔值。如果子代是父代的，则为真。

<hr>

Returns true if an element is parented to another element

Target is Rig Hierarchy

## 图示

![](/uploads/projects/ue-bluprint/20221218-21194230.png)

## Inputs

Target: Rig Hierarchy Object Reference.

In Child: Rig Element Key Structure. The key of the child element to check for a parent.

In Parent: Rig Element Key Structure. The key of the parent element to check for.

## Outputs

Return Value: Boolean. True if the child is parented to the parent.
