# GetPreviousParent

返回一个元素在 reparent 操作之前的上一个父元素。

目标是钻机等级制度

## 图示

![](/uploads/projects/ue-bluprint/20221218-21193205.png)

## Inputs

目标。Rig Hierarchy 对象参考。

在关键。Rig 元素键结构（通过参考）。要请求旧的父元素的键。

## Outputs

返回值。Rig Element Key Structure.

<hr>

Returns the previous parent of an element prior to a reparent operation

Target is Rig Hierarchy

## 图示

![](/uploads/projects/ue-bluprint/20221218-21193205.png)

## Inputs

Target: Rig Hierarchy Object Reference.

In Key: Rig Element Key Structure (by ref). The key of the element to request the old parent for.

## Outputs

Return Value: Rig Element Key Structure.
