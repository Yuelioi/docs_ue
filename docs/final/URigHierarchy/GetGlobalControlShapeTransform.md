# GetGlobalControlShapeTransform

返回一个给定控制元素的全局形状变换。

目标是钻机等级制度

## 图示

![](/uploads/projects/ue-bluprint/20221218-21191111.png)

## Inputs

目标。Rig Hierarchy 对象参考。

在关键。Rig Element Key 结构。检索变换的控件的键。

初始化。布尔值。如果为真，将使用初始变换。

## Outputs

返回值。变换。全局形状变换。

<hr>

Returns the global shape transform for a given control element.

Target is Rig Hierarchy

## 图示

![](/uploads/projects/ue-bluprint/20221218-21191111.png)

## Inputs

Target: Rig Hierarchy Object Reference.

In Key: Rig Element Key Structure. The key of the control to retrieve the transform for.

Initial: Boolean. If true the initial transform will be used.

## Outputs

Return Value: Transform. The global shape transform.
