# ListenTo

听一个特定的播放器。如果没有找到该播放器，这个组件将是无声的。

目标是 Pixel 流媒体音频组件

![](/uploads/projects/ue-bluprint/20221218-20291586.png)

## Inputs

在。执行。

目标。Pixel Streaming Audio Component 对象参考。

听的播放器。弦乐。

## Outputs

出：执行。

返回值。布尔值。听取一个特定的播放器。如果没有找到该播放器，该组件将被沉默。

Listen to a specific player. If the player is not found this component will be silent.

Target is Pixel Streaming Audio Component

## 图示

![](/uploads/projects/ue-bluprint/20221218-20291586.png)

## Inputs

In: Exec.

Target: Pixel Streaming Audio Component Object Reference.

Player to Listen To: String.

## Outputs

Out: Exec.

Return Value: Boolean. Listen to a specific player. If the player is not found this component will be silent..
