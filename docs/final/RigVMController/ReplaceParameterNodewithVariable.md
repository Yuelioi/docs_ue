# ReplaceParameterNodewithVariable

用新的数据刷新变量结点

目标是 Rig VMController

## 图示

![](/uploads/projects/ue-bluprint/20221218-20432246.png)

## Inputs

在。执行。

目标。Rig VMController 对象参考。

在节点名称中。名称（通过参考）。

在变量名称中。名称（通过参考）。

在 CPPT 类型。字符串。

在 CPPT 类型对象中。对象参考。

设置撤销重做：布尔值。

## Outputs

出：执行。

返回值。Rig VMVariable Node Object Reference。用新的数据重新刷新变量节点。

<hr>

Refreshes the variable node with the new data

Target is Rig VMController
