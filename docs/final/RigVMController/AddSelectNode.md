目标是 Rig VMController

在 CPPT 类型对象路径中。名称（通过引用）。

在位。矢量二维结构（按参考）。

Adds a select node to the graph.. Select nodes can be used to pick between multiple values based on an index.

Target is Rig VMController

## 图示

![](/uploads/projects/ue-bluprint/20221218-20421247.png)

## Inputs

In: Exec.

Target: Rig VMController Object Reference.

In CPPType: String.

In CPPType Object Path: Name (by ref).

In Position: Vector 2D Structure (by ref).

In Node Name: String.

Setup Undo Redo: Boolean.

Print Python Command: Boolean.

## Outputs

Out: Exec.

Return Value: Rig VMSelect Node Object Reference. Adds a select node to the graph.. Select nodes can be used to pick between multiple values based on an index..
