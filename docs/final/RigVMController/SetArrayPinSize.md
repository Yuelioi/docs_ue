# SetArrayPinSize

设置阵列引脚的大小。这会引起 PinArraySizeChanged 修改事件。

目标是 Rig VMController

![](/uploads/projects/ue-bluprint/20221218-20432712.png)

## Inputs

在。执行。

目标。Rig VMController 对象参考。

在 Array Pin Path 中。字符串。

在大小：整数。

在默认值中。字符串。

设置撤销重做：布尔值。

打印 Python 命令。布尔值。

## Outputs

出：执行。

返回值。布尔值。设置数组引脚的大小。这会引起 PinArraySizeChanged 修改事件。
