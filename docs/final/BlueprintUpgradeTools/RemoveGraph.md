# RemoveGraph

如果可能的话，从蓝图中删除给定的图形。

目标是蓝图编辑器库

## 图示

![]($-20221218-18114545.png)

## Inputs

在。执行。

蓝图。蓝图对象参考。图形将被移除的蓝图。

图形。Ed Graph Object Reference。要移除的图形。 

## Outputs

输出。Exec.

Removes the given graph from the blueprint if possible

Target is Blueprint Editor Library

![]($-20221218-18114545.png)

In: Exec.

Graph: Ed Graph Object Reference. The graph to remove.

Out: Exec.
