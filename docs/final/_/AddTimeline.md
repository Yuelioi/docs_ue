# AddTimeline

时间线节点允许数值随时间的推移而被关键帧化。双击来打开时间线编辑器。

## 图示

![](/uploads/projects/ue-bluprint/20221218-17453017.png)

## Inputs

播放：执行。

从开始播放。执行：执行。

停止：执行。

逆向播放。执行.

从终点倒退。执行.

设置新时间: 执行。

新时间。浮点数（单精度）。

## Outputs

更新：执行。

完成：完成。执行：执行。

方向。ETimelineDirection Enum.
