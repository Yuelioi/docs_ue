# MIDIDeviceOutputController

\*动作

## Actions

发送 MIDIChannel Aftertouch。发送 MIDI 通道触摸后的事件类型。目标是 MIDID 设备输出控制器。

发送 MIDIC 控制变化。发送 MIDI 控制变化事件类型。目标是 MIDID 设备输出控制器。

发送 MIDIEvent。发送一个事件类型的 MIDI 事件原始数据。目标是 MIDID 设备输出控制器。

发送 MIDINote Aftertouch。发送 MIDI Note Aftertouch 事件类型。目标是 MIDID 设备输出控制器。

发送 MIDINote Off。发送 MIDI 音符关闭事件类型。目标是 MIDIDice 输出控制器。

发送 MIDINote On。发送 MIDI Note On 事件类型。目标是 MIDID 设备输出控制器。

发送 MIDIPitch Bend。发送 MIDI Pitch Bend 事件类型。目标是 MIDIDice 输出控制器。

发送 MIDIProgram Change。发送 MIDI 程序变化事件类型。目标是 MIDID 设备输出控制器。
