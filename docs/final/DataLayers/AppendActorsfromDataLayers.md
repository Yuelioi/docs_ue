# AppendActorsfromDataLayers

添加所有与指定的数据层中的任何一个相关的角色。

目标是数据层编辑子系统

## 图示

![](/uploads/projects/ue-bluprint/20221218-18344745.png)

## Inputs

目标。数据层编辑子系统对象参考。

数据层。数据层对象引用的数组。找演员的 DataLayers。

## Outputs

In Out Actors: Array of Actor Object References. 将找到的演员追加到列表中。
