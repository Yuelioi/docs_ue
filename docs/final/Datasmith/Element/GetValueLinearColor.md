# GetValueLinearColor

获取此属性的线性颜色的记录值。如果该属性不是线性颜色类型，返回的值就没有意义。

目标是 Datasmith 属性捕获元素

## Inputs

在。执行。

## Outputs

输出。输出：执行。

返回值。线性颜色结构。获取该属性的记录值，作为线性颜色。如果该属性不是线性颜色类型，则返回值没有意义。
