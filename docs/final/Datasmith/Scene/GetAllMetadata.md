# GetAllMetadata

查找与给定类型的对象相关的所有元数据元素。

目标是 Datasmith 场景元素基础

## 图示

![](/uploads/projects/ue-bluprint/20221218-18395973.png)

## Inputs

在。执行。

目标。Datasmith 场景元素基本对象参考。

对象类。数据元素类参考。要过滤的对象的类别，如果有具体规定的话；否则就没有过滤功能。

## Outputs

出：执行。

出来的元数据。Datasmith 元数据元素对象参考数组。输出元数据元素的数组。

Find all metadata elements associated with objects of the given type.

Target is Datasmith Scene Element Base

## 图示

![](/uploads/projects/ue-bluprint/20221218-18395973.png)

## Inputs

In: Exec.

Target: Datasmith Scene Element Base Object Reference.

Object Class: Datasmith Object Element Class Reference. Class of the object on which to filter, if specificed; otherwise there's no filtering.

## Outputs

Out: Exec.

Out Metadatas: Array of Datasmith Meta Data Element Object References. Output array of metadata elements..
