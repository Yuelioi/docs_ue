# GetCustomActors

用 Datasmith 场景中的 CustomActor 创建一个数组，该数组位于层次结构的根层。使用 RemoveCustomActor 来修改 Datasmith 场景。

目标是 Datasmith 场景元素基础

在。执行。

目标。Datasmith 场景元素基本对象参考。

## Outputs

出：执行。

返回值。Datasmith 自定义角色元素对象的数组参考。用 Datasmith 场景中处于层次结构根层的 CustomActor 创建一个数组。使用 RemoveCustomActor 来修改 Datasmith 场景。

Target is Datasmith Scene Element Base
