# GetLightActors

用 Datasmith 场景中处于层次结构根层的 LightActor 创建一个数组。使用 RemoveLightActor 来修改 Datasmith 场景。

目标是 Datasmith 场景元素基础

## Outputs

出：执行。

返回值。Datasmith 灯光演员元素对象的数组参考。在 Datasmith 场景中创建一个数组，其中包括处于层次结构根层的 LightActor。使用 RemoveLightActor 来修改 Datasmith 场景。
