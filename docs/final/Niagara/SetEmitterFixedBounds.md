# SetEmitterFixedBounds

设置发射器实例的固定边界，这优先于所有其他的边界。这个盒子应该是在本地空间而不是世界空间。一个默认的未初始化的盒子将清除固定边界并恢复到发射器的固定/动态边界。

目标是Niagara粒子系统组件
