# GetEmitterFixedBounds

获取一个发射器实例的固定边界。如果设置了固定边界，这将返回用户设置的固定边界；如果设置了固定边界，则返回发射器的固定边界。注意：如果没有固定边界，返回的方框可能是无效的。

目标是 Niagara 粒子系统组件

## Inputs

目标。尼亚加拉粒子系统组件对象参考。

发射器名称：名称。

## Outputs

返回值。箱体结构。获取一个发射器实例的固定边界。如果设置了固定边界，这将返回用户设置的固定边界；如果设置了固定边界，则返回发射器的固定边界。注意：如果没有固定边界存在，返回的盒子可能是无效的。

Gets the fixed bounds for an emitter instance.. This will return the user set fixed bounds if set, or the emitters fixed bounds if set.. Note: The returned box may be invalid if no fixed bounds exist
