# SetDefaultShadowColorandOpacity

设置默认文本阴影的颜色和不透明度。注意：如果不透明度为零，将不会绘制阴影。

目标是富文本块

在阴影颜色和不透明度。线性色彩结构。新的阴影颜色和不透明度。
