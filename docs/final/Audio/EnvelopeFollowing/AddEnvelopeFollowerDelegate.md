# AddEnvelopeFollowerDelegate

当包络追随在该子混音上被启用时，将包络追随者委托添加到该子混音中。

目标是声音子混音

![](/uploads/projects/ue-bluprint/20221218-18044556.png)

## Inputs

在。执行。

目标。Sound Submix 对象参考。

Submix Envelope BP 时：委托（通过参考）。当新的包络数据可用时触发的事件。

## Outputs

输出。执行：执行。

Adds an envelope follower delegate to the submix when envelope following is enabled on this submix.

Target is Sound Submix

## 图示

![](/uploads/projects/ue-bluprint/20221218-18044556.png)

## Inputs

In: Exec.

Target: Sound Submix Object Reference.

On Submix Envelope BP: Delegate (by ref). Event to fire when new envelope data is available..

## Outputs

Out: Exec.
