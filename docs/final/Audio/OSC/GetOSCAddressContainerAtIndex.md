# GetOSCAddressContainerAtIndex

返回提供的'Index'处的 OSC 地址容器。如果索引超出范围，则返回空字符串。

目标是 OSCManager

![](/uploads/projects/ue-bluprint/20221218-18053538.png)

## Inputs

在。Exec.

地址。OSCAddress 结构（通过引用）。

索引。整数。

## Outputs

输出。执行：执行。

容器。字符串。返回提供的'Index'处的 OSC 地址容器。如果索引在界外，则返回空字符串。

<hr>

Returns the OSC Address container at the provided 'Index.' Returns empty string if index is out-of-bounds.

Target is OSCManager

![](/uploads/projects/ue-bluprint/20221218-18053538.png)

## Inputs

In: Exec.

Address: OSCAddress Structure (by ref).

Index: Integer.

## Outputs

Out: Exec.

Container: String. Returns the OSC Address container at the provided 'Index.' Returns empty string if index is out-of-bounds..
