# GetOSCMessageIntegeratIndex

如果在范围内且类型匹配，将 OSCMessage 中提供的索引的值设置为整数。

目标是 OSCManager

## 图示

![](/uploads/projects/ue-bluprint/20221218-18054983.png)

## Inputs

在。执行。

消息。OSCMessage 结构（通过引用）。

索引。整数。

## Outputs

输出。Exec.

值。整数。

成功了。布尔型。在 OSCMessage 中，如果在范围内且类型匹配，将值设置为所提供索引的整数。
