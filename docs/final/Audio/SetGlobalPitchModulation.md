# SetGlobalPitchModulation

设置一个全局性的音高调制标度，适用于所有非 UI 的声音。

- 火灾和遗忘。

- 不复制。

目标是游戏性的静力学

## 图示

![](/uploads/projects/ue-bluprint/20221218-18074686.png)

## Inputs

在。执行。

音高调制。浮点数（单精度）。一个全局设置的音高调制值。

时间秒。浮点数（单精度）。一个时间值，从它的当前值线性地插补全局调制的音高。

## Outputs

输出。执行。
