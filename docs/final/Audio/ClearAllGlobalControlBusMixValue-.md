# ClearAllGlobalControlBusMixValue-

如果设置了全局控制总线的混合值，则清除所有全局控制总线的混合值，使用应用的淡出时间将所有总线的参数恢复到各自的默认值。

目标是音频调制静态

在。执行。

渐变时间。浮点数（单精度）。在当前值和新值之间插值时给用户的淡化时间。如果非正数，则立即改变。
