# SetAudioBus

将源键输入设置为提供的 AudioBus 的输出。如果没有提供对象，就将键设置为效果的输入。

目标是子混音效果多段压缩器预设

## Inputs

输入。执行。

目标。Submix Effect Multiband Compressor Preset 对象参考。

音频总线：音频总线对象参考。

## Outputs

Out: 输出。Exec.
