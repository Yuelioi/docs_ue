# RemoveSubmixEffect

从给定的子混音中删除一个子混音效果预设的所有实例。

目标是音频混合器蓝图库

## 图示

![](/uploads/projects/ue-bluprint/20221218-18042425.png)

## Inputs

在。执行。

Sound Submix: Sound Submix Object Reference.

Submix Effect Preset: Sound Effect Submix Preset（子混音效果预设）对象参考。

## Outputs

输出。Exec.
