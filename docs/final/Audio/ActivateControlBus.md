# ActivateControlBus

手动激活一个调制总线。如果被调用，只有在总线被手动停用或销毁的情况下才会发生停用（即当所有的引用都不活动时，不会停用）。

目标是音频调制静态

![](/uploads/projects/ue-bluprint/20221218-18021677.png)

## Inputs

在。Exec.

总线：声音控制总线对象参考。

## Outputs

Out（输出）。Exec.
