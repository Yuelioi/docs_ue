# SetMaxAudioChannelsScaled

按百分比动态地设置声音的最大数量（也称为 "通道"）。例如，如果你想暂时减少 50%的声音数量，使用 0.50。稍后，你可以通过使用 1.0 恢复到原来的最大语音数。

目标是游戏的静力学

## Inputs

在。执行。

最大通道数规模。Float（单精度）。将最大声音数设置为原始声音数的百分比。
