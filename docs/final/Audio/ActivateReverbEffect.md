# ActivateReverbEffect

激活混响效果而不需要音频音量

目标是游戏的静力学

## 图示

![]($-20221218-18022022.png)

## Inputs

在。执行。

混响效果。混响效果对象参考。要使用的混响效果。

标签名称。名称。与混响效果有关的标签。

优先权。Float（单精度）。混响效果的优先级。

音量。Float（单精度）。混响效果的音量水平。

淡出时间。Float（单精度）。在混响效果完全激活之前的时间。 

## Outputs

输出。执行：执行。

Activates a Reverb Effect without the need for an Audio Volume

Target is Gameplay Statics

## 图示

![]($-20221218-18022022.png)

## Inputs

In: Exec.

Reverb Effect: Reverb Effect Object Reference. Reverb Effect to use.

Tag Name: Name. Tag to associate with Reverb Effect.

Priority: Float (single-precision). Priority of the Reverb Effect.

Volume: Float (single-precision). Volume level of Reverb Effect.

Fade Time: Float (single-precision). Time before Reverb Effect is fully active.  

## Outputs

Out: Exec.
