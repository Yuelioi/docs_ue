# ActivateControlBusMix

手动激活一个总线调制器混合。如果被调用，停用将只发生在混合器被手动停用且未被引用或销毁的情况下（即不会停用，当所有的引用变得不活跃）。

目标是音频调制静态

## Inputs

在。Exec.

混合。声音控制总线混合对象参考。

## Outputs

输出。Exec.
