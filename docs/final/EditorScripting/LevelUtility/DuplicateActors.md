# DuplicateActors

世界编辑器中的重复演员。

目标是编辑演员子系统

## 图示

![](/uploads/projects/ue-bluprint/20221218-18504122.png)

## Inputs

在。执行。

目标。编辑器演员子系统对象参考。

要复制的演员。演员对象参考数组。要复制的演员...

致世界。世界对象参考。将被复制的演员放置在世界中。\*.

偏移。矢量。通过翻译来抵消重复的演员。

## Outputs

出：执行。

返回值。演员对象引用数组。重复的演员，如果没有成功，则为空。

<hr>

Duplicate actors from the world editor.

Target is Editor Actor Subsystem

## 图示

![](/uploads/projects/ue-bluprint/20221218-18504122.png)

## Inputs

In: Exec.

Target: Editor Actor Subsystem Object Reference.

Actors to Duplicate: Array of Actor Object References. Actors to duplicate..

To World: World Object Reference. World to place the duplicated actors in. \*.

Offset: Vector. Translation to offset duplicated actors by..

## Outputs

Out: Exec.

Return Value: Array of Actor Object References. The duplicated actors, or empty if it didn't succeed.
