# NewLevel

关闭当前的持久性关卡（不保存它）。创建一个新的空白关卡并保存它。加载新创建的关卡。

目标是等级编辑子系统

## 图示

![]($-20221218-18511366.png)

## Inputs

在。执行。

目标。等级编辑子系统对象参考。

资产路径。字符串。关卡将被保存的资产路径。 例如：/Game/MyFolder/MyAsset。

## Outputs

出：执行。

返回值。布尔值。如果操作成功为真。
Close the current Persistent Level (without saving it). Create a new blank Level and save it. Load the new created level.

Target is Level Editor Subsystem

## 图示

![]($-20221218-18511366.png)

## Inputs

In: Exec.

Target: Level Editor Subsystem Object Reference.

Asset Path: String. Asset Path of where the level will be saved. ie. /Game/MyFolder/MyAsset.  

## Outputs

Out: Exec.

Return Value: Boolean. True if the operation succeeds..
