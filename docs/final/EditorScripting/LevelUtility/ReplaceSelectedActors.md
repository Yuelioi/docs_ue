# ReplaceSelectedActors

用相同数量的不同类型的演员替换所选的演员，使用指定的工厂生成新的演员。 注意，只有位置、旋转、Drawscale、Drawscale3D、标签和组会从旧的演员中复制。

目标是编辑水平库
