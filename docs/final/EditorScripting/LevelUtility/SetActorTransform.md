# SetActorTransform

如果可能的话，设置给定演员的世界转换。

目标是编辑演员子系统

## 图示

![](/uploads/projects/ue-bluprint/20221218-18513472.png)

## Inputs

在。执行。

目标。编辑器演员子系统对象参考。

在演员中。演员对象参考。

在世界转型。转变（由裁判）。

## Outputs

出：执行。

返回值。布尔值。如果不能设置世界转换，则为假。

<hr>

Sets the world transform of the given actor, if possible.

Target is Editor Actor Subsystem

## 图示

![](/uploads/projects/ue-bluprint/20221218-18513472.png)

## Inputs

In: Exec.

Target: Editor Actor Subsystem Object Reference.

In Actor: Actor Object Reference.

In World Transform: Transform (by ref).

## Outputs

Out: Exec.

Return Value: Boolean. false if the world transform could not be set..
