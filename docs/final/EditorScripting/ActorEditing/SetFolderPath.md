# SetFolderPath

为这个演员指定一个新的文件夹。演员文件夹路径只在开发构建中可用。

目标是演员

## 图示

![](/uploads/projects/ue-bluprint/20221218-18464257.png)

## Inputs

在。执行。

目标。演员对象参考。

新的文件夹路径。名称（按参考）。要分配给演员的新文件夹。

## Outputs

出：执行。
