# GetLodReductionSettings

用指定的 LOD 还原设置复制还原选项。

目标是静态网格编辑器子系统

## 图示

![](/uploads/projects/ue-bluprint/20221218-18540804.png)

## Inputs

在。执行。

目标。静态网格编辑器子系统对象参考。

静态网格。静态网格对象参考。网格的处理。

Lod 索引。整数。我们得到的 LOD 的减少设置。

## Outputs

出：执行。

Out Reduction 选项。网格还原设置结构。缩减设置是我们复制缩减选项的地方。

<hr>

Copy the reduction options with the specified LOD reduction settings.

Target is Static Mesh Editor Subsystem

![](/uploads/projects/ue-bluprint/20221218-18540804.png)

In: Exec.

Lod Index: Integer. The LOD we get the reduction settings..

## Outputs
