# SetLodBuildSettings

为指定的LOD索引设置LOD构建选项。

目标是静态网格编辑器子系统

![]($-20221218-18541206.png)

## Inputs

在。执行。

目标。静态网格编辑器子系统对象参考。

静态网格。静态网格对象参考。网格的处理。

Lod索引。整数。我们将应用构建设置的LOD。

构建选项。网格构建设置结构（按参考）。我们要应用到LOD的构建设置。

## Outputs

出：执行。
Set the LOD build options for the specified LOD index.

Target is Static Mesh Editor Subsystem
