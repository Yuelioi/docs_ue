# SetLocalControlRigBools

在特定时间设置ControlRig控件的bool值

目标是Control Rig Sequencer Editor Library

## 图示

![]($-20221218-18532621.png)

## Inputs

在。执行。

水平序列。水平序列对象参考。活跃的序列来设置值。

控制装置。控制装置对象参考。The ControlRig.

控制名称。名称。控件的名称，应该是一个bool控件。

帧。帧号结构的数组。设置数值的时间。

值。布尔运算的数组。在这些时候设置。

时间单位。ESequenceTimeUnit枚举。帧值的单位，可以是显示率，也可以是tick分辨率。

## Outputs

出：执行。
Set ControlRig Control's bool values at specific times

Target is Control Rig Sequencer Editor Library

## 图示

![]($-20221218-18532621.png)

## Inputs

In: Exec.

Level Sequence: Level Sequence Object Reference. Active Sequence to set value on.

Control Rig: Control Rig Object Reference. The ControlRig.

Control Name: Name. Name of the Control, should be a bool control.

Frames: Array of Frame Number Structures. Times to set the values.

Values: Array of Booleans. to set at those times.

Time Unit: ESequenceTimeUnit Enum. Unit for frame values, either in display rate or tick resolution.  

## Outputs

Out: Exec.
