# FilterbyLayer

通过演员所属的层过滤阵列。

目标是编辑器过滤器库

## 图示

![]($-20221218-18542290.png)

## Inputs

在。执行。

目标数组。Actor对象的数组 引用。要过滤的Actor数组。该数组将不会改变。

层名。名称。行为者所属的层的确切名称。

过滤器类型。EEditorScriptingFilterType枚举。如果尊重条件，应包括或不包括数组的项目。

## Outputs

输出。执行。

返回值。演员对象引用的数组。过滤后的列表。
Filter the array by Layer the Actor belongs to.

Target is Editor Filter Library

## 图示

![]($-20221218-18542290.png)

## Inputs

In: Exec.

Target Array: Array of Actor Object References. Array of Actor to filter. The array will not change..

Layer Name: Name. The exact name of the Layer the actor belongs to..

Filter Type: EEditorScriptingFilterType Enum. Should include or not the array's item if it respects the condition..  

## Outputs

Out: Exec.

Return Value: Array of Actor Object References. The filtered list..
