# AddActionbyDuplication

为数据准备资产添加一个动作

目标是Dataprep核心蓝图库

## 图示

![]($-20221218-18485200.png)

## Inputs

在。执行。

Dataprep资产: Dataprep资产对象参考。该动作将被添加到的数据准备资产。

复制的行动。Dataprep行动资产对象参考。将被复制的行动。

## Outputs

出：执行。

返回值。Dataprep行动资产对象参考。新的行动。
Add an action to a dataprep asset

Target is Dataprep Core Blueprint Library

## 图示

![]($-20221218-18485200.png)

## Inputs

In: Exec.

Dataprep Asset: Dataprep Asset Object Reference. The dataprep asset to which the action will added.

Action to Duplicate: Dataprep Action Asset Object Reference. The action that will be duplicated.  

## Outputs

Out: Exec.

Return Value: Dataprep Action Asset Object Reference. the new action.
