# GetStepObject

从数据准备行动中返回一个步骤的对象

目标是 Dataprep 核心蓝图库

## 图示

![](/uploads/projects/ue-bluprint/20221218-18485860.png)

## Inputs

在。执行。

Dataprep 行动。dataprep 行动资产对象参考。步骤对象将检索的数据准备行动。

索引。整数。该步骤的索引。

## Outputs

出：执行。

返回值。Dataprep Parameterizable Object Object Reference.检索的步骤对象（一般是一个数据准备操作或过滤器）。

<hr>

Return the object of a step from the dataprep action

Target is Dataprep Core Blueprint Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-18485860.png)

## Inputs

In: Exec.

Dataprep Action: Dataprep Action Asset Object Reference. The dataprep action on which the step object will retrieve.

Index: Integer. The index of the step.

## Outputs

Out: Exec.

Return Value: Dataprep Parameterizable Object Object Reference. The retrieved step object (Generally a dataprep operation or filter).
