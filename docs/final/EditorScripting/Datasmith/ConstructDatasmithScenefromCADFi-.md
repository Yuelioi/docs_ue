# ConstructDatasmithScenefromCADFi-

在一个单一的 Datasmith 场景中打开一组 CAD 文件作为演员。将一组文件导入单个 DatasmithScene 资产（使用 ImportScene），只支持 CAD 文件。

目标是 Datasmith 场景元素

## 图示

![](/uploads/projects/ue-bluprint/20221218-18491736.png)

## Inputs

在。执行。

文件路径。字符串阵列。

## Outputs

出：执行。

返回值。Datasmith 场景元素对象参考。打开的 DatasmithScene，它可以被修改并可以被导入。

<hr>

Open set of CAD files as actors in a single datasmith scene. Importing set of files into single DatasmithScene asset(with ImportScene) is supported only for CAD files

Target is Datasmith Scene Element

## 图示

![](/uploads/projects/ue-bluprint/20221218-18491736.png)

## Inputs

In: Exec.

File Paths: Array of Strings.

## Outputs

Out: Exec.

Return Value: Datasmith Scene Element Object Reference. The opened DatasmithScene, that can be modified and can be imported..
