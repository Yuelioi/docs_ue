# GetExistingDatasmithScene

从磁盘上打开一个现有的 DatasmithScene 资产文件。

目标是 Datasmith 场景元素

## 图示

![](/uploads/projects/ue-bluprint/20221218-18492136.png)

## Inputs

在。执行。

资产路径。字符串。DFatasmithScene 资产的路径（例如，"/Game/MyAsset"）。

## Outputs

出：执行。

返回值。Datasmith 场景元素对象参考。打开的 DatasmithScene，它可以被修改并且可以被重新导入。

<hr>

Open an existing DatasmithScene asset file from disk.

Target is Datasmith Scene Element

## 图示

![](/uploads/projects/ue-bluprint/20221218-18492136.png)

## Inputs

In: Exec.

Asset Path: String. Path of the DFatasmithScene asset (eg. "/Game/MyAsset").

## Outputs

Out: Exec.

Return Value: Datasmith Scene Element Object Reference. The opened DatasmithScene, that can be modified and can be reimported..
