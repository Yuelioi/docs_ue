# GetRegisteredCommands

检索当前在子系统中注册的所有命令的命令信息列表。

目标是 UICommands 脚本子系统

## 图示

![](/uploads/projects/ue-bluprint/20221218-18482817.png)

## Inputs

目标。UICommands Scripting Subsystem Object Reference。

## Outputs

返回值。脚本命令信息结构的数组。检索当前在子系统中注册的所有命令的命令信息列表。

Retrieves the list of command info for all commands currently registered in the subsystem

Target is UICommands Scripting Subsystem

## 图示

![](/uploads/projects/ue-bluprint/20221218-18482817.png)

## Inputs

Target: UICommands Scripting Subsystem Object Reference.

## Outputs

Return Value: Array of Scripting Command Info Structures. Retrieves the list of command info for all commands currently registered in the subsystem.
