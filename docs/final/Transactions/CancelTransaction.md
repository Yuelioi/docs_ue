Cancel the current transaction, and no longer capture actions to be placed in the undo buffer.. Note: Only available in the editor.

Target is Kismet System Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-21141744.png)

## Inputs

In: Exec.

Index: Integer. The action counter to cancel transactions from (as returned by a call to BeginTransaction)..

## Outputs

Out: Exec.
