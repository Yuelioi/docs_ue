目标是开放的 CVLens 校准器

Feeds an image to the calibration. It must contain a checkerboard somewhere in the image.. The images fed in should come from the same camera.

Target is Open CVLens Calibrator

## 图示

![](/uploads/projects/ue-bluprint/20221218-19402926.png)

## Inputs

In: Exec.

Target: Open CVLens Calibrator Object Reference.

File Path: String.

## Outputs

Out: Exec.

Return Value: Boolean. true if the calibrator found a checkerboard in the image..
