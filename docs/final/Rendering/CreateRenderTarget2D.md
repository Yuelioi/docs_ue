# CreateRenderTarget2D

创建一个新的渲染目标，并将其初始化为指定的尺寸。

目标是 Kismet Rendering Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-20355523.png)

## Inputs

在。执行。

宽度：整数。

高度：整数。

格式。ETextureRenderTargetFormat 枚举。

清澈的颜色：线性颜色结构。

自动生成 Mip 地图。布尔型。

## Outputs

出：执行。

返回值。纹理渲染目标 2D 对象参考。创建一个新的渲染目标并将其初始化为指定的尺寸。

<hr>

Creates a new render target and initializes it to the specified dimensions

Target is Kismet Rendering Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-20355523.png)

## Inputs

In: Exec.

Width: Integer.

Height: Integer.

Format: ETextureRenderTargetFormat Enum.

Clear Color: Linear Color Structure.

Auto Generate Mip Maps: Boolean.

## Outputs

Out: Exec.

Return Value: Texture Render Target 2D Object Reference. Creates a new render target and initializes it to the specified dimensions.
