# DrawDebugFloatHistoryTransform

基于 FDebugFloatHistory 结构绘制大小为'DrawSize'的二维柱状图，使用 DrawTransform 绘制世界上的位置。

目标是 Kismet 系统库

## 图示

![](/uploads/projects/ue-bluprint/20221218-20361049.png)

## Inputs

在。执行。

浮动历史。调试浮动历史结构（通过参考）。

绘制变换。变换（通过参考）。

绘制尺寸。矢量 2D 结构。

绘制颜色：线性颜色结构。

持续时间。浮点数（单精度）。

## Outputs

出：执行。

Draws a 2D Histogram of size 'DrawSize' based FDebugFloatHistory struct, using DrawTransform for the position in the world.

Target is Kismet System Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-20361049.png)

## Inputs

In: Exec.

Float History: Debug Float History Structure (by ref).

Draw Transform: Transform (by ref).

Draw Size: Vector 2D Structure.

Draw Color: Linear Color Structure.

Duration: Float (single-precision).

## Outputs

Out: Exec.
