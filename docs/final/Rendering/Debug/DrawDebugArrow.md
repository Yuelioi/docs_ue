# DrawDebugArrow

绘制方向性的箭头，从LineStart指向LineEnd。

目标是Kismet系统库

![]($-20221218-20355986.png)

## Inputs

在。执行。

线路起点：向量。

线端：向量。

箭头大小。Float（单精度）。

线条颜色：线性颜色结构。

持续时间。浮点数（单精度）。

厚度。Float（单精度）。  

## Outputs

出：执行。

Draw directional arrow, pointing from LineStart to LineEnd.

Target is Kismet System Library
