# DrawDebugCone

绘制一个调试圆锥体。角度以度数指定

目标是Kismet系统库

## Inputs

在。执行。

原产地。向量。

方向。矢量。

长度。Float（单精度）。

角度宽度：浮点数（单精度）。

角度高度。Float（单精度）。

Num Sides: 整数。

线条颜色：线性颜色结构。

持续时间。Float（单精度）。

厚度。Float（单精度）。  

## Outputs

出：执行。

Draw a debug cone. Angles are specified in degrees

Target is Kismet System Library
