# GetStandardRenderableComponents

读取此水体在不被水网渲染时使用的原始组件列表（例如，指定 WaterMeshOverride 时使用的静态网格组件）。

目标是水体部分

## 图示

![](/uploads/projects/ue-bluprint/20221218-20363729.png)

## Inputs

目标。水体组件对象参考。

## Outputs

返回值。原始组件对象引用数组。读取此水体在不被水网渲染时使用的原始组件列表（例如，指定 WaterMeshOverride 时使用的静态网格组件）。

<hr>

Retrieves the list of primitive components that this water body uses when not being rendered by the water mesh (e.g. the static mesh component used when WaterMeshOverride is specified)

Target is Water Body Component

## 图示

![](/uploads/projects/ue-bluprint/20221218-20363729.png)

## Inputs

Target: Water Body Component Object Reference.

## Outputs

Return Value: Array of Primitive Component Object References. Retrieves the list of primitive components that this water body uses when not being rendered by the water mesh (e.g. the static mesh component used when WaterMeshOverride is specified).
