创建一个新的体积渲染目标，并将其初始化为指定的尺寸。

目标是Kismet Rendering Library

深度：整数。

## Outputs

出：执行。

返回值。纹理渲染目标体对象参考。创建一个新的体积渲染目标，并将其初始化为指定的尺寸。
