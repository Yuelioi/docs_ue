# GetCustomPrimitiveDataIndexforVe-

获取自定义原始数据阵列的向量参数的索引

目标是原始组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-20373970.png)

## Inputs

目标。原始组件对象参考。

参数名称。名称。自定义基元的参数名称。

## Outputs

返回值。整数。自定义基元的索引，如果没有找到则为 INDEX_NONE（-1）。

<hr>

Gets the index of the vector parameter for the custom primitive data array

Target is Primitive Component

## 图示

![](/uploads/projects/ue-bluprint/20221218-20373970.png)

## Inputs

Target: Primitive Component Object Reference.

Parameter Name: Name. The parameter name of the custom primitive.

## Outputs

Return Value: Integer. The index of the custom primitive, INDEX_NONE (-1) if not found.
