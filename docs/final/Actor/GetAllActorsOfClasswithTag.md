# GetAllActorsOfClasswithTag

找到世界上所有具有指定标签的指定类别的演员。这是一个缓慢的操作，谨慎使用，例如，不要每一帧都使用。

目标是游戏的静力学

## 图示

![](/uploads/projects/ue-bluprint/20221218-17343063.png)

## Inputs

在。执行。

演员类。Actor Class Reference。要查找的演员的类别。必须指定，否则结果数组将是空的。

标签。Name（名称）。要查找的标签。必须指定，否则结果数组将是空的。

## Outputs

输出。执行：执行。

Out Actors: 演员对象引用数组。Output array of Actors of the specified tag...

<hr>

Find all Actors in the world of the specified class with the specified tag.. This is a slow operation, use with caution e.g. do not use every frame.

Target is Gameplay Statics

## 图示

![](/uploads/projects/ue-bluprint/20221218-17343063.png)

## Inputs

In: Exec.

Actor Class: Actor Class Reference. Class of Actor to find. Must be specified or result array will be empty..

Tag: Name. Tag to find. Must be specified or result array will be empty..

## Outputs

Out: Exec.

Out Actors: Array of Actor Object References. Output array of Actors of the specified tag..
