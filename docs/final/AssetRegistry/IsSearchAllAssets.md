# IsSearchAllAssets

SearchAllAssets 是否被调用过，或者在启动时被自动调用。当异步（编辑器或烹饪）时，如果 SearchAllAssets 曾经被调用过，任何新挂载的目录将被自动搜索。

目标是资产注册处

## Inputs

目标。资产注册处接口。

## Outputs

返回值。布尔值。SearchAllAssets 是否被调用过，或者在启动时被自动调用。当异步（编辑器或烹饪）时，如果 SearchAllAssets 曾经被调用过，任何新挂载的目录将被自动搜索。

Whether SearchAllAssets has been called, or was auto-called at startup. When async (editor or cooking), if SearchAllAssets has ever been called,. any newly-mounted directory will be automatically searched.
