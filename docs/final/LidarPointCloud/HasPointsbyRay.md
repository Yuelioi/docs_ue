# HasPointsbyRay

如果有任何点被给定的射线击中，则返回 true。

目标是激光雷达点云组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-19431756.png)

## Inputs

目标。激光雷达点云组件对象参考。

原产地。向量。

方向。矢量。

半径。Float（单精度）。

仅可见。布尔型。

## Outputs

返回值。布尔值。如果有任何点被给定的射线击中，则返回真。

<hr>

Returns true if there are any points hit by the given ray.

Target is Lidar Point Cloud Component

## 图示

![](/uploads/projects/ue-bluprint/20221218-19431756.png)

## Inputs

Target: Lidar Point Cloud Component Object Reference.

Origin: Vector.

Direction: Vector.

Radius: Float (single-precision).

Visible Only: Boolean.

## Outputs

Return Value: Boolean. Returns true if there are any points hit by the given ray..
