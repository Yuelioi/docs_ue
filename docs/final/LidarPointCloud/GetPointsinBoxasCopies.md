# GetPointsinBoxasCopies

返回一个包含给定框内点的副本的数组。

目标是激光雷达点云蓝图库

## 图示

![](/uploads/projects/ue-bluprint/20221218-19431308.png)

## Inputs

中心。矢量。

延伸。向量。

仅可见。布尔型（通过引用）。

## Outputs

选定的点。激光雷达点云的点结构阵列。
