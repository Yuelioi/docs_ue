获取粒子模块位置的骨质插座道具

目标是 FXConverter Utilities Library

![](/uploads/projects/ue-bluprint/20221218-19022955.png)

## Inputs

在。执行。

粒子模块。骨头/插座位置对象参考。

## Outputs

出：执行。

输出源类型。ELocationBoneSocketSource Enum.

输出通用偏移。矢量。
