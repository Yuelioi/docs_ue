# DoesSaveGameExist

查看是否存在一个指定名称的保存游戏。

目标是游戏性的统计学

## 图示

![]($-20221218-20470379.png)

## Inputs

在。执行。

槽位名称。字符串。保存游戏槽的名称。

用户索引。整数。对于某些平台，主用户索引用于识别进行保存的用户。

## Outputs

出：执行。

返回值。布尔值。

See if a save game exists with the specified name.

Target is Gameplay Statics

## Inputs

In: Exec.

Out: Exec.
