# GetPipelineState

获取指定 Pipeline 的当前状态。参见 EMovieRenderPipelineState，了解每个状态的更多细节。

目标是电影管道蓝图库

![](/uploads/projects/ue-bluprint/20221218-20090675.png)

在管道中。电影管道对象参考。为获取状态的管道。

## Outputs

返回值。EMovieRenderPipelineState Enum.当前状态。
