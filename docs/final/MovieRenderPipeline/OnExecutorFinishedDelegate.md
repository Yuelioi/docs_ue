# OnExecutorFinishedDelegate

On Executor Finished Delegate:.当执行者完成所有工作时被调用。如果没有工作，报告成功，有致命的错误。订阅错误委托以获得更多信息，任何错误。

为Blueprints/Python公开。与本地的同时被调用。
