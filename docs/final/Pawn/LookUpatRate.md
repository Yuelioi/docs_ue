# LookUpatRate

通过输入调用，在给定的速率下向上看（如果速率为负数，则向下看）。

目标是默认卒

## 图示

![](/uploads/projects/ue-bluprint/20221218-20193040.png)

## Inputs

在。执行。

目标。默认的卒子对象参考。

速率。Float（单精度）。这是一个归一化的速率，即 1.0 意味着 100%的期望转弯速率。

## Outputs

出：执行。

<hr>

Called via input to look up at a given rate (or down if Rate is negative).

Target is Default Pawn

## 图示

![](/uploads/projects/ue-bluprint/20221218-20193040.png)

## Inputs

In: Exec.

Target: Default Pawn Object Reference.

Rate: Float (single-precision). This is a normalized rate, i.e. 1.0 means 100% of desired turn rate.

## Outputs

Out: Exec.
