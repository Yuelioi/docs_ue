# MoveUpWorld

输入回调，在世界空间中向上移动（如果 Val 为负数，则向下移动）。

目标是默认卒

![](/uploads/projects/ue-bluprint/20221218-20193380.png)

## Inputs

在。执行。

目标。默认的卒子对象参考。

Val: 浮点数（单精度）。在世界范围内向上移动的数量（如果是负数，则向下）。

## Outputs

出：执行。

Input callback to move up in world space (or down if Val is negative).

Target is Default Pawn

## 图示

![](/uploads/projects/ue-bluprint/20221218-20193380.png)

## Inputs

In: Exec.

Target: Default Pawn Object Reference.

Val: Float (single-precision). Amount of movement in the world up direction (or down if negative)..

## Outputs

Out: Exec.
