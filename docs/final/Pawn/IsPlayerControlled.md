# IsPlayerControlled

如果被人类玩家控制（由 PlayerController 占有），返回 true。如果是由远程客户端控制的玩家，则返回真

目标是卒子

## 图示

![](/uploads/projects/ue-bluprint/20221218-20192814.png)

## Inputs

目标。卒子对象参考。

## Outputs

返回值。布尔值。如果由人类玩家控制（由 PlayerController 占有），则返回 true。如果是由远程客户端控制的玩家，则返回真。

<hr>

Returns true if controlled by a human player (possessed by a PlayerController). This returns true for players controlled by remote clients

Target is Pawn

![](/uploads/projects/ue-bluprint/20221218-20192814.png)

## Inputs

Target: Pawn Object Reference.

## Outputs

Return Value: Boolean. Returns true if controlled by a human player (possessed by a PlayerController). This returns true for players controlled by remote clients.
