# GetPlayerViewPoint

返回玩家的视角。对于AI来说，这意味着小卒的 "眼睛 "视角。对于人类玩家来说，这意味着摄像机的视点。

@output out_Location，查看球员的位置。@output out_rotation，查看球员的旋转。

目标是控制器
