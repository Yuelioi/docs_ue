# IsSectionCollisionEnabled

检查一个特定的 LOD 网格段是否有碰撞。

目标是静态网格编辑器子系统

## 图示

![](/uploads/projects/ue-bluprint/20221218-21035104.png)

## Inputs

目标。静态网格编辑器子系统对象参考。

静态网格。静态网格对象参考。静态网状物，用于消除碰撞。

LODIndex。整数。StaticMesh LOD 的索引。

科目索引。整数。StaticMesh 截面的索引。

## Outputs

返回值。布尔值。True 是指静态网格部分的指定 LOD 的碰撞被启用。

Checks if a specific LOD mesh section has collision.

Target is Static Mesh Editor Subsystem

## 图示

![](/uploads/projects/ue-bluprint/20221218-21035104.png)

## Inputs

Target: Static Mesh Editor Subsystem Object Reference.

Static Mesh: Static Mesh Object Reference. Static mesh to remove collisions from..

LODIndex: Integer. Index of the StaticMesh LOD..

Section Index: Integer. Index of the StaticMesh Section..

## Outputs

Return Value: Boolean. True is the collision is enabled for the specified LOD of the StaticMesh section..
