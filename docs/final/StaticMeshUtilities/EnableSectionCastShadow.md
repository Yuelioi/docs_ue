# EnableSectionCastShadow

启用/禁用特定LOD的网格段阴影投射。

目标是静态网格编辑器子系统

## Inputs

在。执行。

目标。静态网格编辑器子系统对象参考。

静态网格。静态网格对象参考。Static mesh to Enables/disables shadow casting from...

投影：布尔值。如果该部分应该投射阴影。

LODIndex。整数。StaticMesh LOD的索引。

科目索引。整数。StaticMesh截面的索引。  

## Outputs

出：执行。

Enables/disables mesh section shadow casting for a specific LOD.

Target is Static Mesh Editor Subsystem
