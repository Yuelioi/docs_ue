# GetAnimationFrameData

返回存储在主体框架中的主体的框架数据。如果没有找到有效的数据，则返回false。

目标是Live Link蓝图库

主题框架拉手。主题框架手柄结构（通过参考）。  

## Outputs

动画帧数据。Live Link Animation Frame Data Structure.

返回值。布尔值。返回存储在主体框架中的主体的框架数据。如果没有找到有效的数据则返回false。

Returns the Subject's frame data stored in the Subject Frame. Returns false if no valid data found.

Target is Live Link Blueprint Library
