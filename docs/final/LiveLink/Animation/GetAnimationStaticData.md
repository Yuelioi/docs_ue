# GetAnimationStaticData

返回存储在主体框架中的主体的静态数据。如果没有找到有效的数据，则返回false。

目标是Live Link蓝图库

返回值。布尔值。返回存储在主体框架中的主体的静态数据。如果没有找到有效的数据，返回false。
