# AnnounceAccessibleString

如果启用了无障碍功能，让平台向播放器宣布一个字符串。这些公告可以被系统可及性公告或其他可及性公告请求打断。这应该被明智地使用，因为用公告淹没玩家可能是压倒性的和混乱的。尽量使公告简洁明了。注意：目前只支持Win10、Mac、iOS。
