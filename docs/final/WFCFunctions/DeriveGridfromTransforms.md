# DeriveGridfromTransforms

从变换数组中推导出网格。假设：。-每个变换都代表一个瓦片位置的中心点。-如果有一个有效的网格位置没有变换，则设置空启动器选项。-方向是由数组中第一个变换的偏航决定的。

目标是波函数塌陷子系统
