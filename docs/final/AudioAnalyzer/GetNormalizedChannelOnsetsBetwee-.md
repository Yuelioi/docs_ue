# GetNormalizedChannelOnsetsBetwee-

获取特定时间内被分析的声音的特定通道 cqt。

目标是 Onset NRT

## 图示

![](/uploads/projects/ue-bluprint/20221218-18082997.png)

## Inputs

目标。Onset NRT 对象参考。

在起始秒数。Float（单精度）。

在结束秒数中。Float (single-precision).

在通道中。整数。

## Outputs

Out Onset Timestamps: Float（单精度）的数组。

Out Onset Strengths: Float (single-precision)s 的数组。

Target is Onset NRT
