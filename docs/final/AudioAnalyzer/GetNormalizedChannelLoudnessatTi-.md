# GetNormalizedChannelLoudnessatTi-

获取特定时间内被分析声音的特定通道归一化响度。正常化的响度总是在 0.0 到 1.0 之间。0.0 是指噪音底线，而 1.0 是指声音的最大响度。

目标是响度 NRT

以秒计。Float（单精度）。

在通道中。整数。

## Outputs

输出响度。Float（单精度）。

Target is Loudness NRT
