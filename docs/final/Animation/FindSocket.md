# FindSocket

在这个 SkeletalMesh 中按名称查找一个套接字对象。输入 NAME_None 将返回 NULL。如果有多个同名的 socket，将返回第一个。

目标是骨架网

## 图示

![](/uploads/projects/ue-bluprint/20221218-17493781.png)

## Inputs

目标。骨架网对象参考。

在套接字名称中。名称。

## Outputs

返回值。骨骼网套接字对象参考。通过名称查找此 SkeletalMesh 中的一个套接字对象。输入 NAME_None 将返回 NULL。如果有多个同名的套接字，将返回第一个。
