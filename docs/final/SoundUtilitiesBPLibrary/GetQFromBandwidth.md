Helper function to get Q from bandwidth

Target is Sound Utilities BPFunction Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-20594508.png)

## Inputs

In: Exec.

In Bandwidth: Float (single-precision). The bandwidth value to convert to Q.

## Outputs

Out: Exec.

Return Value: Float (single-precision). The Q value that corresponds to the given bandwidth value.
