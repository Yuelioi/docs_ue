# GetPrincipalPoint

获取以像素为单位的主点。

目标是谷歌 ARCore 摄像头内部结构

## 图示

![](/uploads/projects/ue-bluprint/20221218-19145788.png)

## Inputs

目标。Google ARCore Camera Intrinsics 对象参考。

## Outputs

输出 CX。Float（单精度）。X 轴上的原则点。

输出 CY：浮点数（单精度）。Y 轴的原理点。
