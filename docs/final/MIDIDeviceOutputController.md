# MIDIDeviceOutputController

在这个页面上 

  *动作





## Actions

发送MIDIChannel Aftertouch。发送MIDI通道触摸后的事件类型。目标是MIDID设备输出控制器。

发送MIDIC控制变化。发送MIDI控制变化事件类型。目标是MIDID设备输出控制器。

发送MIDIEvent。发送一个事件类型的MIDI事件原始数据。目标是MIDID设备输出控制器。

发送MIDINote Aftertouch。发送MIDI Note Aftertouch事件类型。目标是MIDID设备输出控制器。

发送MIDINote Off。发送MIDI音符关闭事件类型。目标是MIDIDice输出控制器。

发送MIDINote On。发送MIDI Note On事件类型。目标是MIDID设备输出控制器。

发送MIDIPitch Bend。发送MIDI Pitch Bend事件类型。目标是MIDIDice输出控制器。

发送MIDIProgram Change。发送MIDI程序变化事件类型。目标是MIDID设备输出控制器。

Send MIDINote Aftertouch. Sends MIDI Note Aftertouch event type: Target is MIDIDevice Output Controller.

Send MIDINote On. Sends MIDI Note On event type: Target is MIDIDevice Output Controller.

Send MIDIPitch Bend. Sends MIDI Pitch Bend event type: Target is MIDIDevice Output Controller.
