# DirtyStageActorBlueprint

故意弄脏给定的舞台演员生成的类的 ULUeprint。这对测试舞台剧演员在进入 PIE 时的表现非常有用，因为这通常会在非常敏感的 PIE 转换时触发重新编译。

目标是 SDTests 蓝图库
