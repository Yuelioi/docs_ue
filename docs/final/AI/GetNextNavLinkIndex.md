# GetNextNavLinkIndex

返回给定控制器的当前路径的下一个导航链接的路径索引。如果没有路径或没有进入的导航链接，返回INDEX_NONE。

目标是AIBlueprint Helper Library

控制器。控制器对象参考。 

## Outputs

返回值。整数。返回给定控制器的当前路径的下一个导航链接的路径索引。如果没有路径或没有进入的导航链接，返回INDEX_NONE。

Return the path index of the next nav link for the current path of the given controller. Returns INDEX_NONE if no path or no incoming nav link.
