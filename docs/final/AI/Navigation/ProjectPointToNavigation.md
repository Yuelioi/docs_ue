# ProjectPointToNavigation

在 NavigationData 上投影一个点

目标是导航系统 V1

## 图示

![](/uploads/projects/ue-bluprint/20221218-17473642.png)

## Inputs

点。矢量（通过参考）。

导航数据。导航数据对象参考。

过滤器类。导航查询过滤类参考。

Query Extent（查询范围）。矢量。

## Outputs

投射位置。向量。

返回值。布尔值。在 NavigationData 上投影一个点。
