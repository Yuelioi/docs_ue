# SetMaxSimultaneousTileGeneration-

将限制同时运行的 navmesh 瓦片生成作业的数量，使其达到指定的数量。

目标是导航系统V1

## Inputs

在。执行。

目标。导航系统V1对象参考。

最大作业数。整数，被修剪为至少1。 你不能使用此函数来暂停 navmesh 生成。 

## Outputs
