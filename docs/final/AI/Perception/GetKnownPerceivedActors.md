# GetKnownPerceivedActors

如果SenseToUse为零，所有曾经以任何方式被感知的行为者（并且还没有被遗忘）都将被提取。

目标是AIPerception组件

Out Actors: 演员对象参考数组。

If SenseToUse is none all actors ever perceived in any way (and not forgotten yet) will get fetched
