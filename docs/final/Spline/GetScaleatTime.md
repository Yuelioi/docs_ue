# GetScaleatTime

给定一个从 0 到花键持续时间的时间，返回花键的刻度。

目标是花键组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-21010045.png)

## Inputs

目标。花键组件对象参考。

时间。浮点数（单精度）。

使用恒定速度。布尔型。

## Outputs

返回值。矢量。给出一个从 0 到花键持续时间的时间，返回花键的比例。
