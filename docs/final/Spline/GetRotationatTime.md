# GetRotationatTime

给出一个从 0 到花键持续时间的时间，返回一个与花键的位置和方向相对应的旋转。

目标是花键组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-21005603.png)

## Inputs

目标。花键组件对象参考。

时间。浮点数（单精度）。

坐标空间。ESplineCoordinateSpace Enum.

使用恒定速度。布尔型。

## Outputs

返回值。旋转器。给出一个从 0 到花键持续时间的时间，返回一个与花键的位置和方向相对应的旋转。

## Outputs

Return Value: Rotator. Given a time from 0 to the spline duration, return a rotation corresponding to the spline's position and direction there..
