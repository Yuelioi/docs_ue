# GetRollatSplineInputKey

在所提供的输入键值处获取滚动度。

目标是花键组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-21004908.png)

## Inputs

目标。花键组件对象参考。

在关键。Float (single-precision).

坐标空间。ESplineCoordinateSpace Enum.

## Outputs

返回值。浮点数（单精度）。在提供的输入键值处获得滚动度。

<hr>

Get roll in degrees at the provided input key value

Target is Spline Component

## 图示

![](/uploads/projects/ue-bluprint/20221218-21004908.png)

## Inputs

Target: Spline Component Object Reference.

## Outputs

Return Value: Float (single-precision). Get roll in degrees at the provided input key value.
