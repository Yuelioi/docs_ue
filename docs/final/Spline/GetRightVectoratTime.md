# GetRightVectoratTime

给出一个从 0 到花键持续时间的时间，返回花键在那里的右向量。

目标是花键组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-21004676.png)

## Inputs

目标。花键组件对象参考。

时间。浮点数（单精度）。

坐标空间。ESplineCoordinateSpace Enum.

使用恒定速度。布尔型。

## Outputs

返回值。向量。给出一个从 0 到花键持续时间的时间，返回花键的右边向量。
