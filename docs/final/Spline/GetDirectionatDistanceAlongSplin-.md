# GetDirectionatDistanceAlongSplin-

目标是花键组件

![](/uploads/projects/ue-bluprint/20221218-21002373.png)

## Inputs

目标。花键组件对象参考。

距离。Float（单精度）。

坐标空间。ESplineCoordinateSpace Enum.

## Outputs

返回值。向量。给出一个沿此花键长度的距离，返回花键切线的单位方向向量。
