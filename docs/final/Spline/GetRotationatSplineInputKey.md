# GetRotationatSplineInputKey

在提供的输入键值处获得与沿花键旋转相对应的旋转器。

目标是花键组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-21005363.png)

## Inputs

目标。花键组件对象参考。

在关键。Float (single-precision).

坐标空间。ESplineCoordinateSpace Enum.

## Outputs

返回值。旋转器。获取与在提供的输入键值处沿花键旋转对应的旋转器。
