# GetAttributeSignalFormats

返回一个函数名称和它们的数据类型图。活动模式的通道跨度范围之外的函数被忽略。

目标是 DMX 灯具补丁

## 图示

![](/uploads/projects/ue-bluprint/20221218-18433686.png)

## Inputs

目标。DMX 灯具补丁对象参考。

## Outputs

返回值。DMXAttribute 名称结构到 EDMXFixtureSignalFormat 枚举的映射。返回一个函数名称及其数据类型的映射。活动模式的通道跨度范围之外的函数被忽略。

<hr>

Returns a map of function names and their Data Types.. Functions outside the Active Mode's channel span range are ignored.

Target is DMX Fixture Patch

## 图示

![](/uploads/projects/ue-bluprint/20221218-18433686.png)

## Inputs

Target: DMX Fixture Patch Object Reference.

## Outputs

Return Value: Map of DMXAttribute Name Structures to EDMXFixtureSignalFormat Enums. Returns a map of function names and their Data Types.. Functions outside the Active Mode's channel span range are ignored..
