# StopJumping

在下次更新时停止角色的跳跃。从一个输入事件（比如一个按钮 "向上 "事件）中调用这个函数，以停止应用跳跃的 Z 轴速度。如果不调用这个函数，那么跳跃的 Z 轴速度将被应用，直到达到 JumpMaxHoldTime。

目标是角色
