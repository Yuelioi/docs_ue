# AddLinkedGameplayEffect

将 LinkedGameplayEffect 添加到 SpecHandles 中。LinkedGameplayEffectSpec 将在成功应用 SpecHandle 时被应用。这将为你初始化 LinkedGameplayEffect 的 Spec。返回到新的链接的规格，以备你想在其中添加更多的内容。
