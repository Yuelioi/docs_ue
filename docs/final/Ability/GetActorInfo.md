# GetActorInfo

返回与该能力相关的演员信息，有指向有用对象的缓存指针。

目标是游戏中的能力

## 图示

![](/uploads/projects/ue-bluprint/20221218-17320488.png)

## Inputs

目标。游戏能力对象参考。

## Outputs

返回值。游戏能力的演员信息结构。返回与该能力相关的演员信息，有缓存的有用对象的指针。
