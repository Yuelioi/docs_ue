# EndAbilityState

结束任何具有给定名称的活动能力状态任务。如果名字是'无'，所有活动状态都将被结束（以任意顺序）。

目标是游戏中的能力

## Inputs

在。执行。

目标。Gameplay Ability 对象参考。

可选的状态名称为结束。名称。

## Outputs

出场。执行：执行。
