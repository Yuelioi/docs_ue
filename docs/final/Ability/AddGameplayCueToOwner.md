# AddGameplayCueToOwner

为能力所有者添加一个持久的游戏提示。如果能力结束，可以选择删除。

目标是游戏能力

## 图示

![](/uploads/projects/ue-bluprint/20221218-17295467.png)

## Inputs

在。执行。

目标。游戏能力对象参考。

游戏提示标签。Gameplay Tag 结构。

语境。Gameplay Effect Context Handle 结构。

Remove on Ability End: Boolean（布尔）。

## Outputs

输出。执行：执行。
