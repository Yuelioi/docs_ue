# SetAllNavigationRules

设置所有方向的小组件导航规则。这只能在小组件树中的小组件上调用。

目标是小工具

## 图示

![](/uploads/projects/ue-bluprint/20221218-21355067.png)

## Inputs

在。执行。

目标。小工具对象参考。

规则。EUINavigationRule 枚举。当导航发生时，要使用的规则。

焦点的小工具。名称。当使用明确的规则时，聚焦于这个小部件。

## Outputs

出：执行。

<hr>

Sets the widget navigation rules for all directions. This can only be called on widgets that are in a widget tree.

Target is Widget

## 图示

![](/uploads/projects/ue-bluprint/20221218-21355067.png)

## Inputs

In: Exec.

Target: Widget Object Reference.

Rule: EUINavigationRule Enum. The rule to use when navigation is taking place.

Widget to Focus: Name. When using the Explicit rule, focus on this widget.

## Outputs

Out: Exec.
