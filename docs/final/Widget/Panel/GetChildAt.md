# GetChildAt

在一个索引处获取小组件。

目标是面板小工具

## 图示

![]($-20221218-21352779.png)

## Inputs

目标。Panel Widget对象参考。

索引。整数。小工具的索引。  

## Outputs

返回值。小工具对象参考。在给定索引处的小部件，如果那里没有小部件，则什么都没有。

Gets the widget at an index.

Target is Panel Widget

## 图示

![]($-20221218-21352779.png)

## Inputs

Target: Panel Widget Object Reference.

Index: Integer. The index of the widget..  

## Outputs

Return Value: Widget Object Reference. The widget at the given index, or nothing if there is no widget there..
