# MakeBrushfromTexture

从Texture2D创建一个石板刷

目标是小工具蓝图库

## 图示

![]($-20221218-21334035.png)

## Inputs

纹理。纹理2D对象参考。

宽度：整数。当小于或等于0时，画笔的宽度将默认为纹理的宽度。

高度：整数。当小于或等于零时，画笔的高度将默认为纹理的高度。  

## Outputs

返回值。石板刷结构。一个新的石板刷，使用纹理。

Creates a Slate Brush from a Texture2D

Target is Widget Blueprint Library

## 图示

![]($-20221218-21334035.png)

## Inputs

Texture: Texture 2D Object Reference.

Width: Integer. When less than or equal to zero, the Width of the brush will default to the Width of the Texture.

Height: Integer. When less than or equal to zero, the Height of the brush will default to the Height of the Texture.  

## Outputs

Return Value: Slate Brush Structure. A new slate brush using the texture..
