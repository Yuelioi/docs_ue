# GetDesiredSize

获取小组件所需的大小。注意：底层的 Slate 小组件必须存在并且是有效的，同时在这个值有任何用途之前，必须至少有一次预传递。

目标是小工具

## 图示

![](/uploads/projects/ue-bluprint/20221218-21342206.png)

## Inputs

目标。小工具对象参考。

## Outputs

返回值。矢量 2D 结构。小部件的期望尺寸。
