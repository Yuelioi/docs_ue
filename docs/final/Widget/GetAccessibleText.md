# GetAccessibleText

从底层的 Slate 可访问小组件获取可访问的文本

目标是小工具

## 图示

![](/uploads/projects/ue-bluprint/20221218-21341209.png)

## Inputs

目标。小工具对象参考。

## Outputs

返回值。文本。底层 Slate 可访问小组件的可访问文本。如果可访问性被禁用或底层可访问部件无效，则返回一个空文本。

<hr>

Gets the accessible text from the underlying Slate accessible widget

Target is Widget

## 图示

![](/uploads/projects/ue-bluprint/20221218-21341209.png)

## Inputs

Target: Widget Object Reference.

## Outputs

Return Value: Text. The accessible text of the underlying Slate accessible widget. Returns an empty text if accessibility is dsabled or the underlying accessible widget is invalid..
