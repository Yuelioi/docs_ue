# GetViewportWidgetGeometry

获取持有所有添加到 "视口 "的小组件的几何图形。你可以使用这个几何图形在绝对空间和本地空间之间转换在这个小组件上持有的小组件。

目标是 Widget Layout Library

In: Exec.

## Outputs

出：执行。

返回值。几何结构。获取持有所有添加到 "视口 "的部件的几何形状。你可以使用这个几何结构在绝对空间和本地空间之间转换在这个小部件上持有的小部件。
