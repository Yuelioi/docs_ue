# GetFocalLengthPoint

通过给定的焦距和变焦来获取一个焦距点，如果该点不存在则返回false。

目标是镜头文件

![]($-20221218-19404731.png)

## Inputs

目标。镜头文件对象参考。

焦点。Float（单精度）。

在缩放中。Float（单精度）。  

## Outputs

Out Focal Length Info:焦距信息结构。

返回值。布尔值。通过给定的焦距和变焦来获取一个焦距点，如果不存在焦距点，则返回false。

Gets a Focal Length point by given focus and zoom, if point does not exists returns false

Target is Lens File

Target: Lens File Object Reference.
