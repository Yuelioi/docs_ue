# AddFocalLengthPoint

在我们的地图上添加一个焦点。如果该地点已经存在一个点，它将被更新。

目标是镜头文件

![]($-20221218-19403870.png)

## Inputs

在。执行。

目标。镜头文件对象参考。

新的焦点。Float（单精度）。

新的缩放。Float（单精度）。

新的焦距。焦距信息结构（按参考）。  

## Outputs

出：执行。

Adds a focal length point in our map. If a point already exist at the location, it is updated
