# GetSTMapPoint

通过给定的焦点和缩放来获取一个 ST 地图点，如果该点不存在则返回 false。

目标是镜头文件

## 图示

![](/uploads/projects/ue-bluprint/20221218-19405367.png)

## Inputs

目标。镜头文件对象参考。

焦点。Float（单精度）。

在缩放中。Float（单精度）。

## Outputs

输出 STMap 信息。STMap 信息结构。

返回值。布尔值。通过给定的焦点和缩放来获取 ST 地图的一个点，如果该点不存在则返回 false。

<hr>

Gets a ST Map point by given focus and zoom, if point does not exists returns false

Target is Lens File

## 图示

![](/uploads/projects/ue-bluprint/20221218-19405367.png)

## Inputs

Target: Lens File Object Reference.

In Focus: Float (single-precision).

In Zoom: Float (single-precision).

## Outputs

Out STMap Info: STMap Info Structure.

Return Value: Boolean. Gets a ST Map point by given focus and zoom, if point does not exists returns false.
