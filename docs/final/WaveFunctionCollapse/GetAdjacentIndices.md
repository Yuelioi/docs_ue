## 图示

![](/uploads/projects/ue-bluprint/20221218-21325212.png)

## Inputs

In: Exec.

Index: Integer. 2D array index.

Resolution: Int Vector Structure.

## Outputs

Out: Exec.

Return Value: Map of Integers to EWaveFunctionCollapseAdjacency Enums. Map of valid adjacent indices and its adjacency in relation to the given index.
