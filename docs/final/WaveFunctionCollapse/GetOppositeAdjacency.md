# GetOppositeAdjacency

获取给定邻接关系的相反邻接关系。例如，GetOppositeAdjacency(Front)将返回Back。

目标是波函数崩溃BPLibrary

![]($-20221218-21325686.png)

## Inputs

相邻关系。EWaveFunctionCollapseAdjacency枚举。邻接方向。  

## Outputs

返回值。EWaveFunctionCollapseAdjacency Enum.

Get the opposite adjacency for a given adjacency. For example GetOppositeAdjacency(Front) will return Back.

Target is Wave Function Collapse BPLibrary
