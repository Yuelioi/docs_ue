在。执行。

Grab geometry data from a StaticMesh asset.

Target is Kismet Procedural Mesh Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-18254733.png)

## Inputs

In: Exec.

In Mesh: Static Mesh Object Reference.

LODIndex: Integer.

Section Index: Integer.

## Outputs

Out: Exec.

Vertices: Array of Vectors.

Triangles: Array of Integers.

Normals: Array of Vectors.

UVs: Array of Vector 2D Structures.

Tangents: Array of Proc Mesh Tangent Structures.
