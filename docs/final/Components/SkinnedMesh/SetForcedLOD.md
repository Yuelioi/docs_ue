# SetForcedLOD

设置网格组件的 ForcedLodModel

目标是带皮的网格组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-18265911.png)

## Inputs

在。Exec.

目标。Skinned Mesh Component 对象参考。

在 New Forced LOD: Integer. 设置新的 ForcedLODModel，强制设置传入的 LOD。范围是[1, Max Number of LOD]。这将影响到下一次的更新。

## Outputs

输出。执行：执行。

Set ForcedLodModel of the mesh component

Target is Skinned Mesh Component

## 图示

![](/uploads/projects/ue-bluprint/20221218-18265911.png)

## Inputs

In: Exec.

Target: Skinned Mesh Component Object Reference.

In New Forced LOD: Integer. Set new ForcedLODModel that forces to set the incoming LOD. Range from [1, Max Number of LOD]. This will affect in the next tick update..

## Outputs

Out: Exec.
