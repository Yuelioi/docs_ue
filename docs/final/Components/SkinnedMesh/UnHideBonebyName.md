# UnHideBonebyName

解除对指定骨骼的名称隐藏。目前，这只是对隐藏的骨骼强制执行一个0的比例。与HideBone By Index相比--它可以跟踪骨骼列表并在LOD变化时进行更新。

目标是带皮的网格组件
