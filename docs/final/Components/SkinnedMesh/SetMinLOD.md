# SetMinLOD

设置网格组件的MinLodModel

目标是带皮的网格组件

## 图示

![]($-20221218-18270249.png)

## Inputs

在。执行。

目标。Skinned Mesh Component对象参考。

在新的最小LOD：整数。设置新的MinLodModel，确保LOD不低于这个值。范围是[0, Max Number of LOD - 1]。这将影响到下一次的更新。

熄灭。执行。
