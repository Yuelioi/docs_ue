# SetMinLOD

设置网格组件的 MinLodModel

目标是带皮的网格组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-18270249.png)

## Inputs

在。执行。

目标。Skinned Mesh Component 对象参考。

在新的最小 LOD：整数。设置新的 MinLodModel，确保 LOD 不低于这个值。范围是[0, Max Number of LOD - 1]。这将影响到下一次的更新。

## Outputs

熄灭。执行。

<hr>

Set MinLodModel of the mesh component

Target is Skinned Mesh Component

## 图示

![](/uploads/projects/ue-bluprint/20221218-18270249.png)

## Inputs

In: Exec.

Target: Skinned Mesh Component Object Reference.

In New Min LOD: Integer. Set new MinLodModel that make sure the LOD does not go below of this value. Range from [0, Max Number of LOD - 1]. This will affect in the next tick update..

## Outputs

Out: Exec.
