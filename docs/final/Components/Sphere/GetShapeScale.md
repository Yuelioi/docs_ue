# GetShapeScale

获取这个形状所使用的比例。这是一个统一的比例，是任何非统一比例的最小值。

目标是球体碰撞

## 图示

![](/uploads/projects/ue-bluprint/20221218-18272640.png)

## Inputs

目标。Sphere Collision Object Reference。

## Outputs

返回值。Float（单精度）。这个形状所使用的比例。
