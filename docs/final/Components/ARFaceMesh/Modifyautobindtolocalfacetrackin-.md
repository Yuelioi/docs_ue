# Modifyautobindtolocalfacetrackin-

如果自动绑定为真，那么这个组件将在每次打勾时根据本地人脸追踪数据进行自我更新。如果自动绑定是关闭的，则勾选功能将被禁用。

目标是苹果 ARKit 人脸网格组件
