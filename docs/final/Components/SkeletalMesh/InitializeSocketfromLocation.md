# InitializeSocketfromLocation

根据离世界位置和世界法线最近的骨骼设置骨骼名称、相对位置和相对浮动。

目标是骨架网插口

## 图示

![](/uploads/projects/ue-bluprint/20221218-18261747.png)

## Inputs

在。执行。

目标。骨骼网套筒对象参考。

Skel Comp: 骨骼网组件对象参考。

世界位置。矢量。

世界法线。向量。

## Outputs

输出。Exec.
