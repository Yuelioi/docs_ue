# GetPostProcessInstance

返回活动的后处理实例（如果有的话）。这个实例是在这个组件所使用的网格上设置的，并在主实例之后立即进行评估。

目标是骨架网状结构组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-18261403.png)

## Inputs

目标。Skeletal Mesh Component 对象参考。

## Outputs

返回值。动画实例对象参考。返回活动的后期处理实例（如果有的话）。这是在这个组件使用的网格上设置的，并且在主实例之后立即进行评估。

Returns the active post process instance is one is available. This is set on the mesh that this. component is using, and is evaluated immediately after the main instance.

Target is Skeletal Mesh Component
