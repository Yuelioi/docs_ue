# GetScaledCapsuleHalfHeight

返回由组件比例缩放的胶囊半高。这包括圆柱体和半球体的帽子。

目标是胶囊的碰撞

![]($-20221218-18230085.png)

## Inputs

目标。Capsule Collision Object Reference。 

## Outputs

返回值。Float（单精度）。由组件比例缩放的胶囊半高。
