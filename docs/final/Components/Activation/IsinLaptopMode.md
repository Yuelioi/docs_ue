# IsinLaptopMode

检查一个可转换的笔记本电脑是否为笔记本电脑模式。

目标是平台事件组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-18222113.png)

## Inputs

在。Exec.

目标。平台事件组件对象参考。

## Outputs

输出。执行。

返回值。布尔值。如果在笔记本模式下为 true，否则为 false，或者如果不是可转换的笔记本。
