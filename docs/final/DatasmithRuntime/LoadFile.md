使用与之相关的Datasmith翻译器加载一个文件

目标是Datasmith运行时库

## Inputs

在。执行。

Datasmith Runtime Actor。Datasmith 目的地对象参考。要将文件载入的角色。

文件路径。字符串。要加载的文件的路径。  

## Outputs

出：执行。

返回值。布尔值。如果相关的翻译器已经被。

Target is Datasmith Runtime Library
