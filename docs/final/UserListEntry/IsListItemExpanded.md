# IsListItemExpanded

如果该条目所代表的项目目前已经展开并显示其子项，则返回 true。仅限树状视图条目。

目标是用户列表条目库

## 图示

![](/uploads/projects/ue-bluprint/20221218-21225007.png)

## Inputs

用户名单录入。用户名单条目界面。注意：在视觉上没有传送，但这默认为 "自己"。如果在内部调用，不需要勾选...

## Outputs

返回值。布尔值。
