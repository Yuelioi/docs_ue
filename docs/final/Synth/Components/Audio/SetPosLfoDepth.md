# SetPosLfoDepth

设置控制工作台位置的 Lfo 围绕当前位置值的调制深度。0.0=无调制，1.0=当前位置+/-0.5（Lfo+位置结果将夹紧[0.0, 1.0]）。

目标是合成器组件单声道波形表
