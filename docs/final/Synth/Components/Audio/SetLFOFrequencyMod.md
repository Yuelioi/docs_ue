# SetLFOFrequencyMod

设置 LFO 的调制频率，单位是 Hz

目标是模块化合成器组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-21074980.png)

## Inputs

在。执行。

目标。Modular Synth Component Object Reference。

LFOIndex。整数。为哪个 LFO 设置频率。

频率 Mod Hz。浮点数（单精度）。要使用的 LFO 频率。

## Outputs

出：执行。

<hr>

Sets the LFO frequency modulation in hz

Target is Modular Synth Component

## Inputs

In: Exec.

Target: Modular Synth Component Object Reference.

Frequency Mod Hz: Float (single-precision). The LFO frequency to use..

Out: Exec.
