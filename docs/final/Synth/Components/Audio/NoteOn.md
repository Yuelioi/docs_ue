# NoteOn

开始一个新的音符（重新调整调制器等）。

目标是合成器组件单声道波形表

## 图示

![]($-20221218-21064566.png)

## Inputs

在。执行。

目标。Synth Component Mono Wave Table对象参考。

在Midi音符中：浮点数（单精度）。

在速度中。Float（单精度）。  

## Outputs

出：执行。

Starts a new note (retrigs modulators, etc.)

Target is Synth Component Mono Wave Table

![]($-20221218-21064566.png)

## Inputs

In: Exec.

Target: Synth Component Mono Wave Table Object Reference.

In Midi Note: Float (single-precision).

In Velocity: Float (single-precision).  

## Outputs

Out: Exec.
