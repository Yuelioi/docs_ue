# GetBoundObjects

获取与当前级别序列编辑器绑定到给定绑定ID的对象

目标是水平序列编辑器蓝图库

## 图示

![]($-20221218-19411435.png)

## Inputs

对象绑定。电影场景对象绑定的ID结构。  

## Outputs

返回值。对象引用的数组。获取与当前级别序列编辑器的给定绑定ID绑定的对象。

Get the object bound to the given binding ID with the current level sequence editor

Target is Level Sequence Editor Blueprint Library
