# GetBoundObjects

获取与当前级别序列编辑器绑定到给定绑定 ID 的对象

目标是水平序列编辑器蓝图库

## 图示

![](/uploads/projects/ue-bluprint/20221218-19411435.png)

## Inputs

对象绑定。电影场景对象绑定的 ID 结构。

## Outputs

返回值。对象引用的数组。获取与当前级别序列编辑器的给定绑定 ID 绑定的对象。

Object Binding: Movie Scene Object Binding ID Structure.
