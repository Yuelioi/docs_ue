# OnActorBeginOverlap

On Actor Begin Overlap:. Called when another actor begins to overlap with this actor, such as a player walking onto a trigger. Event when an object has an obstructive collision, such as a player hitting a wall, see "Impact" event.

Components on both this actor and the other actor must have bGenerateOverlapEvents set to true to generate overlap events.
