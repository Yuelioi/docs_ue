# GetSocketTransform

获得世界空间的插座转换。

目标是场景组件

## 图示

![]($-20221218-21151205.png)

## Inputs

目标。场景组件对象参考。

在插座名称。名称。套接字或骨骼的名称，以获得转换。

变换空间。ERelativeTransformSpace Enum.  

## Outputs

返回值。转换。如果找到插座，则为世界空间中的插座变换。否则它将返回组件在世界空间的变换。

Get world-space socket transform.

Target is Scene Component

## 图示

![]($-20221218-21151205.png)

## Inputs

Target: Scene Component Object Reference.

In Socket Name: Name. Name of the socket or the bone to get the transform.

Transform Space: ERelativeTransformSpace Enum.  

## Outputs

Return Value: Transform. Socket transform in world space if socket if found. Otherwise it will return component's transform in world space..
