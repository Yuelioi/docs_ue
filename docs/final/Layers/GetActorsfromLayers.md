# GetActorsfromLayers

获取所有与指定图层中的任何一个相关的演员。类似于 AppendActorsFromLayers，但它返回而不是追加演员。

目标是分层子系统

## 图示

![](/uploads/projects/ue-bluprint/20221218-19374869.png)

## Inputs

目标。分层子系统对象参考。

图层名称。名称的数组。要找到演员的层。

## Outputs

返回值。演员对象引用数组。要把找到的演员分配到的列表。

Gets all the actors associated with ANY of the specified layers. Analog to AppendActorsFromLayers but it returns rather than appends the actors.

Target is Layers Subsystem

## 图示

![](/uploads/projects/ue-bluprint/20221218-19374869.png)

## Inputs

Target: Layers Subsystem Object Reference.

Layer Names: Array of Names. The layers to find actors for..

## Outputs

Return Value: Array of Actor Object References. The list to assign the found actors to..
