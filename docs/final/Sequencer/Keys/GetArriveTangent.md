# GetArriveTangent

如果插值模式是RCIM_Cubic，在这个键上到达的正切线

目标是电影场景脚本的浮动键

## 图示

![]($-20221218-20485420.png)

## Inputs

目标。电影场景脚本的浮动键对象参考。  

## Outputs

返回值。浮点数（单精度）。到达切线值。以 "tan(y/x) "的形式表示几何切线，其中y是键的值，x是秒（都是相对于键的）。

If Interpolation Mode is RCIM_Cubic, the arriving tangent at this key

Target is Movie Scene Scripting Float Key

## 图示

![]($-20221218-20485420.png)

## Inputs

Target: Movie Scene Scripting Float Key Object Reference.  

## Outputs

Return Value: Float (single-precision). Arrival Tangent value. Represents the geometric tangents in the form of "tan(y/x)" where y is the key's value and x is the seconds (both relative to key).
