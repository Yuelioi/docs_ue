# ComputeEffectiveRange_Double

计算该通道的有效范围，例如，其关键时间的延伸。

目标是电影场景脚本的双通道

## 图示

![]($-20221218-20484680.png)

## Inputs

目标。电影场景脚本的双通道对象参考。  

## Outputs

返回值。序列器脚本范围结构。一个范围，代表这个通道在序列的帧分辨率中占据的最大时间范围。
