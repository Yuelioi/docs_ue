# GetChildMasterTracks

获取该文件夹所包含的主轨

目标是电影场景文件夹扩展

## 图示

![](/uploads/projects/ue-bluprint/20221218-20541459.png)

## Inputs

文件夹。电影场景文件夹对象参考。要获取主轨的文件夹。

## Outputs

返回值。电影场景轨道对象引用数组。给定文件夹下的主轨道。
