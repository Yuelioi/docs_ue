# RemoveChildMasterTrack

从给定的文件夹中删除一条主轨道

目标是电影场景文件夹扩展

## 图示

![](/uploads/projects/ue-bluprint/20221218-20550938.png)

## Inputs

在。执行。

文件夹。电影场景文件夹 对象参考。要从中删除轨道的文件夹。

在主轨道。电影场景轨道对象参考。要删除的轨道。

## Outputs

出：执行。

返回值。布尔值。如果移除成功，则为真。
