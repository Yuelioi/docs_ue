# GetRootFoldersinSequence

按照提供的顺序获取根文件夹

目标是电影场景序列扩展

## 图示

![](/uploads/projects/ue-bluprint/20221218-20544597.png)

## Inputs

序列。电影场景序列对象参考。要检索文件夹的序列。

## Outputs

返回值。电影场景文件夹对象参考数组。包含在给定序列中的文件夹。
