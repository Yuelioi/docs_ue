# GetPlaybackEndSeconds

获取该序列的播放结束时间（秒）。

目标是电影场景序列扩展

## 图示

![](/uploads/projects/ue-bluprint/20221218-20543792.png)

## Inputs

在。执行。

序列。电影场景序列对象参考。在其中获得播放结束的序列。

## Outputs

出：执行。

返回值。浮点数（单精度）。此序列的播放结束。
