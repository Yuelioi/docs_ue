# GetEndFrame

获取指定范围的结束帧，如果它有一个。定义为超出该范围的第一个后续帧。

目标是序列器脚本范围扩展

## 图示

![]($-20221218-20542254.png)

## Inputs

在。执行。

范围。序列器脚本的范围结构（通过引用）。从中获取终点的范围。  

## Outputs

出：执行。

返回值。整数。

Get the ending frame for the specified range, if it has one. Defined as the first subsequent frame that is outside of the range.

Target is Sequencer Scripting Range Extensions

## 图示

![]($-20221218-20542254.png)

## Inputs

In: Exec.

Range: Sequencer Scripting Range Structure (by ref). The range to get the end from.  

## Outputs

Out: Exec.

Return Value: Integer.
