# GetChildObjectBindings

获取该文件夹所包含的对象绑定。

目标是电影场景文件夹扩展

## 图示

![](/uploads/projects/ue-bluprint/20221218-20541571.png)

## Inputs

文件夹。电影场景文件夹对象参考。要获得绑定的文件夹。

## Outputs

返回值。序列器绑定代理结构的数组。给定文件夹下的对象绑定。
