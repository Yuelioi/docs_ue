# GetChildFolders

获取一个给定文件夹的子文件夹

目标是电影场景文件夹扩展

## 图示

![](/uploads/projects/ue-bluprint/20221218-20541345.png)

## Inputs

文件夹。电影场景文件夹对象参考。要获取子文件夹的文件夹。

## Outputs

返回值。电影场景文件夹对象引用数组。与给定文件夹相关的子文件夹。
