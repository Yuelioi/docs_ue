# GetSequenceName

获取该播放器正在播放的序列的名称

目标是电影场景序列播放器

## 图示

![]($-20221218-20513152.png)

## Inputs

目标。电影场景序列播放器对象参考。

添加客户信息。布尔值。如果为真，如果作为客户运行，则添加客户索引。  

## Outputs

返回值。字符串。序列的名称，如果没有设置序列，则为 None。

Get the name of the sequence this player is playing

Target is Movie Scene Sequence Player

## 图示

![]($-20221218-20513152.png)

## Inputs

Target: Movie Scene Sequence Player Object Reference.

Add Client Info: Boolean. If true, add client index if running as a client.  

## Outputs

Return Value: String. the name of the sequence, or None if no sequence is set.
