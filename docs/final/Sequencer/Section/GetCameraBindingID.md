# GetCameraBindingID

获取此 CameraCut 部分的相机绑定。

目标是电影场景的镜头剪辑部分

## 图示

![](/uploads/projects/ue-bluprint/20221218-20521781.png)

## Inputs

目标。电影场景相机剪切部分对象参考。

## Outputs

返回值。电影场景对象的绑定 ID 结构（通过引用）。获取此 CameraCut 部分的摄像机绑定。
