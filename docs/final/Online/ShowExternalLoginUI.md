# ShowExternalLoginUI

显示当前活动的在线子系统的登录界面，如果该子系统支持登录界面。

目标是显示登录UICallback代理

在。执行。

在播放器控制器中。播放器控制器对象参考。  

## Outputs

出：执行。

关于成功。执政。On Success:.当有一个成功的查询时被调用。

失败时。执行。失败时：。当有一个不成功的查询时被调用。

播放器控制器。播放器控制器对象参考。

Shows the login UI for the currently active online subsystem, if the subsystem supports a login UI.
