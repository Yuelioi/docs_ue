# SetInterfaceClass

设置传入变量的代表类。注意：检查各个针脚上的工具提示。你不能通过连接导线到这个节点来绕过验证！!

目标是相机镜头效果接口类支持库

## Inputs

在。执行。

类。Actor类参考。必须实现CameraLensEffectInterface--当把变量连接到输入时，注意输入类实际上实现了该接口。

Var:相机镜头界面类支持结构（通过引用）。镜头效果类的封装器（用于验证）。

## Outputs

有效。执行者。
