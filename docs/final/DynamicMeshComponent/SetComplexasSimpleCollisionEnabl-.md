# SetComplexasSimpleCollisionEnabl-

如果 bEnabled=true，设置 bEnableComplexCollision=true 和 CollisionType=CTF_UseComplexAsSimple。如果 bEnabled=true，设置 bEnableComplexCollision=false 和 CollisionType=CTF_UseDefault。
