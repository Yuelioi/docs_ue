# GetDependency

通过数值获取索引'Index'处的依赖关系。如果索引无效，将崩溃

目标是变异体

## 图示

![](/uploads/projects/ue-bluprint/20221218-21232086.png)

## Inputs

目标。变体对象参考。

索引：整数。

## Outputs

返回值。Variant Dependency 结构。通过值获取索引'Index'处的依赖关系。如果索引无效，将崩溃。

<hr>

Get the dependency at index 'Index' by value. Will crash if index is invalid

Target is Variant

## 图示

![](/uploads/projects/ue-bluprint/20221218-21232086.png)

## Inputs

Target: Variant Object Reference.

Index: Integer.

## Outputs

Return Value: Variant Dependency Structure. Get the dependency at index 'Index' by value. Will crash if index is invalid.
