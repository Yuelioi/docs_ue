目标是Optimus节点图

出：执行。

返回值。布尔值。复制同一图形中的节点集合，使用InPosition位置，作为被粘贴节点的左上角原点。

Duplicate a collection of nodes from the same graph, using the InPosition position. to be the top-left origin of the pasted nodes.
