# PauseAnimation

暂停这个小组件中已经运行的动画

目标是用户小工具

## 图示

![]($-20221218-21212928.png)

## Inputs

在。执行。

目标。用户小部件对象参考。

在动画中。小工具动画对象参考。  

## Outputs

出：执行。

返回值。Float（单精度）。动画暂停时的时间点，相对于它的起始位置。当你触发PlayAnimation时，用这个作为StartAtTime。

Pauses an already running animation in this widget

Target is User Widget

## 图示

![]($-20221218-21212928.png)

## Inputs

In: Exec.

Target: User Widget Object Reference.

In Animation: Widget Animation Object Reference.  

## Outputs

Out: Exec.

Return Value: Float (single-precision). the time point the animation was at when it was paused, relative to its start position. Use this as the StartAtTime when you trigger PlayAnimation..
