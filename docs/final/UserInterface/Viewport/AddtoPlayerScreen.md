# AddtoPlayerScreen

将部件添加到游戏视口中专门用于玩家的部分。这在分屏游戏中很有价值，因为你只需要在视口的玩家部分显示一个小部件。

目标是用户小工具

## 图示

![](/uploads/projects/ue-bluprint/20221218-21224021.png)

## Inputs

在。执行。

目标。用户小部件对象参考。

ZOrder:整数。这个数字越大，这个小部件就越在上面。

## Outputs

出：执行。

返回值。布尔值。

<hr>

Adds the widget to the game's viewport in a section dedicated to the player. This is valuable in a split screen. game where you need to only show a widget over a player's portion of the viewport.

Target is User Widget

## 图示

![](/uploads/projects/ue-bluprint/20221218-21224021.png)

## Inputs

In: Exec.

Target: User Widget Object Reference.

ZOrder: Integer. The higher the number, the more on top this widget will be..

## Outputs

Out: Exec.

Return Value: Boolean.
