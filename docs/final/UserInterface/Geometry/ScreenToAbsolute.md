# ScreenToAbsolute

将一个以像素为单位的屏幕位置转换为绝对的应用程序坐标。如果bIncludeWindowPosition为真，那么这个方法也将删除游戏窗口的位置（在窗口模式下很有用）。

目标是石板蓝图库

## 图示

![]($-20221218-21215553.png)

## Inputs

屏幕位置。矢量2D结构。

包括窗口位置。布尔值。  

## Outputs

绝对坐标。矢量二维结构。

Translates a screen position in pixels into absolute application coordinates.. If bIncludeWindowPosition is true, then this method will also remove the game window's position (useful when in windowed mode).

Target is Slate Blueprint Library

## 图示

![]($-20221218-21215553.png)

## Inputs

Screen Position: Vector 2D Structure.

Include Window Position: Boolean.  

## Outputs

Absolute Coordinate: Vector 2D Structure.
