# LocaltoAbsolute

将本地坐标转换为绝对坐标

绝对坐标可以是桌面空间，也可以是窗口空间，这取决于小部件层次结构的根在什么空间。

目标是石板蓝图库

## 图示

![]($-20221218-21215318.png)

## Inputs

几何学。几何结构（由裁判）。

本地坐标。矢量二维结构。  

## Outputs

返回值。矢量2D结构。绝对坐标。

Translates local coordinates into absolute coordinates

Absolute coordinates could be either desktop or window space depending on what space the root of the widget hierarchy is in.

Target is Slate Blueprint Library

![]($-20221218-21215318.png)

## Inputs

Geometry: Geometry Structure (by ref).

Local Coordinate: Vector 2D Structure.  

## Outputs

Return Value: Vector 2D Structure. Absolute coordinates.
