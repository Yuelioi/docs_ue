# SetOwnerPlayer

设置拥有这个部件组件的本地玩家。设置拥有的玩家控制。在分屏的情况下，该部件出现在哪个玩家的视口。此外，它将拥有的玩家转发到实际生成的UserWidget上。

目标是小工具组件
