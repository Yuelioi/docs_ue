# GetNodeContainerType

返回定义节点目的的节点容器类型（工厂节点、翻译的场景节点或翻译的资产节点）。

目标是互换基础节点

## 图示

![](/uploads/projects/ue-bluprint/20221218-19361565.png)

## Inputs

目标：互换基础节点对象参考。

## Outputs

返回值。EInterchangeNodeContainerType 枚举。返回定义节点目的的节点容器类型（工厂节点、翻译的场景节点或翻译的资产节点）。
