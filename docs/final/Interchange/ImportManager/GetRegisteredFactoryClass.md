<hr>

Script helper to get a registered factory for a specified class

Target is Interchange Manager

## 图示

![](/uploads/projects/ue-bluprint/20221218-19311481.png)

## Inputs

Target: Interchange Manager Object Reference.

Class to Make: Object Class Reference.

## Outputs

Return Value: Object Class Reference. : if found, we return the factory class that is registered. Return NULL if nothing found..
