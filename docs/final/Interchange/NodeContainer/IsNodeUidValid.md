# IsNodeUidValid

如果节点的唯一 ID 在容器中存在，则返回 true。

目标是互换基础节点容器

## 图示

![](/uploads/projects/ue-bluprint/20221218-19355458.png)

## Inputs

目标：互换基础节点容器对象参考。

节点唯一 ID：字符串。

## Outputs

返回值。布尔值。如果该节点的唯一 ID 存在于容器中，则返回 true。
