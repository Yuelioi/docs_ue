# GetVertexPairEdge

如果有的话，返回由两个给定顶点ID定义的边缘ID；否则，返回INDEX_NONE

目标是网状描述基

## Inputs

## Outputs

返回值。边缘ID结构。如果有的话，返回由两个给定顶点ID定义的边缘ID；否则返回INDEX_NONE。

Returns the edge ID defined by the two given vertex IDs, if there is one; otherwise INDEX_NONE

Target is Mesh Description Base
