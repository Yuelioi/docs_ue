# GetVertexInstancePairEdge

如果有的话，返回由两个给定顶点实例ID定义的边缘ID；否则，返回INDEX_NONE

目标是网格描述基

## Inputs

目标。网格描述 基本对象参考。

顶点实例ID0：顶点实例ID结构。

顶点实例ID1：顶点实例ID结构。  

## Outputs

返回值。边缘ID结构。如果有的话，返回由两个给定顶点实例ID定义的边缘ID；否则返回INDEX_NONE。

Returns the edge ID defined by the two given vertex instance IDs, if there is one; otherwise INDEX_NONE
