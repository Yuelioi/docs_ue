# GetVertexInstancePairEdge

如果有的话，返回由两个给定顶点实例 ID 定义的边缘 ID；否则，返回 INDEX_NONE

目标是网格描述基

## 图示

![](/uploads/projects/ue-bluprint/20221218-20043009.png)

## Inputs

目标。网格描述 基本对象参考。

顶点实例 ID0：顶点实例 ID 结构。

顶点实例 ID1：顶点实例 ID 结构。

## Outputs

返回值。边缘 ID 结构。如果有的话，返回由两个给定顶点实例 ID 定义的边缘 ID；否则返回 INDEX_NONE。

<hr>

Returns the edge ID defined by the two given vertex instance IDs, if there is one; otherwise INDEX_NONE

Target is Mesh Description Base

## 图示

![](/uploads/projects/ue-bluprint/20221218-20043009.png)

## Inputs

Target: Mesh Description Base Object Reference.

Vertex Instance ID0: Vertex Instance ID Structure.

Vertex Instance ID1: Vertex Instance ID Structure.

## Outputs

Return Value: Edge ID Structure. Returns the edge ID defined by the two given vertex instance IDs, if there is one; otherwise INDEX_NONE.
