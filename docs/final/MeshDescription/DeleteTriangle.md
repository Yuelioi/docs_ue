# DeleteTriangle

从网格中删除一个三角形

目标是网状描述基

## 图示

![](/uploads/projects/ue-bluprint/20221218-20033238.png)

## Inputs

在。执行。

目标。网格描述 基本对象参考。

三角形 ID：三角形 ID 结构。

## Outputs

出：执行。

孤立的边缘。边缘 ID 结构的阵列。

孤立的顶点实例。顶点实例 ID 结构的数组。

孤立的多边形组 Ptr。多边形组 ID 的数组结构。
