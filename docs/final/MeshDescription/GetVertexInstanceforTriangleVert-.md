# GetVertexInstanceforTriangleVert-

返回与给定三角形上的给定顶点相对应的顶点实例，或者 INDEX_NONE。

目标是网状描述基

## 图示

![](/uploads/projects/ue-bluprint/20221218-20042892.png)

## Inputs

目标。网格描述 基本对象参考。

三角形 ID：三角形 ID 结构。

顶点 ID：顶点 ID 结构。

## Outputs

返回值。顶点实例 ID 结构。返回与给定三角形上的给定顶点相对应的顶点实例，或者 INDEX_NONE。

<hr>

Return the vertex instance which corresponds to the given vertex on the given triangle, or INDEX_NONE

Target is Mesh Description Base

## 图示

![](/uploads/projects/ue-bluprint/20221218-20042892.png)

## Inputs

Target: Mesh Description Base Object Reference.

Triangle ID: Triangle ID Structure.

Vertex ID: Vertex ID Structure.

## Outputs

Return Value: Vertex Instance ID Structure. Return the vertex instance which corresponds to the given vertex on the given triangle, or INDEX_NONE.
