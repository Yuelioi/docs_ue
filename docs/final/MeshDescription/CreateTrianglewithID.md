# CreateTrianglewithID

以给定的ID为网格添加一个新的三角形。这也会产生一个封装的多边形，以及任何缺失的边。

目标是网状描述基

## 图示

![]($-20221218-20032326.png)

## Inputs

在。执行。

目标。网格描述 基本对象参考。

三角形ID：三角形ID结构。

多边形组ID。多边形组ID结构。

顶点实例ID。顶点实例ID结构的数组。  

## Outputs

出：执行。

新的边缘ID。边缘ID结构的数组。

Adds a new triangle to the mesh with the given ID. This will also make an encapsulating polygon, and any missing edges.

Target is Mesh Description Base

## 图示

![]($-20221218-20032326.png)

## Inputs

In: Exec.

Target: Mesh Description Base Object Reference.

Triangle ID: Triangle ID Structure.

Polygon Group ID: Polygon Group ID Structure.

Vertex Instance IDs: Array of Vertex Instance ID Structures.  

## Outputs

Out: Exec.

New Edge IDs: Array of Edge ID Structures.
