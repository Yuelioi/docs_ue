返回与给定多边形上的给定顶点相对应的顶点实例，或 INDEX_NONE。

目标是网状描述基

返回值。顶点实例ID结构。返回与给定多边形上的给定顶点相对应的顶点实例，或者 INDEX_NONE。

Return the vertex instance which corresponds to the given vertex on the given polygon, or INDEX_NONE
