## 图示

![](/uploads/projects/ue-bluprint/20221218-20033011.png)

## Inputs

在。执行。

目标。网格描述 基本对象参考。

## Outputs

出：执行。

孤立的边缘。边缘 ID 结构的阵列。

孤立的多边形组。多边形组 ID 结构的数组。
