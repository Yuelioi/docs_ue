# EnableMotionTrackingOfSource

启用对指定控制器的追踪，按玩家索引和被追踪设备类型。

目标是运动跟踪设备功能库

## Inputs

在。执行。

球员指数。整数。(in) 球员的索引。

来源名称。名称。(in) 设备ID.

## Outputs

出：执行。

返回值。Boolean. (布尔值) true，如果指定的设备现在被设定为跟踪。这可能是由于跟踪限制，或在无效的输入时失败。
