## Actions

Mouse Wheel Axis. Returns the current value of input axis key Mouse Wheel Axis. If input is disabled for the actor the value will be 0..

Mouse X. Returns the current value of input axis key Mouse X. If input is disabled for the actor the value will be 0..

Mouse Y. Returns the current value of input axis key Mouse Y. If input is disabled for the actor the value will be 0..
