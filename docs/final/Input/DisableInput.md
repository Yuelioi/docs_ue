# DisableInput

将此角色从被 PlayerController 处理的输入栈中移除。

目标是演员

## 图示

![](/uploads/projects/ue-bluprint/20221218-19221035.png)

## Inputs

在。执行。

目标。演员对象参考。

播放器控制器。播放器控制器对象参考。我们不想再接收其输入事件的 PlayerController。如果为空，这个角色将停止接收所有 PlayerController 的输入。

出：执行。

Removes this actor from the stack of input being handled by a PlayerController.

Target is Actor

## 图示

![](/uploads/projects/ue-bluprint/20221218-19221035.png)

## Inputs

In: Exec.

Target: Actor Object Reference.

Player Controller: Player Controller Object Reference. The PlayerController whose input events we no longer want to receive. If null, this actor will stop receiving input from all PlayerControllers..

Out: Exec.
