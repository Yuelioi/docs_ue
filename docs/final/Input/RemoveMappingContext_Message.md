# RemoveMappingContext_Message

移除一个特定的控件上下文。即使没有应用上下文，调用这个也是安全的。

目标是增强型输入子系统接口

## 图示

![](/uploads/projects/ue-bluprint/20221218-19292110.png)

## Inputs

在。执行。

目标。对象参考。

映射语境。输入映射语境对象参考。要从播放器中删除的上下文。

选项。修改上下文选项结构（通过引用）。移除该输入映射上下文时需要考虑的选项。

## Outputs

出：执行。

Remove a specific control context.. This is safe to call even if the context is not applied.

Target is Enhanced Input Subsystem Interface

## 图示

![](/uploads/projects/ue-bluprint/20221218-19292110.png)

## Inputs

In: Exec.

Target: Object Reference.

Mapping Context: Input Mapping Context Object Reference. Context to remove from the player.

Options: Modify Context Options Structure (by ref). Options to consider when removing this input mapping context.

## Outputs

Out: Exec.
