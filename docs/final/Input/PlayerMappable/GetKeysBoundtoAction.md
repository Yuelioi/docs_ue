# GetKeysBoundtoAction

返回该映射配置中映射到特定输入动作的所有按键。

目标是玩家可绑定的输入配置

![](/uploads/projects/ue-bluprint/20221218-19285395.png)

## Inputs

目标。玩家可绑定的输入配置对象参考。

Returns all the keys mapped to a specific Input Action in this mapping config.

Target is Player Bindable Input Config

## 图示

![](/uploads/projects/ue-bluprint/20221218-19285395.png)

## Inputs

Target: Player Bindable Input Config Object Reference.

In Action: Input Action Object Reference.

## Outputs

Return Value: Array of Enhanced Action Key Mapping Structures. Returns all the keys mapped to a specific Input Action in this mapping config..
