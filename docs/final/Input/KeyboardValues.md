## Actions

触摸板按钮 X 轴。返回输入轴键 Touchpad Button X Axis 的当前值。如果该演员的输入被禁用，其值将是 0。

Touchpad Button Y Axis.返回输入轴键 Touchpad Button Y Axis 的当前值。如果该演员的输入被禁用，其值将为 0。
