# BreakInputActionValue

将一个 ActionValue 分成 X、Y、Z 三部分，价值类型不支持的轴将为 0。

目标是增强型输入库

## 图示

![](/uploads/projects/ue-bluprint/20221218-19171455.png)

## Inputs

在行动值。输入行动值结构。

## Outputs

X：浮点数（单精度）。

Y：浮点数（单精度）。

Z：浮点数（单精度）。
