# GetLastScreenSpacePosition

返回上次处理输入事件时光标在屏幕空间中的位置。

目标是 Kismet 输入库

## 图示

![](/uploads/projects/ue-bluprint/20221218-19290994.png)

## Inputs

输入。指针式事件结构（通过引用）。

## Outputs

返回值。矢量 2D 结构。返回上次处理输入事件时光标在屏幕空间中的位置。

<hr>

Returns the position of the cursor in screen space last time we handled an input event

Target is Kismet Input Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-19290994.png)

## Inputs

Input: Pointer Event Structure (by ref).

## Outputs

Return Value: Vector 2D Structure. Returns the position of the cursor in screen space last time we handled an input event.
