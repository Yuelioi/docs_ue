# GetWheelDelta

目标是 Kismet 输入库

## 图示

![](/uploads/projects/ue-bluprint/20221218-19291556.png)

## Inputs

输入。指针式事件结构（通过引用）。

## Outputs

返回值。Float（单精度）。自上次鼠标事件以来，鼠标滚轮转动了多少。
