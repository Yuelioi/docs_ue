# GetPointerIndex

返回指针的唯一标识符（例如，手指索引）。

目标是 Kismet 输入库

## 图示

![](/uploads/projects/ue-bluprint/20221218-19291101.png)

## Inputs

输入。指针式事件结构（通过引用）。

## Outputs

返回值。整数。返回指针的唯一标识符（例如，手指索引）。

## Inputs

Input: Pointer Event Structure (by ref).

## Outputs

Return Value: Integer. Returns the unique identifier of the pointer (e.g., finger index).
