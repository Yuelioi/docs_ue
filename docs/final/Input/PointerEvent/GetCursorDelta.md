# GetCursorDelta

返回自上次事件处理后鼠标移动的距离。

目标是 Kismet 输入库

## 图示

![](/uploads/projects/ue-bluprint/20221218-19290661.png)

## Inputs

输入。指针式事件结构（通过引用）。

## Outputs

返回值。矢量 2D 结构。返回自上次事件处理后鼠标移动的距离。

<hr>

Returns the distance the mouse traveled since the last event was handled.

Target is Kismet Input Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-19290661.png)

## Inputs

Input: Pointer Event Structure (by ref).

## Outputs

Return Value: Vector 2D Structure. Returns the distance the mouse traveled since the last event was handled..
