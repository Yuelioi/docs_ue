# GetCursorDelta

返回自上次事件处理后鼠标移动的距离。

目标是Kismet输入库

## 图示

![]($-20221218-19290661.png)

## Inputs

输入。指针式事件结构（通过引用）。  

## Outputs

返回值。矢量2D结构。返回自上次事件处理后鼠标移动的距离。

Returns the distance the mouse traveled since the last event was handled.

Target is Kismet Input Library
