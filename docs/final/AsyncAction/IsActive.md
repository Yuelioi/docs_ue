# IsActive

如果这个动作仍然处于活动状态，并且没有完成或被取消，则返回 true。

目标是可取消的异步行动

![](/uploads/projects/ue-bluprint/20221218-18015835.png)

## Inputs

目标。Cancellable Async Action 对象参考。

## Outputs

返回值。布尔值。如果这个动作仍在活动，没有完成或被取消，则返回 true。
