# NamespacedSubpointName

对给定的子点名称进行命名。不检查该子点是否存在。

目标是校准点

## 图示

![]($-20221218-18125864.png)

## Inputs

目标。校准点对象参考。

在子点名称中。字符串。命名空间的子点的名称。 

## Outputs

Out Namespaced Name: 字符串。输出命名空间的子点名称。

返回值。布尔值。如果成功则为真。

Namespaces the given subpoint name. Does not check that the subpoint exists.

Target is Calibration Point

## 图示

![]($-20221218-18125864.png)

## Inputs

Target: Calibration Point Object Reference.

In Subpoint Name: String. Name of the subpoint to namespace.  

## Outputs

Out Namespaced Name: String. The output namespaced subpoint name.

Return Value: Boolean. True if successful.
