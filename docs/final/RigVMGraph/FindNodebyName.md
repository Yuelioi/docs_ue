# FindNodebyName

返回一个给定名称的 Node（或 nullptr）。

目标是 Rig VMGraph

![](/uploads/projects/ue-bluprint/20221218-20441322.png)

目标。Rig VMGraph 对象参考。

返回值。Rig VMNode 对象参考。返回一个节点的名称（或 nullptr）。
