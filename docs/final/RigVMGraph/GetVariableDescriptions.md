# GetVariableDescriptions

返回该图形中唯一的变量描述的列表。多个变量节点可以共享相同的描述。

目标是 Rig VMGraph

## 图示

![](/uploads/projects/ue-bluprint/20221218-20443689.png)

## Inputs

目标。Rig VMGraph 对象参考。

## Outputs

返回值。Rig VMGraph 变量描述结构的数组。返回该图形中唯一的变量描述的列表。多个变量节点可以共享相同的描述。

<hr>

Returns a list of unique Variable descriptions within this Graph.. Multiple Variable Nodes can share the same description.

Target is Rig VMGraph

## 图示

![](/uploads/projects/ue-bluprint/20221218-20443689.png)

## Inputs

Target: Rig VMGraph Object Reference.

## Outputs

Return Value: Array of Rig VMGraph Variable Description Structures. Returns a list of unique Variable descriptions within this Graph.. Multiple Variable Nodes can share the same description..
