# Asin_Radians

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19555654.png)

## Inputs

A：浮点数（双精度）。

## Outputs

返回值。浮点数（双精度）。返回 A 的反正弦（arcsin）（结果为弧度）。

## Inputs

A: Float (double-precision).

## Outputs

Return Value: Float (double-precision). Returns the inverse sine (arcsin) of A (result is in Radians).
