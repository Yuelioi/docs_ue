# Asin_Degrees

返回 A 的逆正弦（arcsin）（结果为度数）。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19555518.png)

## Inputs

A：浮点数（双精度）。

## Outputs

返回值。浮点数（双精度）。返回 A 的反正弦（arcsin）（结果为度数）。

<hr>

Returns the inverse sin (arcsin) of A (result is in Degrees)

Target is Kismet Math Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-19555518.png)

## Inputs

A: Float (double-precision).

## Outputs

Return Value: Float (double-precision). Returns the inverse sin (arcsin) of A (result is in Degrees).
