# Atan2_Radians

返回 A/B 的反 tan（atan2）（结果为弧度）。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19555937.png)

## Inputs

Y：浮点数（双精度）。

X：浮点数（双精度）。

## Outputs

返回值。浮点数（双精度）。返回 A/B 的反 tan（atan2）（结果为弧度）。
