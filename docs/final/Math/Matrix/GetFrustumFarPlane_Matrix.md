# GetFrustumFarPlane_Matrix

得到这个矩阵的远平面的 Frustum。(假设矩阵代表一个视图投影矩阵)

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19515956.png)

## Inputs

M：矩阵结构（由参考文献）。

## Outputs

外层平面。平面结构。这个矩阵的 Frustum 的远平面。

返回值。布尔值。
