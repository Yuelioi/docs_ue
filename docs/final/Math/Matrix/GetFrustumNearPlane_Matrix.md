# GetFrustumNearPlane_Matrix

获取该矩阵的 Frustum 的近平面。(假设矩阵代表一个视图投影矩阵)

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19520218.png)

## Inputs

M：矩阵结构（由参考文献）。

## Outputs

外面的平面。平面结构。该矩阵的 Frustum 的近平面。

返回值。布尔值。
