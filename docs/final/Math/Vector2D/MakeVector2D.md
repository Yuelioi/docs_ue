# MakeVector2D

做出一个 2 维的矢量{X，Y}。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19584117.png)

## Inputs

X：浮点数（双精度）。

Y：浮点数（双精度）。

## Outputs

返回值。矢量 2D 结构。制作一个二维向量{X, Y}。
