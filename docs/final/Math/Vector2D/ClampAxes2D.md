# ClampAxes2D

创建一个此向量的副本，两个轴都被夹在给定的范围内。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19582407.png)

## Inputs

A：向量二维结构。

Min Axis Val: Float (double-precision).

Max Axis Val: Float (double-precision).

## Outputs

返回值。矢量 2D 结构。具有钳制轴的新向量。
