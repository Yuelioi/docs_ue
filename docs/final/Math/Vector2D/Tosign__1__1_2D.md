# Tosign**1**1_2D

获取一个只带符号的向量的副本。每个分量被设置为+1 或-1，零的符号被视为+1。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19585679.png)

## Inputs

A：向量二维结构。

## Outputs

返回值。矢量 2D 结构。矢量的副本，每个分量设置为+1 或-1。

<hr>

Get a copy of the vector as sign only.. Each component is set to +1 or -1, with the sign of zero treated as +1.

Target is Kismet Math Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-19585679.png)

## Inputs

A: Vector 2D Structure.

## Outputs

Return Value: Vector 2D Structure. A copy of the vector with each component set to +1 or -1.
