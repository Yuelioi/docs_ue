# VectorClampSize2D

创建这个矢量的副本，其二维幅度/大小/长度被夹在最小和最大之间。Z是不变的。

目标是Kismet数学图书馆

## Outputs

返回值。矢量。创建一个此向量的副本，其二维幅度/大小/长度被夹在最小和最大之间。Z是不变的。
