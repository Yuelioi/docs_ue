# VectorGetSignVector

获得一份仅有符号的向量的副本。每个分量被设置为+1 或-1，零的符号被视为+1。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19575339.png)

## Inputs

A：向量。向量的副本，每个分量设置为+1 或-1。

## Outputs

返回值。向量。

Target is Kismet Math Library

![](/uploads/projects/ue-bluprint/20221218-19575339.png)

## Inputs

A: Vector. copy of the vector with each component set to +1 or -1.

## Outputs

Return Value: Vector.
