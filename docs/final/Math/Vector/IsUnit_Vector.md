# IsUnit_Vector

判断向量是否在指定的平方公差内被归一化/单位（长度为 1）。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19570094.png)

## Inputs

答：向量（通过参考）。

平方长公差。Float（单精度）。

## Outputs

返回值。布尔值。如果是单位，则为真，否则为假。
