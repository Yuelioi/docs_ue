# VectorHeadingAngle

将一个方向矢量转换为 "航向 "角度。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19575469.png)

## Inputs

答：向量。

## Outputs

返回值。浮点数（双精度）。在+/-PI 弧度之间的'航向'角度。0 是指向下+X。

Convert a direction vector into a 'heading' angle.

Target is Kismet Math Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-19575469.png)

## Inputs

A: Vector.

## Outputs

Return Value: Float (double-precision). 'Heading' angle between +/-PI radians. 0 is pointing down +X..
