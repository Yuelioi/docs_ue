# IsUniform_Vector

检查这个向量的所有分量是否相同，是否在一个公差范围内。

目标是Kismet数学图书馆

## Inputs

答：向量（通过参考）。

误差。Float (single-precision).误差容限.  

## Outputs

返回值。布尔值。如果向量在公差范围内相等，则为真，否则为假。
