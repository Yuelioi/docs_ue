# ClampVectorSize

创建这个矢量的副本，其幅度/大小/长度被夹在最小和最大之间。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19562549.png)

## Inputs

答：向量。

最小值。浮点数（双精度）。

最大：浮点数（双精度）。

## Outputs

返回值。矢量。创建此向量的副本，其幅度/大小/长度被夹在最小和最大之间。
