# VectorUnitCartesiantoSpherical

将笛卡尔的单位向量转换为单位球面上的球面坐标。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19581767.png)

## Inputs

答：向量。

## Outputs

返回值。矢量 2D 结构。输出 Theta 的范围是[0, PI]，输出 Phi 的范围是[-PI, PI]。

<hr>

Converts a Cartesian unit vector into spherical coordinates on the unit sphere.

Target is Kismet Math Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-19581767.png)

## Inputs

A: Vector.

## Outputs

Return Value: Vector 2D Structure. Output Theta will be in the range [0, PI], and output Phi will be in the range [-PI, PI]..
