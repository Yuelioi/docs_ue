# ProjectPointontoPlane

将一个点投射到由平面上的一个点和一个平面法线定义的平面上。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19571963.png)

## Inputs

点。向量。投射到平面上的点。

平面的基础。向量。平面上的一个点。

平面法线。矢量。平面的法线。

## Outputs

返回值。向量。投射到平面上的点。
