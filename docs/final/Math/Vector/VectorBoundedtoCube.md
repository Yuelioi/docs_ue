# VectorBoundedtoCube

获得这个矢量的一个副本，夹在以原点为中心的轴对齐的立方体内。

目标是Kismet数学图书馆

## 图示

![]($-20221218-19573666.png)

## Inputs

在Vect: 向量。

在半径。Float（单精度）。正方体的一半大小（或正方体中圆周的球体半径）。  

## Outputs

返回值。向量。此向量的副本，以立方体为界。

Get a copy of this vector, clamped inside of an axis aligned cube centered at the origin.

Target is Kismet Math Library

## 图示

![]($-20221218-19573666.png)

## Inputs

In Vect: Vector.

In Radius: Float (single-precision). Half size of the cube (or radius of sphere circumscribed in the cube)..  

## Outputs

Return Value: Vector. A copy of this vector, bound by cube..
