# GetSlopeDegreeAngles

根据以下信息返回坡度、俯仰和滚动角度，单位为度。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19503089.png)

## Inputs

我的右 Y 轴：矢量（通过参考）。站在斜坡上的演员的右（Y）方向单位矢量。

地面法线。矢量（通过参考）。地面法线（单位）矢量.

向量向上。向量（通过参考）。参考框架的向上矢量。

## Outputs

出坡俯仰度角度。浮点（单精度）。

Out Slope Roll Degree Angle:浮动（单精度）。
