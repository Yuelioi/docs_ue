# MakeRotfromX

构建一个只给定 XAxis 的旋转器。Y 和 Z 是未指定的，但将是正交的。XAxis 不需要被归一化。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19543485.png)

## Inputs

X：向量（通过参考）。

## Outputs

返回值。旋转器。只给定一个 XAxis，建立一个旋转器。Y 和 Z 是未指定的，但将是正交的。XAxis 不需要被归一化。

<hr>

Builds a rotator given only a XAxis. Y and Z are unspecified but will be orthonormal. XAxis need not be normalized.

Target is Kismet Math Library
