# MakeRotfromYX

构建一个具有给定 Y 和 X 轴的矩阵。Y 轴将保持固定，X 轴可以被最小化地改变以执行正交性。Z 将被计算出来。## ＃＃＃输入不需要归一化。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19543999.png)

## Inputs

Y：向量（通过参考）。

X：向量（通过参考）。

## Outputs

返回值。旋转器。以给定的 Y 轴和 X 轴建立一个矩阵。Y 轴将保持固定，X 轴可能会有最小的变化，以强制执行正交性。Z 将被计算出来。## ＃＃输入不需要归一化。
