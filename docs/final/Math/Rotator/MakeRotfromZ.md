# MakeRotfromZ

建立一个只给定 ZAxis 的旋转矩阵。X 和 Y 是未指定的，但将是正交的。ZAxis 不需要被规范化。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19544253.png)

## Inputs

Z：向量（通过参考）。

## Outputs

返回值。旋转器。建立一个旋转矩阵，只给定一个 ZAxis。X 和 Y 是未指定的，但将是正交的。ZAxis 不需要被规范化。

<hr>

Builds a rotation matrix given only a ZAxis. X and Y are unspecified but will be orthonormal. ZAxis need not be normalized.

Target is Kismet Math Library
