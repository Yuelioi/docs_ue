# CombineRotators

将 2 个旋转结合起来，得到的旋转结果是先应用 A，再应用 B。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19542182.png)

## Inputs

答：旋转器。

B：旋转器。

## Outputs

返回值。旋转器。结合 2 个旋转，给你首先应用 A，然后是 B 的旋转结果。

## Outputs
