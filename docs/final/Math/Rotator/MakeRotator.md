# MakeRotator

根据提供的旋转值（单位：度）制作一个旋转器{Roll, Pitch, Yaw}。

目标是Kismet数学图书馆

## Inputs

X（滚动）。浮点数（单精度）。

Y (Pitch):Float（单精度）。

Z（偏航）。浮点数（单精度）。  

## Outputs

返回值。旋转器。根据提供的旋转值（度）制作一个旋转器{Roll, Pitch, Yaw}。

Makes a rotator {Roll, Pitch, Yaw} from rotation values supplied in degrees

Target is Kismet Math Library
