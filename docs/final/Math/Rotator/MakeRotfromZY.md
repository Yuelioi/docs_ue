# MakeRotfromZY

以给定的 Z 轴和 Y 轴建立一个矩阵。Z 轴将保持固定，Y 轴可以被最小化地改变以强制执行正交。X 将被计算出来。## ＃＃＃输入不需要归一化。

目标是 Kismet 数学图书馆

## Inputs

Z：向量（通过参考）。

Y：向量（通过参考）。

## Outputs

返回值。旋转器。以给定的 Z 轴和 Y 轴建立一个矩阵。Z 轴将保持固定，Y 轴可能会有最小的变化，以强制执行正交性。X 将被计算出来。## ＃＃输入不需要规范化。
