# MakeRotfromXY

构建一个具有给定 X 和 Y 轴的矩阵。X 将保持固定，Y 可以被最小化地改变以执行正交性。Z 将被计算出来。## ＃＃＃输入不需要归一化。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19543613.png)

## Inputs

X：向量（通过参考）。

Y：向量（通过参考）。

## Outputs

返回值。旋转器。以给定的 X 和 Y 轴建立一个矩阵。X 轴将保持固定，Y 轴可能会有最小的变化以执行正交。Z 将被计算出来。## ＃＃输入不需要规范化。
