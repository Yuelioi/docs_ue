# MakeRotfromZX

以给定的Z轴和X轴建立一个矩阵。Z轴将保持固定，X轴可以被最小化地改变以强制执行正交性。Y将被计算出来。## ＃＃＃输入不需要归一化。

目标是Kismet数学图书馆
