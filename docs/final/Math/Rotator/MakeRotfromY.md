# MakeRotfromY

构建一个只给定 YA 轴的旋转矩阵。X 和 Z 是未指定的，但将是正交的。YAxis 不需要被归一化。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19543874.png)

## Inputs

Y：向量（通过参考）。

## Outputs

返回值。旋转器。建立一个旋转矩阵，只给定一个 YA 轴。X 和 Z 是未指定的，但将是正交的。YAxis 不需要被归一化。

<hr>

Builds a rotation matrix given only a YAxis. X and Z are unspecified but will be orthonormal. YAxis need not be normalized.

Target is Kismet Math Library
