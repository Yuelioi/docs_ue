# InverseTransformDirection

用所提供的变换的逆变换来变换一个方向向量--不会改变其长度。例如，如果 T 是一个物体的变换，这将把一个方向从世界空间变换到本地空间。

目标是 Kismet 数学图书馆

![](/uploads/projects/ue-bluprint/20221218-19553505.png)

## Inputs

T：转化（通过参考）。

方向。矢量。

## Outputs

返回值。向量。用所提供的变换的逆向来变换一个方向向量--不会改变其长度。例如，如果 T 是一个物体的变换，这将把一个方向从世界空间变换到本地空间。

<hr>

Transform a direction vector by the inverse of the supplied transform - will not change its length.. For example, if T was an object's transform, this would transform a direction from world space to local space.

Target is Kismet Math Library
