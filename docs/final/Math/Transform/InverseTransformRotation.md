# InverseTransformRotation

用所提供的变换的逆向来变换一个旋转器。例如，如果 T 是一个物体的变换，这将把一个旋转从世界空间变换到本地空间。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19553761.png)

## Inputs

T：转化（通过参考）。

旋转： 旋转器。

## Outputs

返回值。旋转器。用所提供的变换的逆向来变换一个旋转器。例如，如果 T 是一个物体的变换，这将把一个旋转从世界空间变换到本地空间。

<hr>

Transform a rotator by the inverse of the supplied transform.. For example, if T was an object's transform, this would transform a rotation from world space to local space.

Target is Kismet Math Library
