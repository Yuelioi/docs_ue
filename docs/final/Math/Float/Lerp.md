# Lerp

根据 Alpha 在 A 和 B 之间进行线性插值（Alpha=0 时为 A 的 100%，Alpha=1 时为 B 的 100%）。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19495448.png)

## Inputs

A：浮点数（双精度）。

B：浮点数（双精度）。

阿尔法。Float (double-precision).

## Outputs

返回值。浮点数（双精度）。根据 Alpha 在 A 和 B 之间进行线性插值（Alpha=0 时为 A 的 100%，Alpha=1 时为 B 的 100%）。

<hr>

Linearly interpolates between A and B based on Alpha (100% of A when Alpha=0 and 100% of B when Alpha=1)

Target is Kismet Math Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-19495448.png)

## Inputs

A: Float (double-precision).

B: Float (double-precision).

Alpha: Float (double-precision).

## Outputs

Return Value: Float (double-precision). Linearly interpolates between A and B based on Alpha (100% of A when Alpha=0 and 100% of B when Alpha=1).
