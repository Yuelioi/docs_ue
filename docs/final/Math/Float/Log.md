# Log

返回 A 基 B 的对数（如果 B^R == A，则返回 R）。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19495580.png)

## Inputs

A：浮点数（双精度）。

基数。浮点数（双精度）。

## Outputs

返回值。浮点数（双精度）。返回 A 基 B 的对数（如果 B^R == A，则返回 R）。
