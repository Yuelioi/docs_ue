# Loge

返回 A 的自然对数（如果 e^R == A，返回 R）。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19495708.png)

## Inputs

A：浮点数（双精度）。

## Outputs

返回值。浮点数（双精度）。返回 A 的自然对数（如果 e^R == A，返回 R）。

<hr>

Returns natural log of A (if e^R == A, returns R)

Target is Kismet Math Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-19495708.png)

## Inputs

A: Float (double-precision).

## Outputs

Return Value: Float (double-precision). Returns natural log of A (if e^R == A, returns R).
