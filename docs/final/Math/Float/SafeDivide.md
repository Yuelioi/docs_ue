# SafeDivide

如果 B（分母）为零，该函数返回 0。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19501666.png)

## Inputs

A：浮点数（双精度）。

B：浮点数（双精度）。

## Outputs

返回值。浮点数（双精度）。如果 B（分母）为零，该函数返回 0。
