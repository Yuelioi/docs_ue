# MinOfFloatArray

返回所有数组条目的最小值以及找到该条目的索引。如果提供的数组是空的，则返回值为 0，索引为-1。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19500620.png)

## Inputs

Float Array:Float（单精度）的数组。

## Outputs

最小值的索引。整数。

最小值。Float（单精度）。
