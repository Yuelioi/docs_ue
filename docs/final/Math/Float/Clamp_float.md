# Clamp_float

返回夹在 A 和 B 之间的值（包括）。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19494000.png)

## Inputs

值。浮点数（双精度）。

最小值。浮点数（双精度）。

最大：浮点数（双精度）。

## Outputs

返回值。浮点数（双精度）。返回夹在 A 和 B（包括）之间的值。

<hr>

Returns Value clamped between A and B (inclusive)

Target is Kismet Math Library

![](/uploads/projects/ue-bluprint/20221218-19494000.png)

Value: Float (double-precision).

Return Value: Float (double-precision). Returns Value clamped between A and B (inclusive).
