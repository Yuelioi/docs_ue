# Sign_float

Sign (float, 如果 A<0，返回-1；如果 A 为 0，返回 0；如果 A>0，返回+1)

目标是 Kismet 数学图书馆

![](/uploads/projects/ue-bluprint/20221218-19501923.png)

## Inputs

A：浮点数（双精度）。

## Outputs

返回值。浮点数（双精度）。符号（浮点数，如果 A<0 则返回-1，如果 A 为 0 则返回 0，如果 A>0 则返回+1）。
