# NextSobolFloat

目标是重要性取样库

## 图示

![](/uploads/projects/ue-bluprint/20221218-19533052.png)

## Inputs

索引。整数。哪个顺序点。

尺寸。整数。哪个 Sobol 维度（0 到 15）。

前一个值。Float（单精度）。索引-1 的 Sobol 值。

## Outputs

返回值。浮点数（单精度）。Sobol 分布的随机数，在 0 和 1 之间。
