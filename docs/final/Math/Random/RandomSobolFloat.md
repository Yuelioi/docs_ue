# RandomSobolFloat

目标是重要性取样库

## 图示

![](/uploads/projects/ue-bluprint/20221218-19535943.png)

## Inputs

索引。整数。哪个顺序点。

尺寸。整数。哪个 Sobol 维度（0 到 15）。

种子：浮点数（单精度）。随机种子（范围为 0-1），在多个序列中随机化。

## Outputs

返回值。浮点数（单精度）。Sobol 分布的随机数，在 0 和 1 之间。

## Inputs

Index: Integer. Which sequential point.

Dimension: Integer. Which Sobol dimension (0 to 15).

Return Value: Float (single-precision). Sobol-distributed random number between 0 and 1.
