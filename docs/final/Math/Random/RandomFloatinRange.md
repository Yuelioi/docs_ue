# RandomFloatinRange

在最小和最大之间生成一个随机数

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19534095.png)

## Inputs

最小值。浮点数（双精度）。

最大：浮点数（双精度）。

## Outputs

返回值。浮点数（双精度）。在 Min 和 Max 之间生成一个随机数。
