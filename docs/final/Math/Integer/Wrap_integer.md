# Wrap_integer

返回 A 和 B 之间的值（包括在内），该值环绕着

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19505695.png)

## Inputs

值。整数。

最小：整数。

最大：整数。

## Outputs

返回值。整数。返回 A 和 B 之间的值（包括在内），并将其包裹起来。
