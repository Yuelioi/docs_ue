# Luminance_LinearColor

返回显示器上颜色的感知亮度，考虑到每个颜色通道对人眼的影响：绿>红>蓝。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19474748.png)

## Inputs

在颜色：线性颜色结构。

## Outputs

返回值。Float（单精度）。考虑到每个颜色通道对人眼的影响，返回显示器上颜色的感知亮度：绿>红>蓝。

<hr>

Returns the perceived brightness of a color on a display taking into account the impact on the human eye per color channel: green > red > blue.

Target is Kismet Math Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-19474748.png)

## Inputs

In Color: Linear Color Structure.

## Outputs

Return Value: Float (single-precision). Returns the perceived brightness of a color on a display taking into account the impact on the human eye per color channel: green > red > blue..
