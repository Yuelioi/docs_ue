# NearEqual_LinearColor

如果线性颜色 A 等于线性颜色 B（A==B），在指定的误差容限内，返回真。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19475170.png)

## Inputs

答：线性色彩结构。

B：线性色彩结构。

误差。Float (single-precision).

## Outputs

返回值。布尔值。如果线性颜色 A 等于线性颜色 B（A == B），在指定的误差范围内，返回真。

<hr>

Returns true if linear color A is equal to linear color B (A == B) within a specified error tolerance

Target is Kismet Math Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-19475170.png)

## Inputs

A: Linear Color Structure.

B: Linear Color Structure.

Tolerance: Float (single-precision).

## Outputs

Return Value: Boolean. Returns true if linear color A is equal to linear color B (A == B) within a specified error tolerance.
