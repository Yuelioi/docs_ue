# Quantizewithroundingto8_bit_Line-

用四舍五入对线性颜色进行量化，并将结果作为8位颜色返回。这绕过了SRGB转换。

目标是Kismet数学图书馆

## Outputs

返回值。颜色结构。对线性颜色进行四舍五入的量化，并将结果作为8位颜色返回。这绕过了SRGB转换。
