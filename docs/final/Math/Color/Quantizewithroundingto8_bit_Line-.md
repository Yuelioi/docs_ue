# Quantizewithroundingto8_bit_Line-

用四舍五入对线性颜色进行量化，并将结果作为 8 位颜色返回。这绕过了 SRGB 转换。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19475382.png)

## Inputs

在颜色：线性颜色结构。

## Outputs

返回值。颜色结构。对线性颜色进行四舍五入的量化，并将结果作为 8 位颜色返回。这绕过了 SRGB 转换。

<hr>

Quantizes the linear color with rounding and returns the result as an 8-bit color. This bypasses the SRGB conversion.

Target is Kismet Math Library

In Color: Linear Color Structure.

## Outputs
