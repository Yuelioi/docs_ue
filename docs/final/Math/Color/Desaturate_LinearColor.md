# Desaturate_LinearColor

返回一个去饱和度的颜色，0表示没有去饱和度，1==完全去饱和度。

目标是Kismet数学图书馆

在颜色：线性颜色结构。

在去饱和度。Float (single-precision).  

## Outputs

返回值。线性颜色结构。不饱和的颜色。

Returns a desaturated color, with 0 meaning no desaturation and 1 == full desaturation
