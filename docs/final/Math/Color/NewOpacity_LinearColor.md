# NewOpacity_LinearColor

使用指定的不透明度/alpha 返回此颜色的副本。

目标是 Kismet 数学图书馆

## 图示

![](/uploads/projects/ue-bluprint/20221218-19475279.png)

## Inputs

在颜色：线性颜色结构。

在不透明度中。Float（单精度）。

## Outputs

返回值。线性颜色结构。使用指定的不透明度/阿尔法返回此颜色的副本。
