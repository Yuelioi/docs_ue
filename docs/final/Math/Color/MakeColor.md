# MakeColor

从单独的颜色成分制作颜色（RGB空间）。

目标是Kismet数学图书馆

R：浮点数（单精度）。

G：浮点数（单精度）。

B：浮点数（单精度）。

A：浮点数（单精度）。  

## Outputs

返回值。线性颜色结构。从单独的颜色成分（RGB空间）制作一个颜色。
