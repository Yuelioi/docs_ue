# MakeQuat

组成一个四合院{X, Y, Z, W}。

目标是Kismet数学图书馆

## 图示

![]($-20221218-19530332.png)

## Inputs

X：浮点数（单精度）。

Y：浮点数（单精度）。

Z：浮点数（单精度）。

W：浮点数（单精度）。  

## Outputs

返回值。Quat结构。制作一个四元组{X, Y, Z, W}。

Makes a quat {X, Y, Z, W}

Target is Kismet Math Library
