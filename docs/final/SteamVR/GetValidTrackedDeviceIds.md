# GetValidTrackedDeviceIds

返回当前跟踪的设备ID的数组

目标是Steam VRF函数库

## 图示

![]($-20221218-21041735.png)

## Inputs

设备类型。ESteamVRTrackedDeviceType枚举。获取设备的哪一类（如控制器、跟踪设备）的设备ID。  

## Outputs

追踪的设备ID。整数的数组。(输出) 包含当前被追踪的每个设备的ID的数组。

Returns an array of the currently tracked device IDs

Target is Steam VRFunction Library

## 图示

![]($-20221218-21041735.png)

## Inputs

Device Type: ESteamVRTrackedDeviceType Enum. Which class of device (e.g. controller, tracking devices) to get Device Ids for.  

## Outputs

Out Tracked Device Ids: Array of Integers. (out) Array containing the ID of each device that's currently tracked.
