# AddModifier

在堆栈中添加一个具有给定名称的修改器。如果这个名字已经被使用，那么这个修改器将不会被添加。如果添加成功，返回创建的修改器。

目标是 VCAM 组件

## Inputs

在。执行。

目标。VCam 组件对象参考。

姓名：姓名

修改器类。VCam 修改器类参考。

## Outputs

出：执行。

创建的修改器。VCam 修改器对象参考。

成功。布尔值。在堆栈中添加一个具有给定名称的修改器。如果该名称已经被使用，那么该修改器将不会被添加。如果添加成功，返回创建的修改器。

Add a modifier to the stack with a given name.. If that name is already in use then the modifier will not be added.. Returns the created modifier if the Add succeeded

Target is VCam Component
