# GetScreenshotInfo

返回与屏幕截图相关的信息。

目标是虚拟摄像机播放器控制器底座

![]($-20221218-21292181.png)

## Inputs

目标。虚拟摄像机播放器控制器基础对象参考。

屏幕截图名称。字符串。要检索的屏幕截图的名称。  

## Outputs

输出屏幕截图信息。虚拟相机截图结构。返回后，将保存截图信息。
