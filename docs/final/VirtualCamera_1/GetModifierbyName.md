# GetModifierbyName

试图在堆栈中找到一个具有给定名称的修改器。返回的修改器在使用前必须经过检查。

目标是VCAM组件

![]($-20221218-21281841.png)

## Inputs

目标。VCam组件对象参考。

姓名：姓名  

## Outputs

返回值。VCam Modifier对象参考。试图在堆栈中找到一个具有给定名称的修改器。返回的修改器在使用前必须经过检查。

Tries to find a Modifier in the Stack with the given name.. The returned Modifier must be checked before it is used.

Target is VCam Component
