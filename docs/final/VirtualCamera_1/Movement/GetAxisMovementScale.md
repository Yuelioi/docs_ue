# GetAxisMovementScale

获取一个特定轴的运动刻度。

目标是虚拟摄像机播放器控制器底座

## 图示

![](/uploads/projects/ue-bluprint/20221218-21285321.png)

## Inputs

目标。虚拟摄像机播放器控制器基础对象参考。

要检索的轴。EVirtualCameraAxis 枚举。需要的刻度值的轴。

## Outputs

返回值。Float (single-precision).给定轴的运动刻度。

Gets movement scale for a specific axis.

Target is Virtual Camera Player Controller Base

## 图示

![](/uploads/projects/ue-bluprint/20221218-21285321.png)

## Inputs

Target: Virtual Camera Player Controller Base Object Reference.

Axis to Retrieve: EVirtualCameraAxis Enum. The axis of the scale value needed.

## Outputs

Return Value: Float (single-precision). the movement scale for the given axis.
