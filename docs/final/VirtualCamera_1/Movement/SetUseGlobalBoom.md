# SetUseGlobalBoom

设置在用操纵杆导航时是否应使用全局吊杆。

目标是虚拟摄像机播放器控制器底座

## 图示

![](/uploads/projects/ue-bluprint/20221218-21290114.png)

## Inputs

在。执行。

目标。虚拟摄像机播放器控制器基础对象参考。

Should Use Global Boom: 布尔值。真，使用全局 Z 轴，假，使用本地 Z 轴。

## Outputs

出：执行。
