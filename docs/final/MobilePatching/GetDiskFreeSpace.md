# GetDiskFreeSpace

获取安装内容的磁盘可用空间（兆字节）。

目标是移动安装的内容

## 图示

![](/uploads/projects/ue-bluprint/20221218-20061321.png)

## Inputs

目标。移动安装的内容对象参考。

## Outputs

返回值。Float (single-precision).获取安装内容的磁盘可用空间，单位为兆字节。
