# GetTotalDownloadedSize

获得以兆字节为单位的总下载大小。在安装期间有效

目标是流动的待定内容

## 图示

![](/uploads/projects/ue-bluprint/20221218-20062587.png)

## Inputs

目标。Mobile Pending Content Object Reference。

## Outputs

返回值。Float (single-precision).获取下载的总大小，单位是兆字节。在安装期间有效。
