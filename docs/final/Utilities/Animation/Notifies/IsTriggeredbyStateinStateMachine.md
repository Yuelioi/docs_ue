# IsTriggeredbyStateinStateMachine

获取特定状态机中的某个特定状态是否触发了通知。

目标是 Anim Notify 状态机检查库

## 图示

![](/uploads/projects/ue-bluprint/20221218-12165191.png)

## Inputs

事件参考。Anim Notify Event Reference 结构（通过 ref）。要检查的事件。

Anim Instance:Anim Instance Object Reference.

状态机名称。名称。要测试的状态机的名称。

州名。名称。要测试的州的名称。

## Outputs

返回值。布尔值。

<hr>

Get whether a particular state in a specific state machine triggered the notify

Target is Anim Notify State Machine Inspection Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-12165191.png)

## Inputs

Event Reference: Anim Notify Event Reference Structure (by ref). The event to inspect.

Anim Instance: Anim Instance Object Reference.

State Machine Name: Name. The name of a state machine to test.

State Name: Name. The name of a state to test.

## Outputs

Return Value: Boolean.
