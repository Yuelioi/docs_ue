# GetDirectionBetweenSockets

计算两个骨头/插座之间的方向。

目标是Kismet动画库

## 图示

![]($-20221218-12164492.png)

## Inputs

组件。骨骼网组件对象参考。寻找插座/骨骼的骨架组件。

插座或骨骼名称来自。名称。第一个插座/骨的名称。

插座或骨骼名称 到。名称。第二个插座/骨的名称。  

## Outputs

返回值。向量。

Computes the direction between two bones / sockets.

Target is Kismet Animation Library

## 图示

![]($-20221218-12164492.png)

## Inputs

Component: Skeletal Mesh Component Object Reference. The skeletal component to look for the sockets / bones within.

Socket or Bone Name From: Name. The name of the first socket / bone.

Socket or Bone Name To: Name. The name of the second socket / bone.  

## Outputs

Return Value: Vector.
