# MakePerlinNoiseandRemap

这个函数为一个单一的浮点数创建perlin噪声，然后将范围映射到RangeOut中。

目标是Kismet动画库

## 图示

![]($-20221218-12164755.png)

## Inputs

值。Float（单精度）。噪声函数的输入值。

Range Out Min:浮点数（单精度）。输出范围的最小值。

范围输出最大值：浮点数（单精度）。输出范围的最大值。  

## Outputs

返回值。浮点数（单精度）。

This function creates perlin noise for a single float and then range map to RangeOut
