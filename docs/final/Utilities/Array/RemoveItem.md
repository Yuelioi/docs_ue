# RemoveItem

目标是 Kismet 阵列库

## 图示

![](/uploads/projects/ue-bluprint/20221218-12171536.png)

## Inputs

在。执行。

目标阵列。通配符阵列。要从\*中删除的数组。

项目。通配符（通过引用）。要从数组中删除的项目\*。

## Outputs

出：执行。

返回值。布尔值。如果一个或多个项目被删除，则为真。

Out: Exec.
