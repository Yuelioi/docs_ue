# GetGamepadControllerName

如果分配给游戏板，则返回控制器的名称（如果没有分配，则返回无）（仅Android和iOS）。

目标是Kismet系统库

## 图示

![]($-20221218-14284078.png)

## Inputs

控制器ID。整数。 

## Outputs

返回值。字符串。如果分配给游戏板，返回控制器的名称（如果没有分配，则返回无）（仅Android和iOS）。

Returns name of controller if assigned to a gamepad (or None if not assigned) (Android and iOS only)

Target is Kismet System Library

![]($-20221218-14284078.png)

## Inputs

Controller Id: Integer.  

## Outputs

Return Value: String. Returns name of controller if assigned to a gamepad (or None if not assigned) (Android and iOS only).
