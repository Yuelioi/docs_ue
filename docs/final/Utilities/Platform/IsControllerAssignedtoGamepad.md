# IsControllerAssignedtoGamepad

如果控制器 ID 分配给游戏手柄，则返回 true（仅限安卓和 iOS）。

目标是 Kismet 系统库

## 图示

![](/uploads/projects/ue-bluprint/20221218-14284508.png)

## Inputs

控制器 ID。整数。

## Outputs

返回值。布尔值。如果控制器 ID 分配给一个游戏手柄，则返回 true（仅 Android 和 iOS）。
