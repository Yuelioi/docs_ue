# BuildString_float

转换一个双数->字符串，以 AppendTo+Prefix+InDouble+Suffix 的形式创建一个新字符串。

目标是 Kismet 字符串库

## 图示

![](/uploads/projects/ue-bluprint/20221218-14291872.png)

## Inputs

附加到。字符串。一个现有的字符串，作为转换字符串的开始。

前缀。字符串。作为前缀的字符串，在 AppendTo 字符串之后使用。

In Double: Float（双精度）。要转换的双倍值。

后缀。字符串。后缀，用于附加到转换字符串的末尾。

## Outputs

返回值。字符串。由传递的参数建立的新字符串。

<hr>

Converts a double->string, create a new string in the form AppendTo+Prefix+InDouble+Suffix

Target is Kismet String Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-14291872.png)

## Inputs

Append To: String. An existing string to use as the start of the conversion string.

Prefix: String. A string to use as a prefix, after the AppendTo string.

In Double: Float (double-precision). The double value to convert.

Suffix: String. A suffix to append to the end of the conversion string.

## Outputs

Return Value: String. A new string built from the passed parameters.
