# StringToFloatVector

将字符串转换回浮点数向量。IsValid 表示字符串是否可以成功转换。

目标是 Kismet 字符串库

## Inputs

在字符串中。字符串。

## Outputs

Out Converted Vector: Vector 3f.

输出是否有效。Boolean（布尔）。
