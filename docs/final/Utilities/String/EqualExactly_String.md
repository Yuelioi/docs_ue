# EqualExactly_String

测试输入的字符串是否相等（A == B）。

目标是 Kismet 字符串库

## 图示

![](/uploads/projects/ue-bluprint/20221218-14293008.png)

## Inputs

A：字符串。要对比的字符串。

B：字符串。要比较的字符串。

## Outputs

返回值。布尔值。如果两个字符串相等，则为真，否则为假。

## 图示

A: String. The string to compare against.
