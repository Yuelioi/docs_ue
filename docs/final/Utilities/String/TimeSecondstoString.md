# TimeSecondstoString

将一个秒数转换成分钟:秒.毫秒的格式字符串（包括前导零）。

目标是 Kismet 字符串库

## 图示

![](/uploads/projects/ue-bluprint/20221218-14300239.png)

## Inputs

以秒为单位。Float（单精度）。

## Outputs

返回值。字符串。一个由秒数参数建立的新字符串。

In Seconds: Float (single-precision).

## Outputs
