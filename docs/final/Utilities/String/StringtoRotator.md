# StringtoRotator

将字符串转换回旋转器。IsValid 表示字符串是否可以成功转换。

目标是 Kismet 字符串库

## Inputs

在字符串中。字符串。

## Outputs

Out 转换后的 Rotator。旋转器。

Out Is Valid: 布尔值。
