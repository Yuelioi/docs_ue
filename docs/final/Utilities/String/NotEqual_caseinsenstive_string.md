# NotEqual_caseinsenstive_string

测试输入的字符串是否不相等（A != B），忽略大小写差异

目标是 Kismet 字符串库

## 图示

![](/uploads/projects/ue-bluprint/20221218-14294575.png)

## Inputs

A：字符串。要对比的字符串。

B：字符串。要比较的字符串。

## Outputs

返回值。布尔值。如果输入的字符串不相等，返回真，如果相等，返回假。

Test if the input string are not equal (A != B), ignoring case differences

Target is Kismet String Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-14294575.png)

## Inputs

A: String. The string to compare against.

B: String. The string to compare.

## Outputs

Return Value: Boolean. Returns true if the input strings are not equal, false if they are equal.
