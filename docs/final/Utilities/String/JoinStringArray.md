# JoinStringArray

将一个字符串数组串联成一个字符串。

目标是 Kismet 字符串库

## 图示

![](/uploads/projects/ue-bluprint/20221218-14293703.png)

## Inputs

源数组。字符串的数组。要串联的字符串数组。

Separator（分隔符）。字符串。用来分隔每个元素的字符串。

## Outputs

返回值。字符串。最后的、连接的、分离的字符串。
