# Append

将两个字符串串联起来，形成一个新的字符串

目标是 Kismet 字符串库

## 图示

![](/uploads/projects/ue-bluprint/20221218-14291699.png)

## Inputs

A：字符串。原始字符串。

B: 字符串。要追加到 A 的字符串。

## Outputs

返回值。字符串。一个新的字符串，是 A+B 的串联。
