# IsSamePath

检查两个路径是否相同。

目标是蓝图路径库

## 图示

![](/uploads/projects/ue-bluprint/20221218-14280807.png)

## Inputs

路径 A：字符串。第一个要检查的路径。

路径 B：字符串。第二个要检查的路径。

## Outputs

返回值。布尔值。如果两个路径相同为真。否则为假。
