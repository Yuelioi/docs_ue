# GetCurrentAssetGroupCulture

获取给定的资产组类别文化。注意：如果组别没有文化覆盖，则返回当前语言。

目标是Kismet国际化库

## 图示

![]($-20221218-14260921.png)

## Inputs

资产组。名称。要获取文化的资产组。 

## Outputs

返回值。字符串。作为IETF语言标签的文化（例如，"zh-Hans-CN"）。

Get the given asset group category culture.. Note: Returns the current language if the group category doesn't have a culture override.

Target is Kismet Internationalization Library

## 图示

![]($-20221218-14260921.png)

## Inputs

Asset Group: Name. The asset group to get the culture for..  

## Outputs

Return Value: String. The culture as an IETF language tag (eg, "zh-Hans-CN")..
