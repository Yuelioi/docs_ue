# IsStringTableRegistered

如果给定的表 ID 对应于一个已注册的字符串表，则返回 true。

目标是 Kismet 字符串表库

## Inputs

表的 ID。名称。

## Outputs

返回值。布尔值。如果给定的表 ID 对应于注册的字符串表，则返回真。
