# GetTimerElapsedTimebyHandle

返回给定句柄的已用时间（从当前倒计时开始的时间）。

目标是 Kismet 系统库

## 图示

![](/uploads/projects/ue-bluprint/20221218-15244387.png)

## Inputs

手柄。计时器句柄结构。计时器的句柄，以获得经过的时间。

## Outputs

返回值。Float（单精度）。从定时器的当前迭代开始，已经过去了多长时间。
