# GetTimerRemainingTimebyHandle

返回到定时器下次执行其句柄的时间。

目标是 Kismet 系统库

## 图示

![](/uploads/projects/ue-bluprint/20221218-15244563.png)

## Inputs

手柄。计时器手柄的结构。计时器的手柄是用来计算剩余时间的。

## Outputs

返回值。Float（单精度）。定时器的当前迭代还剩多长时间。
