# DoesTimerExistbyHandle

如果给定句柄的定时器存在，返回true，否则返回false。

目标是Kismet系统库

## Inputs

手柄。计时器句柄结构。检查其是否存在的句柄...  

## Outputs

返回值。布尔值。如果定时器存在，则为真。

Returns true is a timer for the given handle exists, false otherwise.

Target is Kismet System Library
