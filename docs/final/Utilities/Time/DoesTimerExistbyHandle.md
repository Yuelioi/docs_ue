# DoesTimerExistbyHandle

如果给定句柄的定时器存在，返回 true，否则返回 false。

目标是 Kismet 系统库

## 图示

![](/uploads/projects/ue-bluprint/20221218-15243784.png)

## Inputs

手柄。计时器句柄结构。检查其是否存在的句柄...

## Outputs

返回值。布尔值。如果定时器存在，则为真。
