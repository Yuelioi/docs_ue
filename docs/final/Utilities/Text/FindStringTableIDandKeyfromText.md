# FindStringTableIDandKeyfromText

试图找到给定文本所使用的字符串表 ID 和键。

目标是 Kismet 文本库

## 图示

![](/uploads/projects/ue-bluprint/20221218-15240674.png)

## Inputs

文本。文本。

## Outputs

出表标识。名称。

出键：字符串。

返回值。布尔值。如果找到字符串表的 ID 和键，则为真，否则为假。

<hr>

Attempts to find the String Table ID and key used by the given text.

Target is Kismet Text Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-15240674.png)

## Inputs

Text: Text.

## Outputs

Out Table Id: Name.

Out Key: String.

Return Value: Boolean. True if the String Table ID and key were found, false otherwise..
