# AsDateTime_fromUTC

将传入的日期和时间转换为文本，使用给定的时区（默认为本地时区）格式化为日期和时间。这将把给定的日期和时间从 UTC 转换成给定的时区（考虑到 DST）。

目标是 Kismet 文本库

## 图示

![](/uploads/projects/ue-bluprint/20221218-15235976.png)

## Inputs

在日期时间。日期时间结构（按参考）。

在时区：字符串。

## Outputs

返回值。文本。将传入的日期和时间转换为文本，格式为使用给定时区的日期和时间（默认为本地时区）。这将把给定的日期和时间从 UTC 转换成给定的时区（考虑到 DST）。

<hr>

Converts a passed in date & time to a text, formatted as a date & time using the given timezone (default is the local timezone). This will convert the given date & time from UTC to the given timezone (taking into account DST).

Target is Kismet Text Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-15235976.png)

## Inputs

In Date Time: Date Time Structure (by ref).

In Time Zone: String.

## Outputs

Return Value: Text. Converts a passed in date & time to a text, formatted as a date & time using the given timezone (default is the local timezone). This will convert the given date & time from UTC to the given timezone (taking into account DST)..
