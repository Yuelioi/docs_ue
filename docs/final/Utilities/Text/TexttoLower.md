# TexttoLower

以正确的文化方式将文本转换为小写。注意：返回的实例与原始实例相联系，如果活动文化发生变化，将被重新建立。

目标是 Kismet 文本库

![](/uploads/projects/ue-bluprint/20221218-15241882.png)

## Inputs

在文本中。文本（由参考文献）。

## Outputs

返回值。文本。以正确的文化方式将文本转换为小写。注释 返回的实例与原始实例相联系，如果活动文化发生变化，将被重新建立。
