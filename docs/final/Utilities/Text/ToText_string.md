# ToText_string

将字符串转换为文化不变的文本。使用 "格式化 "或 "制作字面文本 "来创建可本地化的文本

目标是 Kismet 文本库

## 图示

![](/uploads/projects/ue-bluprint/20221218-15243179.png)

## Inputs

在字符串中：字符串。

## Outputs

返回值。文本。将字符串转换为文化不变的文本。使用 Format 或 Make Literal Text 来创建可本地化的文本。
