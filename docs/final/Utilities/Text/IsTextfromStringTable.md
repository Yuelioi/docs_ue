# IsTextfromStringTable

如果给定的文本引用了一个字符串表，则返回真。

目标是 Kismet 文本库

## 图示

![](/uploads/projects/ue-bluprint/20221218-15241104.png)

## Inputs

文本。文本（由裁判）。

## Outputs

返回值。布尔值。如果给定的文本引用的是一个字符串表，则返回真。
