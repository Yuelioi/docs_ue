# TransformFrameTime

将指定的时间从一个帧率转换为另一个帧率。这对于在勾选分辨率和显示速率之间进行转换很有用。

目标是时间管理蓝图库

## Inputs

来源时间。框架时间结构（按参考）。

源率。帧率结构（按参考）。

目的地速率。帧率结构（按参考）。

## Outputs

返回值。帧时间结构。将指定的时间从一个帧速率转换为另一个帧速率。这对于在 tick 分辨率和显示速率之间的转换很有用。
