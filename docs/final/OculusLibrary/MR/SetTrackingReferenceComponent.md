Set the component for the OculusMR camera to track. If this is set to null, the camera will track the player pawn.

Target is Oculus MRFunction Library

In: Exec.

Component: Scene Component Object Reference.

## Outputs

Out: Exec.

Return Value: Boolean. Set the component for the OculusMR camera to track. If this is set to null, the camera will track the player pawn..
