# GetHandScale

获取手的刻度

目标是 Oculus 输入功能库

## 图示

![](/uploads/projects/ue-bluprint/20221218-20152844.png)

## Inputs

设备手。EOculusHandType 枚举。(in) 要获取比例的手。

控制器索引。整数。(in) 可选的不同控制器索引。

## Outputs

返回值。浮点数（单精度）。

Device Hand: EOculusHandType Enum. (in) The hand to get scale of.
