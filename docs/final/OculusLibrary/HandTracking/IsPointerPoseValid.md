# IsPointerPoseValid

检查指针姿势是否是一个有效的姿势

目标是Oculus输入功能库

## 图示

![]($-20221218-20153645.png)

## Inputs

设备手。EOculusHandType枚举。(in) 从哪只手获得指针状态。

控制器索引。整数。(in) 可选的不同控制器索引。  

## Outputs

返回值。布尔值。

Check if the pointer pose is a valid pose
