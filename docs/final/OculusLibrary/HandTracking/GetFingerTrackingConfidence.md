# GetFingerTrackingConfidence

获得手指的追踪信心

目标是 Oculus 输入功能库

## 图示

![](/uploads/projects/ue-bluprint/20221218-20152728.png)

## Inputs

设备手。EOculusHandType 枚举。(in) 要获得追踪信心的手。

手指。EOculusFinger Enum。(in) 要获得跟踪信心的手指。

控制器索引。整数。(in) 可选的不同控制器索引。

## Outputs

返回值。ETTrackingConfidence Enum。

Get the tracking confidence of a finger

Target is Oculus Input Function Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-20152728.png)

## Inputs

Device Hand: EOculusHandType Enum. (in) The hand to get tracking confidence of.

Finger: EOculusFinger Enum. (in) The finger to get tracking confidence of.

Controller Index: Integer. (in) Optional different controller index.

## Outputs

Return Value: ETrackingConfidence Enum.
