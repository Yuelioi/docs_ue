# IsOverFocusableWidget

如果命中结果下的小组件是可聚焦的，则返回 true。例如，Slate 小组件，对于 SupportedKeyboardFocus()，返回 true。

目标是小工具互动组件

目标。小工具交互组件对象参考。

## Outputs

返回值。布尔值。如果命中结果下的小部件是可聚焦的，则返回 true。例如，Slate 小部件，对于 SupportedKeyboardFocus()，返回 true。
