# DynamicPlayRateAdjustment

为给定的播放序列资产计算一个动态调整的播放率值

目标是动态播放率库

![](/uploads/projects/ue-bluprint/20221218-20073305.png)

## Inputs

语境。Anim 更新背景结构（按参考）。

轨迹。轨迹样本范围结构。

设置。动态播放率设置结构（按参考）。

序列。Anim Sequence Base Object Reference.

<hr>

Computes a dynamically adjusted play rate value for a given playing sequence asset

Target is Dynamic Play Rate Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-20073305.png)

## Inputs

Context: Anim Update Context Structure (by ref).

Trajectory: Trajectory Sample Range Structure.

Settings: Dynamic Play Rate Settings Structure (by ref).

Sequence: Anim Sequence Base Object Reference.

Accumulated Time: Float (single-precision).

Play Rate: Float (single-precision).

Looping: Boolean.

## Outputs

Return Value: Float (single-precision). Computes a dynamically adjusted play rate value for a given playing sequence asset.
