# FlattenTrajectory2D

移除轨迹范围内的 Z 轴运动贡献。

目标是运动轨迹蓝图库

## 图示

![](/uploads/projects/ue-bluprint/20221218-20073425.png)

## Inputs

轨迹。轨迹采样范围结构。输入轨迹范围。

保存速度：布尔值。输入布尔值，决定速度大小是保留（真）还是投影（假）。

## Outputs

返回值。轨迹采样范围结构。Z 轴扁平化的、经过修改的轨迹范围。

Removes the Z axis motion contribution from a trajectory range

Target is Motion Trajectory Blueprint Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-20073425.png)

## Inputs

Trajectory: Trajectory Sample Range Structure. Input trajectory range.

Preserve Speed: Boolean. Input bool determining if velocity magnitude is preserved (true) or projected (false).

## Outputs

Return Value: Trajectory Sample Range Structure. Z axis flattened, modified trajectory range.
