# DuplicateNormalizedElements

从给定的列表中复制任何可以复制的元素。注意：这个列表应该已经通过UTypedElementSelectionSet::GetNormalizedSelection或UTypedElementSelectionSet::GetNormalizedElementList预先规范化。
