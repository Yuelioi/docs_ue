# SetSelection

试图对给定的元素进行选择。注意：等同于先调用 ClearSelection 然后再调用 SelectElements，但在一个批次中发生。

目标是类型化的元素选择集

## 图示

![](/uploads/projects/ue-bluprint/20221218-21171549.png)

## Inputs

在。执行。

目标。类型化的元素选择集对象参考。

在元素句柄中。脚本类型的元素处理结构的数组。

在选择选项中。类型化的元素选择选项结构。

## Outputs

出：执行。

返回值。布尔值。如果选择被改变，则为真，否则为假。

<hr>

Attempt to make the selection the given elements.. Note: Equivalent to calling ClearSelection then SelectElements, but happens in a single batch.

Target is Typed Element Selection Set

## 图示

![](/uploads/projects/ue-bluprint/20221218-21171549.png)

## Inputs

In: Exec.

Target: Typed Element Selection Set Object Reference.

In Element Handles: Array of Script Typed Element Handle Structures.

In Selection Options: Typed Element Selection Options Structure.

## Outputs

Out: Exec.

Return Value: Boolean. True if the selection was changed, false otherwise..
