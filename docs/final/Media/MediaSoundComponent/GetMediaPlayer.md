# GetMediaPlayer

获取提供音频样本的媒体播放器。

目标是媒体声音组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-20021310.png)

## Inputs

目标。媒体声音组件对象参考。

## Outputs

返回值。媒体播放器对象参考。该组件的媒体播放器，如果没有设置，则为 nullptr。

Target: Media Sound Component Object Reference.
