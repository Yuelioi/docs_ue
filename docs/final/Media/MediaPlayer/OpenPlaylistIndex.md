在。执行。

目标。媒体播放器对象参考。

在播放列表中。媒体播放列表对象参考。要打开的播放列表...

Open a particular media source in the specified play list.

Target is Media Player

## 图示

![](/uploads/projects/ue-bluprint/20221218-20012578.png)

## Inputs

In: Exec.

Target: Media Player Object Reference.

In Playlist: Media Playlist Object Reference. The play list to open..

Index: Integer. The index of the source to open..

## Outputs

Out: Exec.

Return Value: Boolean. true if the source will be opened, false otherwise..
