# GetViewRotation

获取当前视图旋转（仅适用于 360 视频）。

目标是媒体播放器

## 图示

![](/uploads/projects/ue-bluprint/20221218-20010363.png)

## Inputs

目标。媒体播放器对象参考。

## Outputs

返回值。旋转器。查看旋转，如果没有，则为零旋转器。
