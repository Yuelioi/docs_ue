# SetViewField

设置视场（仅适用于 360 视频）。

目标是媒体播放器

## 图示

![](/uploads/projects/ue-bluprint/20221218-20015142.png)

## Inputs

在。执行。

目标。媒体播放器对象参考。

水平。Float（单精度）。水平视场（以欧拉度为单位）。

垂直。Float（单精度）。垂直视场（单位：欧拉度）。

绝对型：布尔型。

## Outputs

出：执行。

返回值。布尔值。成功时为真，否则为假。

Set the field of view (only for 360 videos).

Target is Media Player

## 图示

![](/uploads/projects/ue-bluprint/20221218-20015142.png)

## Inputs

In: Exec.

Target: Media Player Object Reference.

Horizontal: Float (single-precision). Horizontal field of view (in Euler degrees)..

Vertical: Float (single-precision). Vertical field of view (in Euler degrees)..

Absolute: Boolean.

## Outputs

Out: Exec.

Return Value: Boolean. true on success, false otherwise..
