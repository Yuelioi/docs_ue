# Play

启动媒体播放。

这与将播放率设置为 1.0 相同。

目标是 Bink 媒体播放器

## 图示

![](/uploads/projects/ue-bluprint/20221218-19595326.png)

## Inputs

在。执行。

目标: Bink Media Player 对象参考。

## Outputs

出：执行。

返回值。布尔值。如果播放开始，为 true，否则为 false。参见。CanPlay, Pause, Rewind, Seek, SetRate.

<hr>

Starts media playback.

This is the same as setting the playback rate to 1.0.

Target is Bink Media Player

## 图示

![](/uploads/projects/ue-bluprint/20221218-19595326.png)

## Inputs

In: Exec.

Target: Bink Media Player Object Reference.

## Outputs

Out: Exec.

Return Value: Boolean. true if playback is starting, false otherwise. See: CanPlay, Pause, Rewind, Seek, SetRate.
