<hr>

Checks whether this player has been initialized with a media source.

Target is Bink Media Player

## 图示

![](/uploads/projects/ue-bluprint/20221218-19594160.png)

## Inputs

Target: Bink Media Player Object Reference.

## Outputs

Return Value: Boolean. true if a media source is associated with this player. See: OpenUrl.
