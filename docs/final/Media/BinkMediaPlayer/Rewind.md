# Rewind

将媒体回放至开头。

这与寻求时间归零相同。

目标是 Bink 媒体播放器

## 图示

![](/uploads/projects/ue-bluprint/20221218-19595447.png)

## Inputs

在。执行。

目标: Bink Media Player 对象参考。

## Outputs

出：执行。

返回值。布尔值。如果倒带，为真，否则为假。参见。GetTime, Pause, Play, Seek.

<hr>

Rewinds the media to the beginning.

This is the same as seeking to zero time.

Target is Bink Media Player

## 图示

![](/uploads/projects/ue-bluprint/20221218-19595447.png)

## Inputs

In: Exec.

Target: Bink Media Player Object Reference.

## Outputs

Out: Exec.

Return Value: Boolean. true if rewinding, false otherwise. See: GetTime, Pause, Play, Seek.
