# SetReverbBrightness

正数时增加高频混响时间，负数时减少。设置为 0.0 时没有影响

目标是 Resonance Audio 混响插件预置。

## 图示

![](/uploads/projects/ue-bluprint/20221218-20405564.png)

## Inputs

在。执行。

目标。共振音频混响插件预置对象参考。

在混响亮度中。浮点（单精度）。

## Outputs

出：执行。

<hr>

Increases high frequency reverberation times when positive, decreases when negative.. Has no effect when set to 0.0

Target is Resonance Audio Reverb Plugin Preset

![](/uploads/projects/ue-bluprint/20221218-20405564.png)

In: Exec.

In Reverb Brightness: Float (single-precision).

## Outputs

Out: Exec.
