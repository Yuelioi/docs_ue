# RunAllFunctionalTests

依次触发在该层发现的所有功能测试。

目标是功能测试经理

## 图示

![](/uploads/projects/ue-bluprint/20221218-19010539.png)

## Inputs

在。执行。

新的日志。布尔值。

运行循环。布尔型。

失败的测试 Repro String:字符串。

## Outputs

出：执行。

返回值。布尔值。如果有任何测试被触发，则为真。

<hr>

Triggers in sequence all functional tests found on the level.

Target is Functional Testing Manager

## 图示

![](/uploads/projects/ue-bluprint/20221218-19010539.png)

## Inputs

In: Exec.

New Log: Boolean.

Run Looped: Boolean.

Failed Tests Repro String: String.

## Outputs

Out: Exec.

Return Value: Boolean. true if any tests have been triggered.
