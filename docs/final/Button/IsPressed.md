# IsPressed

如果用户正在主动按下按钮，返回 true。不要用它来检测 "点击"，用 OnClicked 事件代替。

目标是 Button

## 图示

![](/uploads/projects/ue-bluprint/20221218-18123283.png)

## Inputs

目标：Button 对象参考。

## Outputs

返回值。布尔值。如果用户正在主动按下按钮，则为真，否则为假。

<hr>

Target is Button
