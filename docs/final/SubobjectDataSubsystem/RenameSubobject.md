# RenameSubobject

试图将给定的子对象重命名为新的名称。

目标是子对象数据子系统

## 图示

![]($-20221218-21060133.png)

## Inputs

在。执行。

目标。子对象数据子系统对象参考。

手柄。子对象数据处理结构（通过引用）。要重命名的子对象的句柄。

在新名称中。文本（通过引用）。希望给定的子对象的新名称。  

## Outputs

出：执行。

返回值。布尔值。如果重命名成功为真，否则为假。

Attempts to rename the given subobject to the new name.

Target is Subobject Data Subsystem

## 图示

![]($-20221218-21060133.png)

## Inputs

In: Exec.

Target: Subobject Data Subsystem Object Reference.

Handle: Subobject Data Handle Structure (by ref). Handle to the subobject to rename.

In New Name: Text (by ref). The new name that is desired for the given subobject.  

## Outputs

Out: Exec.

Return Value: Boolean. True if the rename was successful, false otherwise..
