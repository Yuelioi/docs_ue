# IsARSessionTypeSupported

测试当前的增强现实平台是否支持这种类型的会话。例如，你的设备是否能够进行位置追踪或只进行方位追踪？

目标是 ARBlueprint 库

## 图示

![](/uploads/projects/ue-bluprint/20221218-17581586.png)

## Inputs

会话类型。EARSessionType 枚举。

## Outputs

返回值。布尔值。测试当前增强现实平台是否支持这种类型的会话。例如，你的设备是否能够进行位置跟踪或只进行方向跟踪？
