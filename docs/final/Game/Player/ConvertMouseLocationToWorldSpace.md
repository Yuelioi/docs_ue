# ConvertMouseLocationToWorldSpace

将当前鼠标的 2D 位置转换成世界空间的 3D 位置和方向。如果无法确定数值，则返回 false。\*

目标是玩家控制器

## Inputs

目标。玩家控制器对象参考。

## Outputs

世界的位置。向量。

世界方向。矢量。

返回值。布尔值。将当前鼠标的 2D 位置转换成世界空间的 3D 位置和方向。如果无法确定数值，则返回 false。\*.
