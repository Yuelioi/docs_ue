# GetNumPlayerStates

返回活跃的玩家状态的数量，每个连接的玩家都有一个玩家状态，即使他们是一个远程客户端。到此为止的索引可以作为其他函数的 PlayerStateIndex 参数使用。

目标是游戏性的统计学
