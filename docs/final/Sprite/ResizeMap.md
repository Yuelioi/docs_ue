# ResizeMap

调整瓦片地图的大小（注意：这只对拥有自己瓦片地图的组件有效（OwnsTileMap 返回 true），你不能修改独立的瓦片地图资产）。

目标是纸质瓦片地图组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-21024294.png)

## Inputs

在。执行。

目标。纸质瓦片地图组件对象参考。

以瓷砖为单位的新宽度。整数。

以瓷砖为单位的新高度。整数。

## Outputs

出：执行。
