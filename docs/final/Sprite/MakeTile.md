# MakeTile

根据指定的信息创建一个瓦片

目标是瓦片地图蓝图库

## 图示

![](/uploads/projects/ue-bluprint/20221218-21023833.png)

## Inputs

瓦片索引。整数。

瓦片组。纸质瓦片组对象参考。

翻转 H：布尔值。

翻转 V：布尔值。

翻转 D。布尔型。

## Outputs

返回值。纸片信息结构。根据指定的信息创建一个纸牌。

<hr>

Creates a tile from the specified information

Target is Tile Map Blueprint Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-21023833.png)

## Inputs

Tile Index: Integer.

Tile Set: Paper Tile Set Object Reference.

Flip H: Boolean.

Flip V: Boolean.

Flip D: Boolean.

## Outputs

Return Value: Paper Tile Info Structure. Creates a tile from the specified information.
