Creates and adds a new layer to the tile map. Note: This will only work on components that own their own tile map (OwnsTileMap returns true), you cannot modify standalone tile map assets

Target is Paper Tile Map Component

## 图示

![](/uploads/projects/ue-bluprint/20221218-21021277.png)

## Inputs

In: Exec.

Target: Paper Tile Map Component Object Reference.

## Outputs

Out: Exec.

Return Value: Paper Tile Layer Object Reference. Creates and adds a new layer to the tile map. Note: This will only work on components that own their own tile map (OwnsTileMap returns true), you cannot modify standalone tile map assets.
