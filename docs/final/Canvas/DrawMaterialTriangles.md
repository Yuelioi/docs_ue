# DrawMaterialTriangles

在 Canvas 上绘制一组三角形。

目标是画布

## 图示

![](/uploads/projects/ue-bluprint/20221218-18144277.png)

## Inputs

在。执行。

目标。Canvas 对象参考。

渲染材料。材料界面对象参考。渲染时使用的材料。记住，只有发光通道可以被渲染，因为在渲染到 Canvas 时不会进行照明。

Triangles（三角形）。Canvas UVTri 结构的阵列。要渲染的三角形。

## Outputs

输出。执行。

<hr>

Draws a set of triangles on the Canvas.

Target is Canvas

## 图示

![](/uploads/projects/ue-bluprint/20221218-18144277.png)

## Inputs

In: Exec.

Target: Canvas Object Reference.

Render Material: Material Interface Object Reference. Material to use when rendering. Remember that only the emissive channel is able to be rendered as no lighting is performed when rendering to the Canvas..

Triangles: Array of Canvas UVTri Structures. Triangles to render..

## Outputs

Out: Exec.
