<hr>

Returns the control settings of a given control

Target is Rig Hierarchy Controller

## 图示

![](/uploads/projects/ue-bluprint/20221218-21204212.png)

## Inputs

Target: Rig Hierarchy Controller Object Reference.

In Key: Rig Element Key Structure. The key of the control to receive the settings for.

## Outputs

Return Value: Rig Control Settings Structure. The settings of the given control.
