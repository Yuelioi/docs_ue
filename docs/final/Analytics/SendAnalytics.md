# SendAnalytics

推迟到 USDClasses 模块来实际发送分析信息

目标是 Usd Exporter 蓝图库

## 图示

![](/uploads/projects/ue-bluprint/20221218-17491436.png)

## Inputs

在。Exec.

Attrs。分析事件 Attr 结构的数组。

事件名称。字符串。

自动的。Boolean.

耗时的秒数。Float (double-precision).

帧的数量。Float (double-precision).

扩展。字符串。

## Outputs

输出。执行。

<hr>

Defer to the USDClasses module to actually send analytics information

Target is Usd Exporter Blueprint Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-17491436.png)

## Inputs

In: Exec.

Attrs: Array of Analytics Event Attr Structures.

Event Name: String.

Automated: Boolean.

Elapsed Seconds: Float (double-precision).

Number Of Frames: Float (double-precision).

Extension: String.

## Outputs

Out: Exec.
