# MakeOutgoingSpec

获得一个传出的 GameplayEffectSpec，准备应用于其他事物。

目标是能力系统组件

## 图示

![](/uploads/projects/ue-bluprint/20221218-19090008.png)

## Inputs

目标。能力系统组件对象参考。

游戏效果类。游戏效果类参考。

水平。浮点数（单精度）。

语境。游戏性效果语境处理结构。

## Outputs

返回值。Gameplay Effect Spec Handle 结构。获取一个出场的 GameplayEffectSpec，准备应用于其他事物。
