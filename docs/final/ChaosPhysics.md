## Actions

设置为当前世界的解算器。使这个求解器成为当前世界的求解器。动态生成的物体将在这个求解器中创建其物理状态。目标是混沌求解器演员。

设置求解器激活。控制求解器是否能够模拟其控制的粒子。目标是混沌求解器角色。

## Actions

Set as Current World Solver. Makes this solver the current world solver. Dynamically spawned objects will have their physics state created in this solver.: Target is Chaos Solver Actor.

Set Solver Active. Controls whether the solver is able to simulate particles it controls: Target is Chaos Solver Actor.
