# ToSeparateTexts

将一个地理坐标值转换为3个独立的文本值

目标是地理坐标功能库

## 图示

![]($-20221218-19142390.png)

## Inputs

地理坐标。地理坐标结构（按参考）。

整数位 Lat Lon:整数。

整数位 Alti: 整数。

作为DMS：布尔值。  

## Outputs

出来的纬度。文本。

出经度。文本。

出海拔。文字。
