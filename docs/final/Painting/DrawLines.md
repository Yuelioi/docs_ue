# DrawLines

绘制几个线段。

目标是小工具蓝图库

## 图示

![]($-20221218-20175194.png)

## Inputs

在。执行。

语境。油漆背景结构（按参考）。

点。矢量二维结构的数组。线对，每条线需要是数组中的2个独立点。

色调。线性色彩结构。用颜色来呈现线...

反别名：布尔值。线条是否应该被反锯齿化。

厚度。Float（单精度）。这条线应该有多少像素厚。

## Outputs

出：执行。

Draws several line segments.

Target is Widget Blueprint Library

## 图示

![]($-20221218-20175194.png)

## Inputs

In: Exec.

Context: Paint Context Structure (by ref).

Points: Array of Vector 2D Structures. Line pairs, each line needs to be 2 separate points in the array..

Tint: Linear Color Structure. Color to render the line..

Anti Alias: Boolean. Whether the line should be antialiased..

Thickness: Float (single-precision). How many pixels thick this line should be..  

## Outputs

Out: Exec.
