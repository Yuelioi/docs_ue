# ExecuteConsoleCommand

执行一个控制台命令，可以选择在一个特定的控制器上执行。

目标是 Kismet 系统库

## 图示

![](/uploads/projects/ue-bluprint/20221218-18423800.png)

## Inputs

在。执行。

命令。字符串。发送给控制台的命令。

具体的球员。播放器控制器对象参考。如果指定了，控制台命令将通过指定的播放器被传送。

## Outputs

出：执行。

<hr>

Executes a console command, optionally on a specific controller

Target is Kismet System Library

## 图示

![](/uploads/projects/ue-bluprint/20221218-18423800.png)

## Inputs

In: Exec.

Command: String. Command to send to the console.

Specific Player: Player Controller Object Reference. If specified, the console command will be routed through the specified player.

## Outputs

Out: Exec.
