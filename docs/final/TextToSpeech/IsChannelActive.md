# IsChannelActive

如果文字转语音通道处于激活状态，则返回真，否则返回假。

目标是文字转语音引擎子系统

## Inputs

目标。文本到语音引擎子系统对象参考。

频道标识。名称。频道的Id，以检查它是否处于活动状态。  

## Outputs

返回值。布尔值。

Returns true if the text to speech channel is active, otherwise false.

Target is Text to Speech Engine Subsystem

Target: Text To Speech Engine Subsystem Object Reference.

Channel Id: Name. The Id for the channel to check if it is active..
