# DoesChannelExist

如果存在一个与频道ID相关的文本到语音频道，则返回true。否则，该函数返回false。

目标是文字转语音引擎子系统

## Inputs

## Outputs

返回值。布尔值。

Returns true if a text to speech channel associated with a channel Id exists. Otherwise, the function returns false.
