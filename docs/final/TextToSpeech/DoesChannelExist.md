# DoesChannelExist

如果存在一个与频道 ID 相关的文本到语音频道，则返回 true。否则，该函数返回 false。

目标是文字转语音引擎子系统

## Inputs

目标。文本到语音引擎子系统对象参考。

频道标识。名称。检验是否与一个频道相关联的频道 Id。

## Outputs

返回值。布尔值。
