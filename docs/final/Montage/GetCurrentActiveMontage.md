# GetCurrentActiveMontage

在这个 AnimInstance 中获得一个当前的 Active Montage。请注意，在同一时间可能有多个活动。这将只返回它找到的第一个活跃的。\*

目标是 Anim Instance

Get a current Active Montage in this AnimInstance.. Note that there might be multiple Active at the same time. This will only return the first active one it finds. \*

Target: Anim Instance Object Reference.

Return Value: Anim Montage Object Reference. Get a current Active Montage in this AnimInstance.. Note that there might be multiple Active at the same time. This will only return the first active one it finds. \*.
