# GetLinearFrequency

返回输入值的对数频率。将线性域和范围值映射到对数输出（对控制频率的线性滑块有好处）

目标是合成实用程序蓝图功能库

## 图示

![]($-20221218-21090416.png)

## Inputs

在。执行。

在对数频率值。Float (single-precision).

在域内最小。Float (single-precision).

域内最大值：浮点数（单精度）。

在范围内最小。浮点数（单精度）。

最大范围内：浮点数（单精度）。  

## Outputs

出：执行。

返回值。浮点数（单精度）。返回输入值的对数频率。将线性域和范围值映射到对数输出（有利于线性滑块控制频率）。
