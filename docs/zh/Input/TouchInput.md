## Actions

输入时触摸开始。On Input Touch Begin:.当触摸事件在播放器控制器中被启用时，通过该角色收到触摸输入时被调用。

输入时触摸结束。On Input Touch End:.当触摸事件在播放器控制器中被启用时，当触摸输入在此组件上被释放时调用的事件。

在输入中 触摸输入。On Input Touch Enter:.当玩家控制器中的触摸事件被启用时，当手指移动到这个角色上时被调用。

输入时触摸离开。On Input Touch Leave:.当玩家控制器中的触摸事件被启用时，当手指从这个组件上移开时调用的事件。
