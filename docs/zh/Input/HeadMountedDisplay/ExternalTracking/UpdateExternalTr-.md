# UpdateExternalTr-

在校准后调用，尝试将内部追踪器（例如HMD追踪）与外部追踪器拉到一起。(例如，Mocap追踪器）。这将设置内部跟踪器的基准偏移和旋转，以匹配并重新调整这两个系统。这可以在每次打勾时调用，或者在需要重新对齐时调用。请注意，如果两个系统相对于对方有分歧，这可能会导致不稳定的运动，或者如果在有重大漂移的情况下不经常调用，则会导致大跳动。

目标是头戴式显示器功能库
