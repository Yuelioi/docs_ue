# MakeInputActionValue

从X、Y、Z建立一个ActionValue，继承现有ActionValue的类型。忽略所提供的值类型未使用的轴值。注：旨在用于输入修改器修改原始重载，以修改现有的输入动作值。

目标是增强型输入库

## 图示

![]($-20221218-19271694.png)

## Inputs

X：浮点数（单精度）。

Y：浮点数（单精度）。

Z：浮点数（单精度）。

匹配值类型。输入行动值结构（通过参考）。  

## Outputs

返回值。输入动作值结构。从X、Y、Z建立一个动作值，继承现有动作值的类型。忽略所提供的值类型未使用的轴值。注释 用于输入修改器修改原始重载以修改现有的输入动作值。
