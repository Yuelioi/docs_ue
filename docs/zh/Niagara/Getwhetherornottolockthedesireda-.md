# Getwhetherornottolockthedesireda-

获取在使用期望年龄时用于勾选系统实例的delta时间是否被锁定为寻址delta。当为true时，系统实例只有在期望年龄的变化超过了寻址delta时才会被勾选。当为 "假 "时，系统实例将根据期望年龄的变化在不寻求时被勾选。

目标是Niagara粒子系统组件

![]($-20221218-20122420.png)

## Inputs

目标。尼亚加拉粒子系统组件对象参考。  

## Outputs

返回值。布尔值。获取当使用期望年龄时，用于勾选系统实例的delta时间是否锁定在寻址delta上。当为true时，系统实例只有在期望年龄的变化超过了寻址delta时才会被勾选。当为假时，系统实例将根据期望年龄的变化来勾选，当不寻求时。
