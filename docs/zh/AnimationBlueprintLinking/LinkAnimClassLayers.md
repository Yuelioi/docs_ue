# LinkAnimClassLayers

运行所有的层结点，试图找到由指定的类实现的层结点，然后为每个层结点建立一个类的链接实例。分配一个链接实例来运行类中指定的每个组，所以状态是共享的。如果一个层没有被分组（即NAME_NONE），那么状态就不会被共享，并且为每个层节点分配一个单独的链接实例。如果InClass是空的，那么所有层都被重置为默认状态。

目标是Anim Instance
