# ActivateScreenReader

激活底层的屏幕阅读器。使用这个来允许屏幕阅读器用户在屏幕阅读器上注册，并通过文本到语音接收无障碍反馈，并获得对其他屏幕阅读器服务的访问。一个带有激活功能的基本工作流程是：。ActivateScreenReader() -> RegisterScreenReaderUser() -> ActivateScreenReaderUser()。见。RegisterScreenReaderUser(), ActivateScreenReaderUser()

目标是Slate屏幕阅读器引擎子系统
