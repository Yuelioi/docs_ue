# JoinStaticMeshActors

在关卡中创建一个新的演员，包含所有演员静态网格组件的副本。ActorsToJoin需要在同一个关卡中。这对性能的影响不大，但可以通过将网格归入一个Actor来帮助版本。

目标是静态网格编辑器子系统
