# GetAllObjectsandValuesforKey

查找所有具有Datasmith用户数据的给定类型的加载对象，该对象包含给定的键和它们的相关值。这是一个缓慢的操作，所以只限于编辑器。

目标是Datasmith内容蓝图库

## 图示

![]($-20221218-18492549.png)

## Inputs

在。执行。

钥匙。名称。在Datasmith用户数据中找到的键...

对象类。对象类参考。要过滤的对象的类别，如果有具体规定的话；否则就没有过滤功能。

## Outputs

出：执行。

输出对象。对象引用数组。输出对象数组，其中Datasmith用户数据与给定的键匹配。

输出值。字符串数组。输出与OutObjects中每个对象相关的值数组。
