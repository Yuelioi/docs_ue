# VectorSpringInterp

使用一个简单的弹簧模型来插值一个从当前到目标的矢量。

目标是Kismet数学图书馆

## 图示

![]($-20221218-19513486.png)

## Inputs

在。执行。

目前。矢量。当前值。

目标。矢量。目标值。

弹簧状态。向量弹簧状态结构（通过参考）。与弹簧模型相关的数据（速度，误差，等等...） - 为每个弹簧创建一个唯一的变量。

硬度。浮点（单精度）。弹簧模型的刚度如何（更多的刚度意味着围绕目标值的更多摆动）。

临界阻尼系数。浮动（单精度）。对弹簧施加多少阻尼（0意味着没有阻尼，1意味着临界阻尼，意味着没有振荡）。

三角洲时间。浮点数（单精度）。自上次更新以来的时间差。

质量。浮点（单精度）。倍数，作用类似于弹簧上的质量。

目标速度量。浮点（单精度）。如果是1，那么将计算并使用目标速度，这将导致更紧密地跟随目标/没有滞后。数值降至零（建议在使用此功能平滑数据时）将逐步禁用此效果。

钳制。布尔值。是否使用最小/最大值来夹紧运动。

最小值。矢量。夹住最小输出值，如果达到这个限制，就取消速度。

最大值。矢量。夹住最大输出值，如果达到这个极限，就取消速度。

从目标初始化：布尔值。如果设置，那么当前值将在第一次更新时从目标设置。  

## Outputs

出：执行。

返回值。向量。
