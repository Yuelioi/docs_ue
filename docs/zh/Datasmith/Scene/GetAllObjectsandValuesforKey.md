# GetAllObjectsandValuesforKey

查找所有具有元数据元素的给定类型的对象，该元数据元素包含给定的键和它们的相关值。

目标是Datasmith场景元素基础

## 图示

![]($-20221218-18400090.png)

## Inputs

在。执行。

目标。Datasmith场景元素基本对象参考。

钥匙。字符串。要在元数据元素中找到的键。

对象类。数据元素类参考。要过滤的对象的类别，如果有具体规定的话；否则就没有过滤功能。  

## Outputs

出：执行。

输出对象。Datasmith对象元素对象引用数组。输出元数据元素包含给定关键字的对象数组。

输出值。字符串数组。输出与OutObjects中每个对象相关的值数组。
