# GetTrianglesinPolygroup

在给定的GroupLayer中创建具有给定的Polygroup ID的所有三角形的列表（不一定是单一的连接构件）。

目标是几何脚本库网格多组函数

## Inputs

在。执行。

目标网格。动态网格对象参考。

组图层。Geometry Script Group Layer Structure.

Polygroup ID。整数。

三角形ID出来：索引列表结构（按参考）。  

## Outputs

出：执行。

目标网格。动态网格对象参考。创建给定GroupLayer中具有给定Polygroup ID的所有三角形的列表（不一定是单一的连接构件）。
