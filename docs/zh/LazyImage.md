## Actions

正在加载。正在加载。目标是普通的懒惰图像。

加载状态改变时。BP在加载状态改变时。

从懒惰显示资产设置画笔。仅从字符串资产参考中设置画笔 - 希望参考的资产是纹理或材料。目标是普通懒惰图像。

从懒惰材料设置画笔。从懒惰的材质资产指针设置笔刷 - 将根据需要加载材质。目标是普通懒惰图像。

从懒惰纹理设置画笔。从懒惰的纹理资产指针中设置画笔--将根据需要加载纹理。目标是普通懒惰图像。

Set Material Texture Param Name（设置材质纹理参数名称）。确定当前应用的画笔材质上的纹理参数名称，纹理应该应用于该材质。如果当前的画笔资源对象不是材质，则什么也不做。注意：如果/当在画笔上建立一个新的材质时，这个参数会自动清空。如果新材料有一个纹理参数，你必须在这样做之后再次调用这个函数。目标是普通的懒惰图像。
