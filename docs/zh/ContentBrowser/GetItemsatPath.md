# GetItemsatPath

获取存在于给定虚拟路径的项目（文件夹和/或文件）。注意：如果多个项目是不同的类型，或来自不同的数据源，它们可能有相同的虚拟路径。

目标是内容浏览器数据子系统

## Inputs

目标。内容浏览器数据子系统对象参考。

在路径中。名称。

在项目类型过滤器中。EContentBrowserItemTypeFilter枚举。 

## Outputs

返回值。内容浏览器项目结构的数组。获取存在于给定虚拟路径的项目（文件夹和/或文件）。注意：如果多个项目是不同的类型，或来自不同的数据源，它们可能有相同的虚拟路径。
