# CopyMaterialInstanceParameters

复制给定材料接口的参数（按照层次结构复制每个实例）。实现起来非常慢，避免在运行时使用。希望我们以后能用类似CopyInterpParameters()的东西来代替它。输出是对象本身（this）。复制 "仅快速参数 "将导致一个更快的复制过程，但将仅在客户端复制动态标量、矢量和纹理参数。

目标是材料实例动态
