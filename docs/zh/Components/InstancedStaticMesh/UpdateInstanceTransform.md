# UpdateInstanceTransform

为指定的实例更新变换。

目标是实例化的静态网格组件

## 图示

![]($-20221218-18244691.png)

## Inputs

在。执行。

目标：静止网格组件对象参考。

实例索引。整数。要更新的实例的索引。

新实例变换。变换（通过引用）。新的变换。

世界空间。布尔值。如果为真，新的变换被解释为世界空间变换，否则被解释为本地空间。

Mark Render State Dirty（标记渲染状态）。布尔值。如果为真，变化应该立即可见。如果你要更新许多实例，你应该只为最后一个实例设置为真。

传送。布尔值。实例的物理特性是否应该被正常移动，或者被传送（瞬间移动，忽略速度）。 

## Outputs

输出。执行。

返回值。布尔值。成功时为真。
