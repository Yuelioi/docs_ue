# SetColorVisionDeficiencyType

在游戏窗口中应用颜色缺失校正设置

目标是小工具蓝图库

## 图示

![]($-20221218-21332640.png)

## Inputs

在。执行。

类型。EColorVisionDeficiency Enum。要应用的颜色缺失校正类型。

严重程度。Float（单精度）。颜色缺失校正效果的强度，从0到1。

纠正缺陷。布尔值。根据当前的缺陷类型，将颜色光谱转移到可见范围。

显示有缺陷的纠正。布尔值。如果你正在修正颜色缺陷，你可以用它来直观地看到修正后的缺陷是什么样子。  

## Outputs

出：执行。
