# ResetSeatedPosition

将坐姿跟踪器坐标系的零姿态设置为HMD的当前位置和偏航。在这个调用之后，以TrackingUniverseSeated为原点的调用将相对于这个新的零点姿态。

注意：这个函数会覆盖用户之前保存的坐姿零度，并且只能作为用户动作的结果被调用。用户也可以通过SteamVR仪表板来设置他们的坐姿零度。

目标是Steam VRInput设备功能库
