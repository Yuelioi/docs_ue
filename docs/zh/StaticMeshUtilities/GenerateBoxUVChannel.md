# GenerateBoxUVChannel

在给定的静态网格的LOD上生成指定UV通道的箱体UV贴图。

目标是静态网格编辑器子系统

## 图示

![]($-20221218-21033210.png)

## Inputs

在。执行。

目标。静态网格编辑器子系统对象参考。

静态网格。静态网格对象参考。用于生成UV贴图的静止网格。

LODIndex。整数。StaticMesh LOD的索引。

UVChannel索引。整数。保存UV贴图的通道。

位置。矢量（通过参考）。投影仪中心的位置。

方向。旋转器（通过参考）。应用于投影仪的旋转。

尺寸。矢量（通过参考）。箱体投影仪的尺寸。  

## Outputs

出：执行。

返回值。布尔值。如果生成了UV贴图，则为true。
