# GetDisplayName

返回显示名称（或演员标签），作为调试帮助显示。注意：在编辑器的构建中，这是演员的标签。在非编辑器构建中，这是实际的对象名称。这个函数不应该被用来唯一地识别演员！。它没有被本地化，也不应该被用于向游戏的终端用户显示。
