# GetCurrentCulture

获取当前文化作为IETF语言标签。

  * 一个两个字母的ISO 639-1语言代码（例如，"zh"）。

  * 一个可选的四个字母的ISO 15924脚本代码（例如，"Hans"）。

  * 一个可选的两个字母的ISO 3166-1国家代码（例如，"CN"）。注意：这个函数的存在是为了与SetCurrentCulture保持一致，相当于GetCurrentLanguage。





目标是Kismet国际化库
