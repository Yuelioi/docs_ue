# GetPositiontoOptionMapfromActor

从一个有ISM组件的给定角色中获取PositionToOptionsMap。当你想从一个WFC解决的行为体中获得邻近的瓦片数据以用于后期处理时，这很有用。这将只评估ISM组件。

目标是波函数崩溃BPLibrary

## 图示

![]($-20221218-21330141.png)

## Inputs

在。执行。

演员。Actor对象参考。具有ISM组件的演员。

瓦片大小。浮点数（单精度）。瓦片之间的距离。

位置到期权地图。Int矢量结构到波函数崩溃期权结构的地图。  

## Outputs

出：执行。

返回值。布尔值。
