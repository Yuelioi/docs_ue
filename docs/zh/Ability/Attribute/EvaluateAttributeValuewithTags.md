# EvaluateAttributeValuewithTags

用源标签和目标标签评估能力系统组件AbilitySystem后，返回Attribute的值。 bSuccess表示该操作的成败。

目标是能力系统蓝图库

## 图示

![]($-20221218-17301014.png)

## Inputs

能力系统。能力系统组件对象参考。

Attribute（属性）。Gameplay Attribute结构。

源标签。游戏玩法标签容器结构（通过参考）。

目标标签。游戏性标签容器结构（根据参考文献）。 

## Outputs

成功。布尔值。

返回值。浮点数（单精度）。返回能力系统组件AbilitySystem用源标签和目标标签评估后的Attribute值。bSuccess表示该操作的成败。
