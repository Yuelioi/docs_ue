# CopyPropertiestoStaticMesh

尝试复制Actor属性到一个StaticMeshActor。可选择将DynamicMeshComponent材料列表复制到StaticMeshComponent。这个功能在（例如）从一个动态MeshActor交换到一个静态MeshActor时很有用，因为它可以保留许多配置的Actor设置（如分配的DataLayers等）。

目标是生成的动态网格角色
