# MoveUpdatedComponent

将我们的UpdateComponent移动到给定的Delta，并将旋转设置为NewRotation。如果启用的话，尊重平面约束。

目标是运动组件

![]($-20221218-18250063.png)

## Inputs

在。执行。

目标。运动组件对象参考。

三角洲。矢量。

新的旋转。旋转器。

扫频。布尔值。

远距传输。布尔值  

## Outputs

输出。执行：执行。

Out Hit: 命中结果结构。

返回值。布尔值。如果发生一些运动，则为真，如果没有发生运动，则为假。任何撞击的结果将被存储在OutHit.中。
