# GetAnimInstance

返回驱动该类的动画实例（如果有的话）。这通常是设置为AnimBlueprintGeneratedClass（由动画蓝图生成）的一个实例。因为这个实例是瞬时的，所以在构造脚本中使用它并不安全。

目标是骨骼网格组件
