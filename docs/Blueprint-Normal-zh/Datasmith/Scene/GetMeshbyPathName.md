# GetMeshbyPathName

在Datasmith场景中找到与Mesh路径名称对应的MeshElement。如果MeshPathName是空的，或者它不是相对于Datasmith场景的，或者没有找到，该函数将返回一个无效的MeshElement。

目标是Datasmith场景元素基础
