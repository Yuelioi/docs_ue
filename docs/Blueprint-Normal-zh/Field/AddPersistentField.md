# AddPersistentField

AddPersistentField。这个函数将调度一个命令给物理线程来应用，是对用户定义的场网的通用评估。这个命令在时间上将是持久的，并将一直存在，直到组件被销毁或RemovePersistenFields函数被调用。关于如何使用场网络重新创建上述通用场的变体的例子，请参见文件，。

(https://wiki.it.epicgames.net/display/~Brice.Criswell/Fields)
