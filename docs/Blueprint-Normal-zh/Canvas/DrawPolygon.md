# DrawPolygon

在Canvas上画一个多边形。

目标是画布

## 图示

![]($-20221218-18144412.png)

## Inputs

在。执行。

目标。Canvas 对象参考。

渲染纹理。纹理对象参考。渲染三角形时要使用的纹理。如果没有设置纹理，那么就使用默认的白色纹理。

Screen Position（屏幕位置）。矢量2D结构。渲染文本的屏幕空间位置。

半径。二维矢量结构。这个多边形应该有多大的像素。

边的数量。整数。这个多边形应该有多少个边。应该大于或等于3。

渲染颜色：线性颜色结构。为多边形着色的颜色。 

## Outputs

输出。执行：执行。
