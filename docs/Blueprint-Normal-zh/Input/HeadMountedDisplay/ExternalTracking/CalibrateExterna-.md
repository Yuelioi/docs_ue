# CalibrateExterna-

用于校准外部跟踪源与内部跟踪源之间的偏移变换。(例如，Mocap跟踪器与HMD跟踪器的关系）。每个会话应该调用一次，或者当外部跟踪器和内部跟踪器之间的物理关系发生变化时（例如，它被碰撞或重新连接）。校准后，调用UpdateExternalTrackingPosition将尝试将内部跟踪器校正到校准的偏移量，以防止两个系统之间的漂移。

目标是头戴式显示器功能库
