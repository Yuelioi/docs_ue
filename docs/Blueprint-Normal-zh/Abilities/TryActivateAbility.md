# TryActivateAbility

试图激活给定的能力，在这样做之前会检查成本和要求。如果它认为已经激活，则返回 "true"，但由于激活过程中的失败，它可能会返回 "假阳性"。如果bAllowRemoteActivation为真，它将远程激活本地/服务器能力，如果为假，它将只尝试本地激活该能力。

目标是能力系统组件
