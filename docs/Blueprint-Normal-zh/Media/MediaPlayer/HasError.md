# HasError

检查播放器是否处于错误状态。

当播放器处于错误状态时，不可能有进一步的操作。必须关闭当前的媒体，并打开一个新的媒体源，才能再次使用播放器。错误通常是由错误的媒体文件或中断的网络连接引起的。
