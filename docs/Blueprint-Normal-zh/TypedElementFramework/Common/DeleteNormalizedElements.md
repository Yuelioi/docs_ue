# DeleteNormalizedElements

从给定的列表中删除任何可以被删除的元素。注意：这个列表应该已经通过UTypedElementSelectionSet::GetNormalizedSelection或UTypedElementSelectionSet::GetNormalizedElementList预先规范化。

目标是类型化的元素 共同行动
