# AsyncLoadPrimaryAssetList

将主要资产对象的列表加载到内存中，这将导致它们一直被加载，直到明确卸载。当加载成功或失败时，将发生完成事件，加载的列表将包含在完成时发现的所有请求的资产。如果指定了LoadBundles，这些捆绑的资产将被加载。

目标是Async Action Load Primary Asset List
