# GetBoundObjects

  * 检索当前绑定在指定绑定标识符上的所有对象。该序列将在指定范围的下限进行评估。

  * 这允许在该时间段内检索产卵器。





目标是Sequencer Tools功能库

## Inputs

在。执行。

在世界。世界对象参考。

在序列中。水平序列对象参考。

在绑定中。序列器绑定代理结构的阵列。

在范围内。定序器脚本的范围结构（由参考）。

## Outputs

出：执行。

返回值。序列器绑定对象的数组 结构。检索当前绑定在指定绑定标识符上的所有对象。序列将在指定范围的下限进行评估，：这允许在该时间段内检索可产卵器。
