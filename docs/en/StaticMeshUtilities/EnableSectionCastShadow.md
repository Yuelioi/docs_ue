# EnableSectionCastShadow

Enables/disables mesh section shadow casting for a specific LOD.

Target is Static Mesh Editor Subsystem

## 图示

![]($-20221218-21032953.png)

## Inputs

In: Exec.

Target: Static Mesh Editor Subsystem Object Reference.

Static Mesh: Static Mesh Object Reference. Static mesh to Enables/disables shadow casting from..

Cast Shadow: Boolean. If the section should cast shadow..

LODIndex: Integer. Index of the StaticMesh LOD..

Section Index: Integer. Index of the StaticMesh Section..  

## Outputs

Out: Exec.
