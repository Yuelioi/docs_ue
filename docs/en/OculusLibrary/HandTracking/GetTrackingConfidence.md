# GetTrackingConfidence

Get the tracking confidence of the hand

Target is Oculus Input Function Library

![]($-20221218-20153187.png)

## Inputs

Device Hand: EOculusHandType Enum. (in) The hand to get tracking confidence of.

Controller Index: Integer. (in) Optional different controller index.  

## Outputs

Return Value: ETrackingConfidence Enum.
