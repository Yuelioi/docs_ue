# DrawText

Draws text.

Target is Widget Blueprint Library

## 图示

![]($-20221218-20175317.png)

## Inputs

In: Exec.

Context: Paint Context Structure (by ref).

Text: Text (by ref). The string to draw..

Position: Vector 2D Structure. The starting position where the text is drawn in local space..

Font: Font Object Reference.

Font Size: Integer.

Font Type Face: Name.

Tint: Linear Color Structure. Color to render the line..  

## Outputs

Out: Exec.
