# SetPlayerCameraManager

Set player camera manager to bind the render target to.

Target is Composure Player Compositing Target

![]($-20221218-20294805.png)

## Inputs

In: Exec.

Target: Composure Player Compositing Target Object Reference.

Player Camera Manager: Player Camera Manager Object Reference.  

## Outputs

Out: Exec.

Return Value: Player Camera Manager Object Reference. Set player camera manager to bind the render target to..
