# GetSource

Returns the source asset for a UFoliageType.. It can be a UStaticMesh in case we're dealing with a UFoliageType_InstancedStaticMesh, but it can be other types of objects.

Target is Usd Exporter Blueprint Library

## Inputs

In: Exec.

Foliage Type: Foliage Type Object Reference.  

## Outputs

Out: Exec.

Return Value: Object Reference. Returns the source asset for a UFoliageType.. It can be a UStaticMesh in case we're dealing with a UFoliageType_InstancedStaticMesh, but it can be other types of objects..
