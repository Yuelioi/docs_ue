# GetPossessedObjectClass

Get this binding's possessed object class

Target is Movie Scene Binding Extensions

![]($-20221218-20544482.png)

## Inputs

In: Exec.

In Binding: Sequencer Binding Proxy Structure (by ref). The binding to get the possessed object class of.  

## Outputs

Out: Exec.

Return Value: Object Class Reference. The possessed object class of the binding.
