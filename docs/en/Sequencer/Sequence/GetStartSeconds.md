# GetStartSeconds

Get the starting time for the specified range in seconds, if it has one. Defined as the first valid time that is inside the range.

Target is Sequencer Scripting Range Extensions

![]($-20221218-20544938.png)

## Inputs

In: Exec.

Range: Sequencer Scripting Range Structure (by ref). The range to get the start from.  

## Outputs

Out: Exec.

Return Value: Float (single-precision).
