# GetNumChannelsUsed

Get the number of channels used for this track

Target is Movie Scene Float Vector Track Extensions

## 图示

![]($-20221218-20555259.png)

## Inputs

In: Exec.

Track: Movie Scene Float Vector Track Object Reference. The track to query for the number of channels used.  

## Outputs

Out: Exec.

Return Value: Integer. The number of channels used for this track.
