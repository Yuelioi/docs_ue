# GetArriveTangentWeight

If Tangent Weight Mode is RCTWM_WeightedArrive or RCTWM_WeightedBoth, the weight of the arriving tangent on the left side.

Target is Movie Scene Scripting Double Key

## 图示

![]($-20221218-20485580.png)

## Inputs

Target: Movie Scene Scripting Double Key Object Reference.  

## Outputs

Return Value: Float (single-precision). Tangent Weight. Represents the length of the hypotenuse in the form of "sqrt(x*x+y*y)" using the same definitions for x and y as tangents..
