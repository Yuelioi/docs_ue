# SetScalarParameterforCustomPrimi-

Set a scalar parameter for custom primitive data. This sets the run-time data only, so it doesn't serialize.

Target is Primitive Component

## 图示

![]($-20221218-20382213.png)

## Inputs

In: Exec.

Target: Primitive Component Object Reference.

Parameter Name: Name. The parameter name of the custom primitive.

Value: Float (single-precision). The new value of the custom primitive.  

## Outputs

Out: Exec.
