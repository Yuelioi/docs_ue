# GetRollatTime

Given a time from 0 to the spline duration, return the spline's roll there, in degrees.

Target is Spline Component

## Inputs

Coordinate Space: ESplineCoordinateSpace Enum.

Use Constant Velocity: Boolean.  

## Outputs

Return Value: Float (single-precision). Given a time from 0 to the spline duration, return the spline's roll there, in degrees..
