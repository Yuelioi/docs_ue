# GetUpVectoratTime

Given a time from 0 to the spline duration, return the spline's up vector there.

Target is Spline Component

## Inputs

Target: Spline Component Object Reference.

Time: Float (single-precision).

Coordinate Space: ESplineCoordinateSpace Enum.

Use Constant Velocity: Boolean.  

## Outputs

Return Value: Vector. Given a time from 0 to the spline duration, return the spline's up vector there..
