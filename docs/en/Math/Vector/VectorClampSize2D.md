# VectorClampSize2D

Create a copy of this vector, with the 2D magnitude/size/length clamped between Min and Max. Z is unchanged.

Target is Kismet Math Library

## 图示

![]($-20221218-19573791.png)

## Inputs

A: Vector.

Min: Float (double-precision).

Max: Float (double-precision).  

## Outputs

Return Value: Vector. Create a copy of this vector, with the 2D magnitude/size/length clamped between Min and Max. Z is unchanged..
