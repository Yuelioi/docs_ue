# FindClosestPointonSegment

Find the closest point on a segment to a given point.

Target is Kismet Math Library

![]($-20221218-19563883.png)

## Inputs

Point: Vector. Point for which we find the closest point on the segment..

Segment Start: Vector. Start of the segment..

Segment End: Vector. End of the segment..  

## Outputs

Return Value: Vector. The closest point on the segment to the given point..
