# MakeRotfromX

Builds a rotator given only a XAxis. Y and Z are unspecified but will be orthonormal. XAxis need not be normalized.

Target is Kismet Math Library

## Inputs

X: Vector (by ref).  

## Outputs

Return Value: Rotator. Builds a rotator given only a XAxis. Y and Z are unspecified but will be orthonormal. XAxis need not be normalized..
