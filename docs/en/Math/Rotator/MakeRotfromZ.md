# MakeRotfromZ

Builds a rotation matrix given only a ZAxis. X and Y are unspecified but will be orthonormal. ZAxis need not be normalized.

Target is Kismet Math Library

## Inputs

Z: Vector (by ref).  

## Outputs

Return Value: Rotator. Builds a rotation matrix given only a ZAxis. X and Y are unspecified but will be orthonormal. ZAxis need not be normalized..
