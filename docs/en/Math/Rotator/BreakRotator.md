# BreakRotator

Breaks apart a rotator into {Roll, Pitch, Yaw} angles in degrees

Target is Kismet Math Library

## 图示

![]($-20221218-19541798.png)

## Inputs

Rotation: Rotator.  

## Outputs

X (Roll): Float (single-precision).

Y (Pitch): Float (single-precision).

Z (Yaw): Float (single-precision).
