# Ease

Interpolates from value A to value B using a user specified easing function

![]($-20221218-19511256.png)

## Inputs

Function: EEasingFunc Enum. EEasingFunc Enum Function. Specifies the desired ease function to be applied. If connected no customization is possible..

Alpha: Float (double-precision). Float (double-precision) Alpha. Alpha value used to specify the easing in time..

A: Wildcard. Wildcard A. Easing start value.

B: Wildcard. Wildcard B. Easing end value.  

## Outputs

Result: Wildcard. Wildcard Result. Easing result value.
