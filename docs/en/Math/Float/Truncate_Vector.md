# Truncate_Vector

Rounds A to an integer with truncation towards zero for each element in a vector. (e.g. -1.7 truncated to -1, 2.8 truncated to 2)

Target is Kismet Math Library

## Inputs

In Vector: Vector (by ref).  

## Outputs

Return Value: Int Vector Structure. Rounds A to an integer with truncation towards zero for each element in a vector. (e.g. -1.7 truncated to -1, 2.8 truncated to 2).
