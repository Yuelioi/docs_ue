# Truncate

Rounds A towards zero, truncating the fractional part (e.g., -1.6 becomes -1 and 1.6 becomes 1)

Target is Kismet Math Library

## 图示

![]($-20221218-19502437.png)

## Inputs

A: Float (double-precision).  

## Outputs

Return Value: Integer. Rounds A towards zero, truncating the fractional part (e.g., -1.6 becomes -1 and 1.6 becomes 1).
