# SetOscSemitones

Sets the oscillator semitones.

Target is Modular Synth Component

## 图示

![]($-20221218-21081616.png)

## Inputs

In: Exec.

Target: Modular Synth Component Object Reference.

Osc Index: Integer. Which oscillator to set the type for..

Semitones: Float (single-precision). The amount of semitones to set the oscillator to (relative to base frequency/pitch)..  

## Outputs

Out: Exec.
