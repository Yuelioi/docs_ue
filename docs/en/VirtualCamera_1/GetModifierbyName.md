# GetModifierbyName

Tries to find a Modifier in the Stack with the given name.. The returned Modifier must be checked before it is used.

Target is VCam Component

## Inputs

Target: VCam Component Object Reference.

Name: Name.  

## Outputs

Return Value: VCam Modifier Object Reference. Tries to find a Modifier in the Stack with the given name.. The returned Modifier must be checked before it is used..
