# GetScreenshotInfo

Returns the information associated with a Screenshot.

Target is Virtual Camera Player Controller Base

## Inputs

Target: Virtual Camera Player Controller Base Object Reference.

Screenshot Name: String. The name of the screenshot to retrieve.  

## Outputs

Out Screenshot Info: Virtual Camera Screenshot Structure. Upon return, will hold the info for the screenshot..
