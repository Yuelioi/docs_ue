# AddBlendabletoCamera

Adds a blendable object to the camera's post process settings.

Target is Virtual Camera Player Controller Base

## 图示

![]($-20221218-21272510.png)

## Inputs

In: Exec.

Target: Virtual Camera Player Controller Base Object Reference.

Blendable to Add: Blendable Interface Interface. The blendable that will be added to the post process settings.

Weight: Float (single-precision). The weighting of the blendable's alpha.  

## Outputs

Out: Exec.
