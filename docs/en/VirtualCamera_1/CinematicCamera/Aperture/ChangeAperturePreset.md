# ChangeAperturePreset

Adjusts the aperture of the camera to an adjacent preset value.

Target is Virtual Camera Player Controller Base

## 图示

![]($-20221218-21272630.png)

## Inputs

In: Exec.

Target: Virtual Camera Player Controller Base Object Reference.

Shift Up: Boolean. If true, the aperture will increase; if false, it will decrease.  

## Outputs

Out: Exec.

Return Value: Float (single-precision). the new aperture of the camera.
