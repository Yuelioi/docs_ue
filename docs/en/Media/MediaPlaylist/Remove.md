# Remove

Remove all occurrences of the given media source in the play list.

Target is Media Playlist

## 图示

![]($-20221218-20020834.png)

## Inputs

In: Exec.

Target: Media Playlist Object Reference.

Media Source: Media Source Object Reference. The media source to remove..  

## Outputs

Out: Exec.

Return Value: Boolean. true if the media source was removed, false otherwise..
