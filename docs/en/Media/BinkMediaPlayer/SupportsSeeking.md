# SupportsSeeking

Checks whether the currently loaded media can jump to a certain position.

Target is Bink Media Player

![]($-20221218-20000407.png)

## Inputs

Target: Bink Media Player Object Reference.  

## Outputs

Return Value: Boolean. true if seeking is supported, false otherwise. See: SupportsRate, SupportsScrubbing.
