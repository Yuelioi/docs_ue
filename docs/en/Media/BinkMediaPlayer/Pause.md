# Pause

Pauses media playback.

This is the same as setting the playback rate to 0.0.

Target is Bink Media Player

## 图示

![]($-20221218-19595207.png)

## Inputs

In: Exec.

Target: Bink Media Player Object Reference.  

## Outputs

Out: Exec.

Return Value: Boolean. true if playback is being paused, false otherwise. See: CanPause, Play, Rewind, Seek, SetRate.
