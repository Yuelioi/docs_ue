# DuplicateNodes

Duplicate a collection of nodes from the same graph, using the InPosition position. to be the top-left origin of the pasted nodes.

Target is Optimus Node Graph

![]($-20221218-20171287.png)

## Inputs

In: Exec.

Target: Optimus Node Graph Object Reference.

In Nodes: Array of Optimus Node Object References.

In Position: Vector 2D Structure (by ref).  

## Outputs

Out: Exec.

Return Value: Boolean. Duplicate a collection of nodes from the same graph, using the InPosition position. to be the top-left origin of the pasted nodes..
