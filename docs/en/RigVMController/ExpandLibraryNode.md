# ExpandLibraryNode

Turns a library node into its contained nodes

Target is Rig VMController

## 图示

![]($-20221218-20423528.png)

## Inputs

In: Exec.

Target: Rig VMController Object Reference.

In Node Name: Name (by ref).

Setup Undo Redo: Boolean.

Print Python Command: Boolean.  

## Outputs

Out: Exec.

Return Value: Array of Rig VMNode Object References. Turns a library node into its contained nodes.
