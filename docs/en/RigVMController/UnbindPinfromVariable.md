# UnbindPinfromVariable

Removes the binging of a pin to a variable. This causes a PinBoundVariableChanged modified event.

Target is Rig VMController

Print Python Command: Boolean.  

## Outputs

Out: Exec.

Return Value: Boolean. Removes the binging of a pin to a variable. This causes a PinBoundVariableChanged modified event..
