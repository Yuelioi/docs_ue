# PromotePintoVariable

Promotes a pin to a variable

Target is Rig VMController

![]($-20221218-20430199.png)

## Inputs

In: Exec.

Target: Rig VMController Object Reference.

In Pin Path: String.

Create Variable Node: Boolean.

In Node Position: Vector 2D Structure (by ref).

Setup Undo Redo: Boolean.

Print Python Command: Boolean.  

## Outputs

Out: Exec.

Return Value: Boolean. Promotes a pin to a variable.
