# SetExposedPinIndex

Sets the index for an exposed pin. This can be used to move the pin up and down on the node.

Target is Rig VMController

![]($-20221218-20433074.png)

## Inputs

In: Exec.

Target: Rig VMController Object Reference.

In Pin Name: Name (by ref).

In New Index: Integer.

Setup Undo Redo: Boolean.

Print Python Command: Boolean.  

## Outputs

Out: Exec.

Return Value: Boolean. Sets the index for an exposed pin. This can be used to move the pin up and down on the node..
