# CreateJobfromSequence

Create a job from a level sequence. Sets the map as the currently editor world, the author, the sequence and the job name as the sequence name on the new job. Returns the newly created job.

Target is Movie Pipeline Editor Blueprint Library

## Inputs

In: Exec.

In Pipeline Queue: Movie Pipeline Queue Object Reference.

In Sequence: Level Sequence Object Reference.  

## Outputs

Out: Exec.

Return Value: Movie Pipeline Executor Job Object Reference. Create a job from a level sequence. Sets the map as the currently editor world, the author, the sequence and the job name as the sequence name on the new job. Returns the newly created job..
