# SetControlSettings

Sets a control's settings given a control key

Target is Rig Hierarchy Controller

## 图示

![]($-20221218-21205689.png)

## Inputs

Target: Rig Hierarchy Controller Object Reference.

In Key: Rig Element Key Structure. The key of the control to set the settings for.

In Settings: Rig Control Settings Structure.

Setup Undo: Boolean.  

## Outputs

Return Value: Boolean. Returns true if the settings have been set correctly.
