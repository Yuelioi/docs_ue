# AddUnique

Add item to array (unique)

Target is Kismet Array Library

## 图示

![]($-20221218-12165892.png)

## Inputs

In: Exec.

Target Array: Array of Wildcards. The array to add item to.

New Item: Wildcard (by ref). The item to add to the array.  

## Outputs

Out: Exec.

Return Value: Integer. The index of the newly added item, or INDEX_NONE if the item is already present in the array.
