# SetDisableCollision

Sets whether bodies attched to the constraint can collide or not

Target is Constraint Instance Blueprint Library

![]($-20221218-20272063.png)

## Inputs

In: Exec.

Accessor: Constraint Instance Accessor Structure (by ref). Constraint accessor to change.

Disable Collision: Boolean. true to disable collision between constrained bodies.  

## Outputs

Out: Exec.
