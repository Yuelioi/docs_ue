# ExportAsset

Call this to start an export asset process, the caller must specify a source data.

Target is Interchange Manager

## 图示

![]($-20221218-19310773.png)

## Inputs

In: Exec.

Target: Interchange Manager Object Reference.

Asset: Object Reference.

Is Automated: Boolean. If true the exporter will not show any UI or dialog.  

## Outputs

Out: Exec.

Return Value: Boolean. true if the import succeed, false otherwise..
