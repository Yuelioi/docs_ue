# CreateNewBlueprintComponent

Creates a new Blueprint component from the specified class type. The user will be prompted to pick a new subclass name and a blueprint asset will be created

Target is Subobject Data Subsystem

## 图示

![]($-20221218-21054569.png)

## Inputs

In: Exec.

Component Class: Actor Component Class Reference.

New Class Path: String.

New Class Name: String.  

## Outputs

Out: Exec.

Return Value: Object Class Reference. The new class that was created.
