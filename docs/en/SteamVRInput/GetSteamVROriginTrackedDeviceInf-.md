# GetSteamVROriginTrackedDeviceInf-

Returns information about the tracked device associated from the input source.

Target is Steam VRInput Device Function Library

## 图示

![]($-20221218-21044182.png)

## Inputs

In: Exec.

Steam VRAction: Steam VRAction Structure. The action that's the source of the input.  

## Outputs

Out: Exec.

Input Origin Info: Steam VRInput Origin Info Structure.

Return Value: Boolean. InputOriginInfo - The origin info of the action.
