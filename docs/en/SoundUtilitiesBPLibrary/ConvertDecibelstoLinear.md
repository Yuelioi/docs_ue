# ConvertDecibelstoLinear

Converts decibel to linear scale

Target is Sound Utilities BPFunction Library

## 图示

![]($-20221218-20593236.png)

## Inputs

In: Exec.

In Decibels: Float (single-precision). The decibels to convert to a linear gain scalar.  

## Outputs

Out: Exec.

Return Value: Float (single-precision). The resulting linear gain.
