# SetSwapEyes

Set swap eyes state

Target is NDisplay View Origin

## 图示

![]($-20221218-21050052.png)

## Inputs

In: Exec.

Target: NDisplay View Origin Object Reference.

Swap Eyes: Boolean. New eyes swap state. False - normal eyes left|right, true - swapped eyes right|left.  

## Outputs

Out: Exec.
