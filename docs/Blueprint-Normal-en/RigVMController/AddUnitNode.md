# AddUnitNode

Adds a Function / Struct Node to the edited Graph.. UnitNode represent a RIGVM_METHOD declaration on a USTRUCT.. This causes a NodeAdded modified event.

Target is Rig VMController

## Inputs

In: Exec.

Target: Rig VMController Object Reference.

In Script Struct: Script Struct Object Reference.

In Method Name: Name (by ref).

In Position: Vector 2D Structure (by ref).

In Node Name: String.

Setup Undo Redo: Boolean.

Print Python Command: Boolean.  

## Outputs

Out: Exec.

Return Value: Rig VMUnit Node Object Reference. Adds a Function / Struct Node to the edited Graph.. UnitNode represent a RIGVM_METHOD declaration on a USTRUCT.. This causes a NodeAdded modified event..
