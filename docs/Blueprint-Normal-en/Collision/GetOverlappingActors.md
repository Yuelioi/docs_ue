# GetOverlappingActors

Returns list of actors this actor is overlapping (any component overlapping any component). Does not return itself.

Target is Actor

## 图示

![]($-20221218-18190430.png)

## Inputs

Target: Actor Object Reference.

Class Filter: Actor Class Reference. [optional] If set, only returns actors of this class or subclasses.  

## Outputs

Overlapping Actors: Array of Actor Object References. [out] Returned list of overlapping actors.
