# CeiltoInteger64

Rounds A up towards positive infinity / up to the next integer (e.g., -1.6 becomes -1 and 1.6 becomes 2)

Target is Kismet Math Library

![]($-20221218-19493735.png)

## Inputs

A: Float (double-precision).  

## Outputs

Return Value: Integer64. Rounds A up towards positive infinity / up to the next integer (e.g., -1.6 becomes -1 and 1.6 becomes 2).
