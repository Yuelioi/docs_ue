# Lerp_Vector

Linearly interpolates between A and B based on Alpha (100% of A when Alpha=0 and 100% of B when Alpha=1)

Target is Kismet Math Library

## 图示

![]($-20221218-19570229.png)

## Inputs

A: Vector.

B: Vector.

Alpha: Float (single-precision).  

## Outputs

Return Value: Vector. Linearly interpolates between A and B based on Alpha (100% of A when Alpha=0 and 100% of B when Alpha=1).
