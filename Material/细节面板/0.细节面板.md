## 物理材质 Physical Material

用于此材质的物理材质类型。

![[../../../_Config/Attatchment/Pasted image 20220419035238.png]]

| 属性         | 描述                                                                                                                                                   |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 物理材料     | 与此材料相关的物理材料。物理材料提供了物理属性的定义<br>例如在碰撞（弹力）和其他基于物理的方面保留了多少能量。<br>物理材质对材质在视觉上的显示方式没有影响。 |
| 物理材质遮罩 | 用于此图形材质的物理材质遮罩。用于声音、效果等。                                                                                                       |
