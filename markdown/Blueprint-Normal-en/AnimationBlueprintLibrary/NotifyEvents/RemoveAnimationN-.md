# RemoveAnimationN-

Removes Animation Notify Events found by Name within the Animation Sequence, and returns the number of removed name instances

Target is Animation Blueprint Library

![]($-20221218-17523288.png)

## Inputs

In: Exec.

Animation Sequence Base: Anim Sequence Base Object Reference.

Notify Name: Name.  

## Outputs

Out: Exec.

Return Value: Integer. Removes Animation Notify Events found by Name within the Animation Sequence, and returns the number of removed name instances.
