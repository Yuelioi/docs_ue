# SetLFOGain

Sets the LFO gain factor

Target is Modular Synth Component

![]($-20221218-21075096.png)

## Inputs

In: Exec.

Target: Modular Synth Component Object Reference.

LFOIndex: Integer. Which LFO to set the frequency for..

Gain: Float (single-precision). The gain factor to use for the LFO..  

## Outputs

Out: Exec.
