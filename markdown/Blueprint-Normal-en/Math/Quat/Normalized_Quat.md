# Normalized_Quat

Get a normalized copy of this quaternion.. If it is too small, returns an identity quaternion.

Target is Kismet Math Library

![]($-20221218-19530464.png)

## Inputs

Q: Quat Structure (by ref).

Tolerance: Float (single-precision). Minimum squared length of quaternion for normalization..  

## Outputs

Return Value: Quat Structure.
