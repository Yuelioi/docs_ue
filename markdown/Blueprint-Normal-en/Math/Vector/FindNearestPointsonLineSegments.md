# FindNearestPointsonLineSegments

Find closest points between 2 segments.

Target is Kismet Math Library

## 图示

![]($-20221218-19564023.png)

## Inputs

Segment 1Start: Vector. Start of the 1st segment..

Segment 1End: Vector. End of the 1st segment..

Segment 2Start: Vector. Start of the 2nd segment..

Segment 2End: Vector. End of the 2nd segment..  

## Outputs

Segment 1Point: Vector. Closest point on segment 1 to segment 2..

Segment 2Point: Vector. Closest point on segment 2 to segment 1..
