# RGBtoHSV

Breaks apart a color into individual HSV components (as well as alpha) (Hue is [0..360) while Saturation and Value are 0..1)

Target is Kismet Math Library

![]($-20221218-19475488.png)

## Inputs

In Color: Linear Color Structure.  

## Outputs

H: Float (single-precision).

S: Float (single-precision).

V: Float (single-precision).

A: Float (single-precision).
