# IsUnitXYZ_Vector4

Determines if vector is normalized / unit (length 1) within specified squared tolerance. The W element is ignored.

Target is Kismet Math Library

## 图示

![]($-20221218-19591072.png)

## Inputs

A: Vector 4 Structure (by ref).

Squared Lenth Tolerance: Float (single-precision).  

## Outputs

Return Value: Boolean. true if unit, false otherwise..
