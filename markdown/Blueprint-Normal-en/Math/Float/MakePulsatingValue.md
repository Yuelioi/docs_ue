# MakePulsatingValue

Simple function to create a pulsating scalar value

Target is Kismet Math Library

## 图示

![]($-20221218-19495963.png)

## Inputs

In Current Time: Float (single-precision). Current absolute time.

In Pulses Per Second: Float (single-precision). How many full pulses per second?.

In Phase: Float (single-precision). Optional phase amount, between 0.0 and 1.0 (to synchronize pulses).  

## Outputs

Return Value: Float (single-precision). Pulsating value (0.0-1.0).
