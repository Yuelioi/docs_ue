# MakeRotfromY

Builds a rotation matrix given only a YAxis. X and Z are unspecified but will be orthonormal. YAxis need not be normalized.

Target is Kismet Math Library

## Inputs

Y: Vector (by ref).  

## Outputs

Return Value: Rotator. Builds a rotation matrix given only a YAxis. X and Z are unspecified but will be orthonormal. YAxis need not be normalized..
