# ToDirectionAndLength

Util to convert this vector into a unit direction vector and its original length.

Target is Kismet Math Library

## Inputs

A: Vector 2D Structure.  

## Outputs

Out Dir: Vector 2D Structure. Reference passed in to store unit direction vector..

Out Length: Float (double-precision). Reference passed in to store length of the vector..
