# SetBrushfromAtlasInterface

Sets the Brush to the specified Atlas Region.

Target is Image

## 图示

![]($-20221218-17543116.png)

## Inputs

In: Exec.

Target: Image Object Reference.

Atlas Region: Slate Texture Atlas Interface Interface. Region of the Atlas to use to set on Brush..

Match Size: Boolean. If true, image will change its size to atlas region size. If false, atlas region will be stretched to image size..  

## Outputs

Out: Exec.
