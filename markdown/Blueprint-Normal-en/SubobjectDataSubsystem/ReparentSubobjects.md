# ReparentSubobjects

Attempts to reparent all subobjects in the HandlesToMove array to the new parent handle.

Target is Subobject Data Subsystem

![]($-20221218-21060512.png)

## Inputs

In: Exec.

Target: Subobject Data Subsystem Object Reference.

Params: Reparent Subobject Params Structure (by ref).

Handles to Move: Array of Subobject Data Handle Structures.  

## Outputs

Out: Exec.

Return Value: Boolean.
