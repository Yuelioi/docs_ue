# AddMappedSkeletalMeshBoneArray

Adds a series of mapped bones to the rig, should not be used before OnPreInitialize Event

Target is Control Rig Component

![]($-20221218-18310921.png)

## Inputs

In: Exec.

Target: Control Rig Component Object Reference.

Skeletal Mesh Component: Skeletal Mesh Component Object Reference.

Bones: Array of Mapped Bone Structures.

Curves: Array of Mapped Curve Structures.  

## Outputs

Out: Exec.
