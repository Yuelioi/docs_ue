# FindorGetDefaultSettingforShot

Allows access to a setting of provided type for specific shot.

Target is Movie Pipeline Blueprint Library

## 图示

![]($-20221218-20082820.png)

## Inputs

In: Exec.

In Setting Type: Movie Pipeline Setting Class Reference.

In Master Config: Movie Pipeline Master Config Object Reference.

In Shot: Movie Pipeline Executor Shot Object Reference.  

## Outputs

Out: Exec.

Return Value: Movie Pipeline Setting Object Reference. Allows access to a setting of provided type for specific shot..
