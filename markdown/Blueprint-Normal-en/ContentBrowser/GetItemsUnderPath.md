# GetItemsUnderPath

Get the items (folders and/or files) that exist under the given virtual path.

Target is Content Browser Data Subsystem

![]($-20221218-18302409.png)

## Inputs

Target: Content Browser Data Subsystem Object Reference.

In Path: Name.

In Filter: Content Browser Data Filter Structure (by ref).  

## Outputs

Return Value: Array of Content Browser Item Structures. Get the items (folders and/or files) that exist under the given virtual path..
