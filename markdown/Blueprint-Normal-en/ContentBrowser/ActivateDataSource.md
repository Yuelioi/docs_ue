# ActivateDataSource

Attempt to activate the named data source.

Target is Content Browser Data Subsystem

## 图示

![]($-20221218-18301466.png)

## Inputs

In: Exec.

Target: Content Browser Data Subsystem Object Reference.

Name: Name.  

## Outputs

Out: Exec.

Return Value: Boolean. True if the data source was available and not already active, false otherwise..
