# Sendbinaryeventtoaspecifichost

Sends binary cluster event to a specific target (outside of the cluster).

Target is Display Cluster Blueprint API

![]($-20221218-20110193.png)

## Inputs

In: Exec.

Target: Display Cluster Blueprint API Interface.

Address: String.

Port: Integer.

Event: Display Cluster Cluster Event Binary Structure (by ref).

Primary Only: Boolean.  

## Outputs

Out: Exec.
