# SetControlShapeTransform

Sets the shape transform for a given control element by key

Target is Rig Hierarchy

## 图示

![]($-20221218-21200701.png)

## Inputs

In: Exec.

Target: Rig Hierarchy Object Reference.

In Key: Rig Element Key Structure. The key of the control element to set the shape transform for.

In Transform: Transform. The new shape transform value to set.

Initial: Boolean. If true the initial value will be used.

Setup Undo: Boolean. If true the transform stack will be setup for undo / redo.  

## Outputs

Out: Exec.
