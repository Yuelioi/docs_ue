# GetTrackRowDisplayName

Get this track row's display name

Target is Movie Scene Track Extensions

## 图示

![]($-20221218-20560051.png)

## Inputs

In: Exec.

Track: Movie Scene Track Object Reference. The track to use.

Row Index: Integer. The row index for the track.  

## Outputs

Out: Exec.

Return Value: Text. This track's display name.
