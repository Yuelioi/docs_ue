# GetMasterTracks

Get all master tracks

Target is Movie Scene Sequence Extensions

![]($-20221218-20543086.png)

## Inputs

In: Exec.

Sequence: Movie Scene Sequence Object Reference. The sequence to use.  

## Outputs

Out: Exec.

Return Value: Array of Movie Scene Track Object References. An array containing all master tracks in this sequence.
