# RemoveChildObjectBinding

Remove an object binding from the given folder

Target is Movie Scene Folder Extensions

## 图示

![]($-20221218-20551058.png)

## Inputs

In: Exec.

Folder: Movie Scene Folder Object Reference. The folder from which to remove an object binding.

In Object Binding: Sequencer Binding Proxy Structure. The object binding to remove.  

## Outputs

Out: Exec.

Return Value: Boolean. True if the operation succeeds.
