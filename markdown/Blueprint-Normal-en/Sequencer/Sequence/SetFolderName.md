# SetFolderName

Set the name of the given folder

Target is Movie Scene Folder Extensions

## 图示

![]($-20221218-20552588.png)

## Inputs

In: Exec.

Folder: Movie Scene Folder Object Reference. The folder to set the name of.

In Folder Name: Name. The new name for the folder.  

## Outputs

Out: Exec.

Return Value: Boolean. True if the setting of the folder name succeeds.
