# VisLogLocation

Logs location as sphere with given radius - recording for Visual Logs has to be enabled to record this data

Target is Visual Logger Kismet Library

## 图示

![]($-20221218-18412624.png)

## Inputs

In: Exec.

World Context Object: Object Reference.

Location: Vector.

Text: String.

Object Color: Linear Color Structure.

Radius: Float (single-precision).

Log Category: Name.

Add to Message Log: Boolean.  

## Outputs

Out: Exec.
