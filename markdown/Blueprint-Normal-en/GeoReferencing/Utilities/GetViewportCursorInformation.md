# GetViewportCursorInformation

Retrieve information about the viewport under the mouse cursor.

Target is Geo Referencing Editor BPLibrary

## 图示

![]($-20221218-19143934.png)

## Inputs

In: Exec.  

## Outputs

Out: Exec.

Focused: Boolean. If the Level editor not are in focus it will return false..

Screen Location: Vector 2D Structure. Viewport-Space position of cursor..

World Location: Vector. Location of viewport origin (camera) in world space..

World Direction: Vector. Direction of viewport (camera) in world space..
