## Actions

Is Active. Returns if this output provider is currently active or not: Target is VCam Output Provider Base.

Is Initialized. Returns if this output provider has been initialized or not: Target is VCam Output Provider Base.

Set Active. Called to turn on or off this output provider: Target is VCam Output Provider Base.

Set Target Camera. Sets the TargetCamera parameter: Target is VCam Output Provider Base.

Set UMGClass. Sets the UMG class to be rendered in this output provider: Target is VCam Output Provider Base.
