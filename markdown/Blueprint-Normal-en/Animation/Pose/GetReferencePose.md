# GetReferencePose

Populates an Anim Pose with the reference poses stored for the provided USkeleton

Target is Anim Pose Extensions

![]($-20221218-17503455.png)

## Inputs

Skeleton: Skeleton Object Reference. USkeleton object to retrieve the reference pose from.  

## Outputs

Out Pose: Anim Pose Structure. Anim pose to hold the reference pose.
