# GetNodesOfClass

Returns all Animation Graph Nodes of the provided Node Class contained by the Animation Blueprint

Target is Animation Blueprint Library

## 图示

![]($-20221218-17495489.png)

## Inputs

In: Exec.

Animation Blueprint: Anim Blueprint Object Reference.

Node Class: Anim Graph Node Base Class Reference.

Include Child Classes: Boolean.  

## Outputs

Out: Exec.

Graph Nodes: Array of Anim Graph Node Base Object References.
