# GetNumUVChannels

Returns the number of UV channels for the given LOD of a StaticMesh.

Target is Static Mesh Editor Subsystem

## Inputs

In: Exec.

Target: Static Mesh Editor Subsystem Object Reference.

Static Mesh: Static Mesh Object Reference. Static mesh to query..

LODIndex: Integer. Index of the StaticMesh LOD..  

## Outputs

Out: Exec.

Return Value: Integer. the number of UV channels..
