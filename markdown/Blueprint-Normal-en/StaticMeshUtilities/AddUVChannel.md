# AddUVChannel

Adds an empty UV channel at the end of the existing channels on the given LOD of a StaticMesh.

Target is Static Mesh Editor Subsystem

## 图示

![]($-20221218-21032613.png)

## Inputs

In: Exec.

Target: Static Mesh Editor Subsystem Object Reference.

Static Mesh: Static Mesh Object Reference. Static mesh on which to add a UV channel..

LODIndex: Integer. Index of the StaticMesh LOD..  

## Outputs

Out: Exec.

Return Value: Boolean. true if a UV channel was added..
