# GetClassfromPrimaryAssetId

Returns the Blueprint Class associated with a Primary Asset Id, this will only return a valid object if it is in memory, it will not load it

Target is Kismet System Library

## 图示

![]($-20221218-18000211.png)

## Inputs

Primary Asset Id: Primary Asset Id Structure.  

## Outputs

Return Value: Object Class Reference. Returns the Blueprint Class associated with a Primary Asset Id, this will only return a valid object if it is in memory, it will not load it.
