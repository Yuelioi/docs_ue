## Actions

Map Key. Map a key to an action within the mapping context.: Target is Input Mapping Context.

Unmap Action. Unmap all key maps to an action within the mapping context.: Target is Input Mapping Context.

Unmap All. Unmap everything within the mapping context.: Target is Input Mapping Context.

Unmap Key. Unmap a key from an action within the mapping context.: Target is Input Mapping Context.
