# IsElementSelected

Test to see whether the given element is currently considered selected.

Target is Typed Element Selection Set

## 图示

![]($-20221218-21170795.png)

## Inputs

Target: Typed Element Selection Set Object Reference.

In Element Handle: Script Typed Element Handle Structure (by ref).

In Selection Options: Typed Element Is Selected Options Structure.  

## Outputs

Return Value: Boolean. Test to see whether the given element is currently considered selected..
