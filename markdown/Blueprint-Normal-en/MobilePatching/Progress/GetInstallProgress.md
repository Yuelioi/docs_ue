# GetInstallProgress

Get the current installation progress. Between 0 and 1 for known progress, or less than 0 for unknown progress

Target is Mobile Pending Content

## Outputs

Return Value: Float (single-precision). Get the current installation progress. Between 0 and 1 for known progress, or less than 0 for unknown progress.
