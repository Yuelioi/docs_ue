# GetMatrixProperties

Gets the Matrix Fixture properties, returns false if the patch is not using a matrix fixture

Target is DMX Fixture Patch

![]($-20221218-18434949.png)

## Inputs

Target: DMX Fixture Patch Object Reference.  

## Outputs

Matrix Properties: DMXFixture Matrix Structure.

Return Value: Boolean. Gets the Matrix Fixture properties, returns false if the patch is not using a matrix fixture.
