# GetTextureCenterColors

Reads pixel color in the middle of each "Texture" and output linear colors

Target is DMXFixture Component

## 图示

![]($-20221218-18442896.png)

## Inputs

In: Exec.

Target: DMXFixture Component Object Reference.

Texture Atlas: Texture 2D Object Reference.

Num Textures: Integer.  

## Outputs

Out: Exec.

Return Value: Array of Linear Color Structures. Reads pixel color in the middle of each "Texture" and output linear colors.
