# SetPointCloudwithColors

Updates the point cloud data with the new set of points and colors

Target is Point Cloud Component

## 图示

![]($-20221218-20301683.png)

## Inputs

In: Exec.

Target: Point Cloud Component Object Reference.

Points: Array of Vectors. the new set of points to use.

Colors: Array of Color Structures. the new set of colors to use, must match points size.  

## Outputs

Out: Exec.
