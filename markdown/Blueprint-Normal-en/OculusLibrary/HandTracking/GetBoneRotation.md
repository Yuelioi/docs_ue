# GetBoneRotation

Get the rotation of a specific bone

Target is Oculus Input Function Library

## 图示

![]($-20221218-20152493.png)

## Inputs

Device Hand: EOculusHandType Enum. (in) The hand to get the rotations from.

Bone Id: EBone Enum. (in) The specific bone to get the rotation from.

Controller Index: Integer. (in) Optional different controller index.  

## Outputs

Return Value: Quat Structure.
