# AreAnyListenersWithinRange

Determines if any audio listeners are within range of the specified location

Target is Gameplay Statics

## 图示

![]($-20221218-18023148.png)

## Inputs

In: Exec.

Location: Vector (by ref). The location from which test if a listener is in range.

Maximum Range: Float (single-precision). The distance away from Location to test if any listener is within.  

## Outputs

Out: Exec.

Return Value: Boolean.
