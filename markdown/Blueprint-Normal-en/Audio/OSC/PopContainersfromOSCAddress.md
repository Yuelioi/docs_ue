# PopContainersfromOSCAddress

Pops container from ordered array of containers. If NumContainers is greater than or equal to the number of containers in address, returns all containers.

Target is OSCManager

![]($-20221218-18061827.png)

## Inputs

In: Exec.

Address: OSCAddress Structure (by ref).

Num Containers: Integer.  

## Outputs

Out: Exec.

Containers: Array of Strings. Pops container from ordered array of containers. If NumContainers is greater than or equal to the number of containers in address, returns all containers..
