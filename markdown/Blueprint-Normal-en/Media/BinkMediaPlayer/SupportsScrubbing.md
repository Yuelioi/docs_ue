# SupportsScrubbing

Checks whether the currently loaded media supports scrubbing.

Target is Bink Media Player

## Inputs

Target: Bink Media Player Object Reference.  

## Outputs

Return Value: Boolean. true if scrubbing is supported, false otherwise. See: SupportsRate, SupportsSeeking.
