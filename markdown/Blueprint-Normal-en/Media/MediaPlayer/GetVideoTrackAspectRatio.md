# GetVideoTrackAspectRatio

Get the aspect ratio of the specified video track.

Target is Media Player

## 图示

![]($-20221218-20005785.png)

## Inputs

Target: Media Player Object Reference.

Track Index: Integer. Index of the video track, or INDEX_NONE for the selected one..

Format Index: Integer. Index of the track format, or INDEX_NONE for the selected one..  

## Outputs

Return Value: Float (single-precision). Aspect ratio..
