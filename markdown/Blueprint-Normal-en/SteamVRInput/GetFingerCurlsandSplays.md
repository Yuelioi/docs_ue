# GetFingerCurlsandSplays

Get the finger curl and splay for a give hand in the current frame

Target is Steam VRInput Device Function Library

## 图示

![]($-20221218-21042740.png)

## Inputs

In: Exec.

Hand: EHand Enum. Which hand to get the finger curls and splay values for.

Summary Data Type: ESkeletalSummaryDataType Enum.  

## Outputs

Out: Exec.

Finger Curls: Steam VRFinger Curls Structure. Curl values for each finger pair this frame.

Finger Splays: Steam VRFinger Splays Structure.
