# CopyParams

Copies all properties from one constraint to another

Target is Constraint Instance Blueprint Library

## 图示

![]($-20221218-20232202.png)

## Inputs

In: Exec.

Accessor: Constraint Instance Accessor Structure (by ref). Constraint accessor to write to.

Source Accessor: Constraint Instance Accessor Structure (by ref). Constraint accessor to read from.

Keep Position: Boolean. Whether to keep original constraint positions.

Keep Rotation: Boolean. Whether to keep original constraint rotations.  

## Outputs

Out: Exec.
