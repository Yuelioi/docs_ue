# GetDatasmithUserData

Get the Datasmith User Data of a given object

Target is Datasmith Content Blueprint Library

## 图示

![]($-20221218-18405973.png)

## Inputs

In: Exec.

Object: Object Reference. The Object from which to retrieve the Datasmith User Data..  

## Outputs

Out: Exec.

Return Value: Datasmith User Data Object Reference. The Datasmith User Data if it exists; nullptr, otherwise.
