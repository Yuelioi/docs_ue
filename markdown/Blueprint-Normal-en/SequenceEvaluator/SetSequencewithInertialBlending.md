# SetSequencewithInertialBlending

Set the current sequence of the sequence evaluator with an inertial blend time

Target is Sequence Evaluator Library

## Inputs

In: Exec.

Update Context: Anim Update Context Structure (by ref).

Sequence Evaluator: Sequence Evaluator Reference Structure (by ref).

Sequence: Anim Sequence Base Object Reference.

Blend Time: Float (single-precision).  

## Outputs

Out: Exec.

Return Value: Sequence Evaluator Reference Structure. Set the current sequence of the sequence evaluator with an inertial blend time.
