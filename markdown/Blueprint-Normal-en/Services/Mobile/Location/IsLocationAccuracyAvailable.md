# IsLocationAccuracyAvailable

Checks if the supplied Accuracy is available on the current device.

Target is Location Services

## 图示

![]($-20221218-20563290.png)

## Inputs

In: Exec.

Accuracy: ELocationAccuracy Enum. the accuracy to check.  

## Outputs

Out: Exec.

Return Value: Boolean. true if the mobile device can support the Accuracy, false if it will use a different accuracy.
