# SetIsSelected

Change the selected state manually.

Target is Common Button Base

## Inputs

In: Exec.

Target: Common Button Base Object Reference.

In Selected: Boolean.

Give Click Feedback: Boolean. If true, the button may give user feedback as if it were clicked. IE: Play a click sound, trigger animations as if it were clicked..

Out: Exec.
