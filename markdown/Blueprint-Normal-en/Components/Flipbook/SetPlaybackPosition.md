# SetPlaybackPosition

Jump to a position in the flipbook (expressed in seconds). If bFireEvents is true, event functions will fire, otherwise they will not.

Target is Paper Flipbook Component

![]($-20221218-18234726.png)

## Inputs

In: Exec.

Target: Paper Flipbook Component Object Reference.

New Position: Float (single-precision).

Fire Events: Boolean.  

## Outputs

Out: Exec.
