# ShowMaterialSection

Allows hiding of a particular material (by ID) on this instance of a SkeletalMesh.

Target is Skinned Mesh Component

## 图示

![]($-20221218-18271507.png)

## Inputs

In: Exec.

Target: Skinned Mesh Component Object Reference.

Material ID: Integer. Index of the material show/hide.

Section Index: Integer.

Show: Boolean. True to show the material, false to hide it.

LODIndex: Integer. Index of the LOD to modify material visibility within.  

## Outputs

Out: Exec.
