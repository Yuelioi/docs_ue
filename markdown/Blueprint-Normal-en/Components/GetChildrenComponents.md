# GetChildrenComponents

Gets all components that are attached to this component, possibly recursively

Target is Scene Component

## 图示

![]($-20221218-18242638.png)

## Inputs

Target: Scene Component Object Reference.

Include All Descendants: Boolean. Whether to include all descendants in the list of children (i.e. grandchildren, great grandchildren, etc.).  

## Outputs

Children: Array of Scene Component Object References. The list of attached child components.
