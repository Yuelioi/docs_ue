# RemoveNodebyName

Removes a node from the graph given the node's name.. This causes a NodeRemoved modified event.

Target is Rig VMController

In: Exec.

Target: Rig VMController Object Reference.

In Node Name: Name (by ref).

Setup Undo Redo: Boolean.

Recursive: Boolean.

Print Python Command: Boolean.

Relink Pins: Boolean.  

## Outputs

Out: Exec.

Return Value: Boolean. Removes a node from the graph given the node's name.. This causes a NodeRemoved modified event..
