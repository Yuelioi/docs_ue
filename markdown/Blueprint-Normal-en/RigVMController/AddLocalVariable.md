# AddLocalVariable

Add a local variable to the graph

Target is Rig VMController

## 图示

![]($-20221218-20420269.png)

## Inputs

In: Exec.

Target: Rig VMController Object Reference.

In Variable Name: Name (by ref).

In CPPType: String.

In CPPType Object: Object Reference.

In Default Value: String.

Setup Undo Redo: Boolean.

Print Python Command: Boolean.  

## Outputs

Out: Exec.

Return Value: Rig VMGraph Variable Description Structure. Add a local variable to the graph.
