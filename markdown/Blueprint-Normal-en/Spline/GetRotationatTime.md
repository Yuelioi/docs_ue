# GetRotationatTime

Given a time from 0 to the spline duration, return a rotation corresponding to the spline's position and direction there.

Target is Spline Component

![]($-20221218-21005603.png)

## Inputs

Target: Spline Component Object Reference.

Time: Float (single-precision).

Coordinate Space: ESplineCoordinateSpace Enum.

Use Constant Velocity: Boolean.  

## Outputs

Return Value: Rotator. Given a time from 0 to the spline duration, return a rotation corresponding to the spline's position and direction there..
