# MoveRight

Input callback to strafe right in local space (or left if Val is negative).

Target is Default Pawn

## 图示

![]($-20221218-20193266.png)

## Inputs

In: Exec.

Target: Default Pawn Object Reference.

Val: Float (single-precision). Amount of movement in the right direction (or left if negative)..  

## Outputs

Out: Exec.
