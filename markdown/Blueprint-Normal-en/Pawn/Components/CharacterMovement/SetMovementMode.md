# SetMovementMode

Change movement mode.

Target is Character Movement Component

## 图示

![]($-20221218-20183792.png)

## Inputs

In: Exec.

Target: Character Movement Component Object Reference.

New Movement Mode: EMovementMode Enum. The new movement mode.

New Custom Mode: Byte. The new custom sub-mode, only applicable if NewMovementMode is Custom..  

## Outputs

Out: Exec.
