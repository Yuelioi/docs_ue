## Actions

Script Extract Display Labels. Extract all the filename labels.: Target is Interchange Asset Import Data.

Script Extract Filenames. Extract all the (resolved) filenames.: Target is Interchange Asset Import Data.

Script Get First Filename. Return the first filename stored in this data. The resulting filename will be absolute (ie, not relative to the asset).: Target is Interchange Asset Import Data.
