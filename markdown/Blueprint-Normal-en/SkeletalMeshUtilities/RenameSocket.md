# RenameSocket

Rename a socket within a skeleton

Target is Skeletal Mesh Editor Subsystem

## 图示

![]($-20221218-20583150.png)

## Inputs

In: Exec.

Skeletal Mesh: Skeletal Mesh Object Reference. The mesh inside which we are renaming a socket.

Old Name: Name. The old name of the socket.

New Name: Name. The new name of the socket.  

## Outputs

Out: Exec.

Return Value: Boolean. true if the renaming succeeded..
