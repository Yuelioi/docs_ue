# FilterbySelection

Filter the array based on Object's selection.

Target is Editor Filter Library

## 图示

![]($-20221218-18542572.png)

## Inputs

In: Exec.

Target Array: Array of Actor Object References. Array of Object to filter. The array will not change..

Filter Type: EEditorScriptingFilterType Enum. Should include or not the array's item if it respects the condition..  

## Outputs

Out: Exec.

Return Value: Array of Actor Object References. The filtered list..
