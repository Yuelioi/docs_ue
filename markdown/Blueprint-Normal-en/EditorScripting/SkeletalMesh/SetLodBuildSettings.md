# SetLodBuildSettings

Set the LOD build options for the specified LOD index.

Target is Skeletal Mesh Editor Subsystem

## 图示

![]($-20221218-18540237.png)

## Inputs

In: Exec.

Skeletal Mesh: Skeletal Mesh Object Reference. Mesh to process..

Lod Index: Integer. The LOD we will apply the build settings..

Build Options: Skeletal Mesh Build Settings Structure (by ref). The build settings we want to apply to the LOD..  

## Outputs

Out: Exec.
