# GetControlRigs

Get all of the control rigs and their bindings in the level sequence

Target is Control Rig Sequencer Editor Library

## 图示

![]($-20221218-18522751.png)

## Inputs

In: Exec.

Level Sequence: Level Sequence Object Reference. The movie scene sequence to look for Control Rigs.  

## Outputs

Out: Exec.

Return Value: Array of Control Rig Sequencer Binding Proxy Structures. returns list of Control Rigs in the level sequence..
