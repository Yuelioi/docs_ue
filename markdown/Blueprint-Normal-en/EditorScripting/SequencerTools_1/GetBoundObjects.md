# GetBoundObjects

  * Retrieve all objects currently bound to the specified binding identifiers. The sequence will be evaluated in lower bound of the specified range,

  * which allows for retrieving spawnables in that period of time.





Target is Sequencer Tools Function Library

## Inputs

In: Exec.

In World: World Object Reference.

In Sequence: Level Sequence Object Reference.

In Bindings: Array of Sequencer Binding Proxy Structures.

In Range: Sequencer Scripting Range Structure (by ref).  

## Outputs

Out: Exec.

Return Value: Array of Sequencer Bound Objects Structures. Retrieve all objects currently bound to the specified binding identifiers. The sequence will be evaluated in lower bound of the specified range,: which allows for retrieving spawnables in that period of time..
