# GetPathNameforLoadedAsset

Return a valid AssetPath for a loaded asset. The asset need to be a valid asset in the Content Browser.. Similar to GetPathName(). The format will be: /Game/MyFolder/MyAsset.MyAsset

Target is Editor Asset Library

![]($-20221218-18471841.png)

## Inputs

In: Exec.

Loaded Asset: Object Reference. Loaded Asset that exist in the Content Browser..  

## Outputs

Out: Exec.

Return Value: String. If valid, the asset Path of the loaded asset..
