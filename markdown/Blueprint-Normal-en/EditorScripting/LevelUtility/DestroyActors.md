# DestroyActors

Destroy the actors from the world editor. Notify the Editor that the actor got destroyed.

Target is Editor Actor Subsystem

![]($-20221218-18503858.png)

## Inputs

In: Exec.

Target: Editor Actor Subsystem Object Reference.

Actors to Destroy: Array of Actor Object References. Actors to destroy..  

## Outputs

Out: Exec.

Return Value: Boolean. True if the operation succeeds..
