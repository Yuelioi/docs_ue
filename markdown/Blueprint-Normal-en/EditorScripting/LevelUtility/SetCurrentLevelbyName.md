# SetCurrentLevelbyName

Set the current level used by the world editor.. If more than one level shares the same name, the first one encounter of that level name will be used.

Target is Level Editor Subsystem

## 图示

![]($-20221218-18513868.png)

## Inputs

In: Exec.

Target: Level Editor Subsystem Object Reference.

Level Name: Name. The name of the Level the actor belongs to (same name as in the ContentBrowser)..  

## Outputs

Out: Exec.

Return Value: Boolean. True if the operation succeeds..
