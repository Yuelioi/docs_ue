# SetComponentTransform

Sets the world transform of the given component, if possible.

Target is Editor Actor Subsystem

## 图示

![]($-20221218-18513739.png)

## Inputs

In: Exec.

Target: Editor Actor Subsystem Object Reference.

In Scene Component: Scene Component Object Reference.

In World Transform: Transform (by ref).  

## Outputs

Out: Exec.

Return Value: Boolean. false if the world transform could not be set..
