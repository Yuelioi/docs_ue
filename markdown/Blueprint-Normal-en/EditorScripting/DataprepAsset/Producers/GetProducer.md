# GetProducer

Get a producer from a dataprep asset.

Target is Dataprep Core Blueprint Library

![]($-20221218-18491197.png)

## Inputs

In: Exec.

Dataprep Asset Interface: Dataprep Asset Interface Object Reference. The dataprep asset from which the producer will be retrieved.

Index: Integer. The index of the producer in the dataprep asset.  

## Outputs

Out: Exec.

Return Value: Dataprep Content Producer Object Reference.
