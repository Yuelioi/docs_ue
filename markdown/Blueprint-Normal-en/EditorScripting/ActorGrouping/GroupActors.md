# GroupActors

Creates a new group from the provided list of actors removing the actors from any existing groups they are already in

Target is Actor Grouping Utils

![]($-20221218-18464716.png)

## Inputs

In: Exec.

Target: Actor Grouping Utils Object Reference.

Actors to Group: Array of Actor Object References.  

## Outputs

Out: Exec.

Return Value: Group Actor Object Reference. Creates a new group from the provided list of actors removing the actors from any existing groups they are already in.
