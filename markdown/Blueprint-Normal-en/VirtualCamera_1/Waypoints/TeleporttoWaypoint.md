# TeleporttoWaypoint

Teleports the pawn to a specific location.

Target is Virtual Camera Player Controller Base

## 图示

![]($-20221218-21303592.png)

## Inputs

In: Exec.

Target: Virtual Camera Player Controller Base Object Reference.

Waypoint Index: String. The index of the target waypoint in the waypoints array.  

## Outputs

Out: Exec.

Return Value: Boolean. if the teleport was successful, return true.
