# GetAxisStabilizationScale

Gets stabalization scale for a specific axis.

Target is Virtual Camera Player Controller Base

## 图示

![]($-20221218-21285440.png)

## Inputs

Target: Virtual Camera Player Controller Base Object Reference.

Axis to Retrieve: EVirtualCameraAxis Enum. The axis of the stabilization value needed.  

## Outputs

Return Value: Float (single-precision). the stabilization scale for the given axis.
