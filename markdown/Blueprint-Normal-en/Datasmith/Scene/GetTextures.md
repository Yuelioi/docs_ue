# GetTextures

Create an array with all the Textures in the Datasmith scene. Call CreateTexture -or- RemoveTexture to modify the Datasmith scene.

Target is Datasmith Scene Element Base

![]($-20221218-18402243.png)

## Inputs

In: Exec.

Target: Datasmith Scene Element Base Object Reference.  

## Outputs

Out: Exec.

Return Value: Array of Datasmith Texture Element Object References. Create an array with all the Textures in the Datasmith scene. Call CreateTexture -or- RemoveTexture to modify the Datasmith scene..
