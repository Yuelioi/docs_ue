# DrawLine

Draws a 2D line on the HUD.

Target is HUD

## 图示

![]($-20221218-19161466.png)

## Inputs

In: Exec.

Target: HUD Object Reference.

Start Screen X: Float (single-precision). Screen-space X coordinate of start of the line..

Start Screen Y: Float (single-precision). Screen-space Y coordinate of start of the line..

End Screen X: Float (single-precision). Screen-space X coordinate of end of the line..

End Screen Y: Float (single-precision). Screen-space Y coordinate of end of the line..

Line Color: Linear Color Structure. Color to draw line.

Line Thickness: Float (single-precision). Thickness of the line to draw.  

## Outputs

Out: Exec.
