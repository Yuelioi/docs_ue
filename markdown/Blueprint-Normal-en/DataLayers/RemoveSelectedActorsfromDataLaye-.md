# RemoveSelectedActorsfromDataLaye-

Removes the selected actors from the DataLayer.

Target is Data Layer Editor Subsystem

## 图示

![]($-20221218-18351192.png)

## Inputs

In: Exec.

Target: Data Layer Editor Subsystem Object Reference.

Data Layer: Data Layer Object Reference. A DataLayer..  

## Outputs

Out: Exec.

Return Value: Boolean. true if at least one actor was added. false is returned if all selected actors already belong to the DataLayer..
