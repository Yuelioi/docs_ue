# RemoveActorfromDataLayer

Removes an actor from the specified DataLayer.

Target is Data Layer Editor Subsystem

## 图示

![]($-20221218-18350618.png)

## Inputs

In: Exec.

Target: Data Layer Editor Subsystem Object Reference.

Actor: Actor Object Reference. The actor to remove from the provided DataLayer.

Data Layer to Remove: Data Layer Object Reference. The DataLayer to remove the actor from.  

## Outputs

Out: Exec.

Return Value: Boolean. true if the actor was removed from the DataLayer. false is returned if the actor already belonged to the DataLayer..
