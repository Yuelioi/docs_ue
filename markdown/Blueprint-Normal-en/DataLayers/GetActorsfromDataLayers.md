# GetActorsfromDataLayers

Gets all the actors associated with ANY of the specified DataLayers. Analog to AppendActorsFromDataLayers but it returns rather than appends the actors.

Target is Data Layer Editor Subsystem

## 图示

![]($-20221218-18345324.png)

## Inputs

Target: Data Layer Editor Subsystem Object Reference.

Data Layers: Array of Data Layer Object References. The DataLayers to find actors for..  

## Outputs

Return Value: Array of Actor Object References. The list to assign the found actors to..
