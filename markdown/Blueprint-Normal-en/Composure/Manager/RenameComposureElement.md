# RenameComposureElement

Rename a specific composure element

Target is Composure Blueprint Library

## 图示

![]($-20221218-18293454.png)

## Inputs

In: Exec.

Original Element Name: Name. The name of the composure element that we want to rename..

New Element Name: Name. The new name for the composure element..  

## Outputs

Out: Exec.

Return Value: Boolean. bool Whether the renaming operation is successful or not..
