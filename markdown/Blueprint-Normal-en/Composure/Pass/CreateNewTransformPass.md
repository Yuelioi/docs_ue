# CreateNewTransformPass

Create a new Transform pass into the public list which directly shows in the editor.

Target is Empty Comp Shot

## 图示

![]($-20221218-18294097.png)

## Inputs

In: Exec.

Target: Empty Comp Shot Object Reference.

Pass Name: Name. The name for the new pass..

Transform Type: Compositing Element Transform Class Reference. The class type of the created pass..  

## Outputs

Out: Exec.

Return Value: Compositing Element Transform Object Reference. CompositingElementTransform The newly created transform pass object..
