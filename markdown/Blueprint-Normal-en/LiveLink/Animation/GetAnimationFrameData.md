# GetAnimationFrameData

Returns the Subject's frame data stored in the Subject Frame. Returns false if no valid data found.

Target is Live Link Blueprint Library

## Inputs

Subject Frame Handle: Subject Frame Handle Structure (by ref).  

## Outputs

Animation Frame Data: Live Link Animation Frame Data Structure.

Return Value: Boolean. Returns the Subject's frame data stored in the Subject Frame. Returns false if no valid data found..
