# CreateTriangle

Adds a new triangle to the mesh and returns its ID. This will also make an encapsulating polygon, and any missing edges.

Target is Mesh Description Base

![]($-20221218-20032208.png)

## Inputs

In: Exec.

Target: Mesh Description Base Object Reference.

Polygon Group ID: Polygon Group ID Structure.

Vertex Instance IDs: Array of Vertex Instance ID Structures.  

## Outputs

Out: Exec.

New Edge IDs: Array of Edge ID Structures.

Return Value: Triangle ID Structure. Adds a new triangle to the mesh and returns its ID. This will also make an encapsulating polygon, and any missing edges..
