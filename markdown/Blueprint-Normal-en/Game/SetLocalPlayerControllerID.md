# SetLocalPlayerControllerID

Sets what physical controller ID a player should be using. This only works for local player controllers.

Target is Gameplay Statics

## 图示

![]($-20221218-19074418.png)

## Inputs

In: Exec.

Player: Player Controller Object Reference. The player controller of the player to change the controller ID of.

Controller Id: Integer. The controller ID to assign to this player.  

## Outputs

Out: Exec.
