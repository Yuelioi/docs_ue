# GetInputAnalogKeyState

Returns the analog value for the given key/button. If analog isn't supported, returns 1 for down and 0 for up.

Target is Player Controller

![]($-20221218-19070580.png)

## Inputs

Target: Player Controller Object Reference.

Key: Key Structure.  

## Outputs

Return Value: Float (single-precision). Returns the analog value for the given key/button. If analog isn't supported, returns 1 for down and 0 for up..
