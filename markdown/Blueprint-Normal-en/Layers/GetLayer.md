# GetLayer

Gets the ULayer Object of the named layer

Target is Layers Subsystem

## 图示

![]($-20221218-19374977.png)

## Inputs

Target: Layers Subsystem Object Reference.

Layer Name: Name (by ref). The name of the layer whose ULayer Object is returned.  

## Outputs

Return Value: Layer Object Reference. The ULayer Object of the provided layer name.
