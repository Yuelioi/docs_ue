# CreateLayer

Creates a ULayer Object for the named layer

Target is Layers Subsystem

## 图示

![]($-20221218-19373937.png)

## Inputs

In: Exec.

Target: Layers Subsystem Object Reference.

Layer Name: Name (by ref). The name of the layer to create.  

## Outputs

Out: Exec.

Return Value: Layer Object Reference. The newly created ULayer Object for the named layer.
