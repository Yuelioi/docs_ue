# SetSourceBusSendPostEffect

允许设计者以特定的音频组件实例的声音为目标，设置发送电平（复制的声音音量）到指定的源总线。如果源总线还不是声音发送的一部分，参考将被添加到这个实例的覆盖发送中。这种特殊的发送发生在源效果处理链之后。

目标是音频组件
