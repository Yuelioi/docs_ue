# SetCubemapBlend

通过两个cubemaps之间的混合创建天空照明，只有当SourceType被设置为SpecifiedCubemap时才有效。这可以用来在一天中的不同时段无缝过渡天空照明。调用者应该继续更新混合，直到BlendFraction为0或1，以降低渲染成本。调用者要负责避免因改变源点或终点而产生的弹出。

目标是天灯组件
