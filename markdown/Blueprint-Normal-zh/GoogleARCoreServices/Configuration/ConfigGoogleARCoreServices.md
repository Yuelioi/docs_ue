# ConfigGoogleARCoreServices

用所需的GoogleARCoreServices配置来配置当前的虚幻AR会话。如果有一个正在运行的AR会话，该配置将立即生效。否则，.它将在下一个AR会话运行时生效。

目标是谷歌ARCore服务功能库
