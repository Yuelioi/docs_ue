# InsertModifier

在堆栈中插入一个具有给定名称和索引的修改器。如果这个名字已经被使用，那么这个修改器将不会被添加。索引必须在零和现有修改器的数量之间，包括在内。如果添加成功，返回创建的修改器。

目标是VCAM组件
