# AddProducer

在数据处理资产中添加一个生产者（生产者的行为就像从数据处理编辑器中调用一样，如果你不想要任何ui，就使用自动版本）。

目标是Dataprep核心蓝图库

## 图示

![]($-20221218-18490933.png)

## Inputs

在。执行。

Dataprep资产接口。Dataprep资产接口对象参考。生产者将被添加到的数据准备资产上。

生产者类。Dataprep Content Producer Class Reference。要添加的生产者的类型。

## Outputs

出：执行。

返回值。Dataprep Content Producer对象参考。创建的生产者。
