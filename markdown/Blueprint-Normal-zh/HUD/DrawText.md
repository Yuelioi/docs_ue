# DrawText

在HUD上画一个字符串。

目标是HUD

## 图示

![]($-20221218-19162065.png)

## Inputs

在。执行。

目标：HUD对象参考。

文本。字符串。要画的字符串。

文本颜色：线性颜色结构。绘制字符串的颜色。

屏幕X。Float（单精度）。字符串左上角的屏幕空间X坐标。

屏幕Y。Float（单精度）。屏幕空间的Y坐标，即字符串左上角的坐标。

字体。字体对象参考。用于绘制文本的字体。如果为空，则选择默认字体。

规模。Float (single-precision).缩放倍数，用于控制文本的大小。

规模位置。布尔值。参数 "Scale "是否应同时缩放此绘图调用的位置。
