# ClientPlayForceFeedback

在玩家的控制器上播放一个力反馈图案

目标是玩家控制器

## 图示

![]($-20221218-19054040.png)

## Inputs

在。执行。

目标。玩家控制器对象参考。

力反馈效应。力反馈效果对象参考。要播放的力反馈模式。

标签。名称。一个允许停止一个效果的标签。如果另一个带有此标签的效果正在播放，它将被停止并替换。

循环。布尔型。该模式是否应重复播放或成为一个单一的镜头。

忽略时间膨胀。布尔值。该模式是否应忽略时间膨胀。

暂停时播放。布尔值。游戏暂停时，图案是否应继续播放。  

## Outputs

出：执行。
