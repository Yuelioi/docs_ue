# RegisterNavigationInvoker

将给定的角色注册为 "导航执行者"，这意味着导航系统将确保导航在其周围的指定半径内生成。注意：：你需要导航系统的GenerateNavigationOnlyAroundNavigationInvokers被设置为 "true"，以利用这个功能。

目标是导航系统V1
