# IsVectorValueSet

如果向量值已被设置（且未被清除），该函数返回true（表示该值应该是有效的）。如果没有设置，则矢量值无效，该函数将返回false。(如果指定的键没有容纳一个矢量，也会返回false)。

目标是黑板组件
