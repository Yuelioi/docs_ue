# ReadRenderTargetUV

令人难以置信的低效和缓慢的操作!使用UV[0,1]x[0,1]坐标从渲染目标中读取一个sRGB颜色值。LDR渲染目标被认为是在sRGB空间。而HDR的目标则假定为线性空间。结果是在sRGB空间中每通道8位的[0,255]BGRA。

目标是Kismet Rendering Library
