# BeginDrawCanvastoRenderTarget

返回一个Canvas对象，可以用来绘制到指定的渲染目标上。Canvas 有像 DrawMaterial 一样的函数，带有尺寸参数，可以用来绘制到渲染目标的特定区域。请确保调用 EndDrawCanvasToRenderTarget 来完成渲染工作!

目标是Kismet Rendering Library
