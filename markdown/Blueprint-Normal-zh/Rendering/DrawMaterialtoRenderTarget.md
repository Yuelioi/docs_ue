# DrawMaterialtoRenderTarget

渲染一个四边形，并将该材质应用到指定的渲染目标上。这将设置渲染目标，即使它已经被设置，这是一个昂贵的操作。如果向同一渲染目标渲染多个基元，请使用BeginDrawCanvasToRenderTarget / EndDrawCanvasToRenderTarget。

目标是Kismet Rendering Library
