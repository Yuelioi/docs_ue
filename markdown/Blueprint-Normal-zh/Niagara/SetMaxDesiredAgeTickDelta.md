# SetMaxDesiredAgeTickDelta

设置我们将模拟到期望年龄的最大CPU时间（秒），当我们超过这个限制时，刻度将在下一帧中处理。这只与使用DesiredAge年龄更新模式时有关。注意：如果我们有刻度线要处理，并且SetCanRenderWhileSeeking被设置为 "true"，那么这个组件就不会被看到了。

目标是Niagara粒子系统组件
