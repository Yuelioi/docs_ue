# GetLinearFrequencyClamped

返回输入值的线性频率。将对数域和范围值映射到线性输出（有利于线性。对数频率的滑块表示/视觉化）。与GetLogFrequencyClamped相反。

目标是Sound Utilities BPFunction Library

## 图示

![]($-20221218-20594053.png)

## Inputs

在。执行。

在值。Float（单精度）。要转换为线性频率的对数值。

在领域内。矢量二维结构（通过参考）。在线性尺度和对数尺度之间转换时使用的域。

在范围内。矢量2D结构（通过参考）。在线性尺度和对数尺度之间转换时使用的范围。  

## Outputs

出：执行。

返回值。Float（单精度）。给定的对数输入的线性频率。
