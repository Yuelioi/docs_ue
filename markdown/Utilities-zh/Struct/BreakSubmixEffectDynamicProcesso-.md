# BreakSubmixEffectDynamicProcesso-

Adds a node that breaks a 'SubmixEffectDynamicProcessorFilterSettings' into its member fields

## 图示

![]($-20221218-14435921.png)

## Inputs

Submix Effect Dynamic Processor Filter Settings: Submix Effect Dynamic Processor Filter Settings Structure (by ref).  

## Outputs

Enabled: Boolean. Enabled:. Whether or not filter is enabled.

Cutoff (Hz): Float (single-precision). Cutoff:. The cutoff frequency of the HPF applied to key signal.

Gain (D B): Float (single-precision). Gain Db:. The gain of the filter shelf applied to the key signal.
