# MakeLiveLinkBasicBlueprintData

Adds a node that create a 'LiveLinkBasicBlueprintData' from its members

## 图示

![]($-20221218-14565029.png)

## Inputs

Static Data: Live Link Base Static Data Structure. Static Data:. Static data that should not change every frame.

Frame Data: Live Link Base Frame Data Structure. Frame Data:. Dynamic data that can change every frame.  

## Outputs

Live Link Basic Blueprint Data: Live Link Basic Blueprint Data Structure.
