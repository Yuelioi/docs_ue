# MakeSubmixEffectDynamicProcessor-

Adds a node that create a 'SubmixEffectDynamicProcessorFilterSettings' from its members

## 图示

![]($-20221218-15024243.png)

## Inputs

Enabled: Boolean. Enabled:. Whether or not filter is enabled.

Cutoff (Hz): Float (single-precision). Cutoff:. The cutoff frequency of the HPF applied to key signal.

Gain (D B): Float (single-precision). Gain Db:. The gain of the filter shelf applied to the key signal.  

## Outputs

Submix Effect Dynamic Processor Filter Settings: Submix Effect Dynamic Processor Filter Settings Structure.
