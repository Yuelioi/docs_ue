# MakeDisplayClust-_19

Adds a node that create a 'DisplayClusterConfigurationViewport_ColorGradingWhiteBalanceSettings' from its members

## Inputs

Temperature Type: ETemperatureMethod Enum. Temperature Type:. Selects the type of temperature calculation.. White Balance uses the Temperature value to control the virtual camera's White Balance. This is the default selection.. Color Temperature uses the Temperature value to adjust the color temperature of the scene, which is the inverse of the White Balance operation..

Temp: Float (single-precision). White Temp:. White temperature.

Tint: Float (single-precision). White Tint:. White tint.  

## Outputs

Display Cluster Configuration Viewport Color Grading White Balance Settings: Display Cluster Configuration Viewport Color Grading White Balance Settings Structure.
