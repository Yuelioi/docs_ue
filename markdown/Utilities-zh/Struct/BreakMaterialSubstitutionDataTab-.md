# BreakMaterialSubstitutionDataTab-

Adds a node that breaks a 'MaterialSubstitutionDataTable' into its member fields

## 图示

![]($-20221218-14394705.png)

## Inputs

Material Substitution Data Table: Material Substitution Data Table Structure (by ref).  

## Outputs

Search String: String. Search String:. Name of the material(s) to search for. Wildcard is supported.

String Match: EEditorScriptingStringMatchType Enum. String Match:. Type of matching to perform with SearchString string.

Material Replacement: Material Interface Object Reference. Material Replacement:. Material to use for the substitution.
