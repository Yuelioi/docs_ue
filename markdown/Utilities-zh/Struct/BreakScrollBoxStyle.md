# BreakScrollBoxStyle

添加一个节点，将'ScrollBoxStyle'分解到其成员字段中。

## 图示

![]($-20221218-14422468.png)

## Inputs

卷轴箱风格。卷轴箱风格结构（按参考）。  

## Outputs

棒材厚度。Float（单精度）。条形厚度。

顶部的阴影刷。石板刷的结构。顶部阴影画笔：。用于绘制卷轴箱顶部阴影的画笔。

底部阴影刷。石板刷的结构。底部阴影画笔：。用于绘制卷轴箱的底部阴影的画笔。

左边的阴影画笔。石板刷的结构。左影刷：。用于绘制滚动框的左影的画笔。

右边的阴影刷。石板画笔结构。右影画笔：。用于绘制滚动框的右影的画笔。
