## Inputs

Display Cluster Configuration Viewport All Nodes Color Grading（显示集群配置视口所有节点颜色分级）。显示群集配置视口所有节点颜色分级结构（通过参考）。

启用所有节点的颜色分级。布尔值。Enable Inner Frustum All Nodes Color Grading：. 启用所有节点的内壳上的调色设置，并将它们添加到 nDisplay 的调色栈中。

Include Entire Cluster Color Grading（包括整个集群调色）。布尔值。启用整个 Cluster Color Grading：. 可选择将根演员上指定的整个群组调色设置包括在nDisplay的调色堆栈中，用于内部地壳。

Color Grading（色彩分级）。Display Cluster Configuration Viewport Color Grading Rendering Settings Structure. 颜色分级设置：. 色彩调控。
