# MakeAbcConversionSettings

添加一个节点，从其成员中创建一个'AbcConversionSettings'。

## Inputs

预设。EAbcConversionPreset Enum.预设：。目前应该应用的预设。

翻转U：布尔值。翻转U：。标记是否翻转纹理坐标中的U通道。

翻转V：布尔值。翻转V：。标记是否翻转纹理坐标中的V通道。

规模。矢量。规模：。应该应用的刻度值。

旋转。矢量。旋转：。应该应用的欧拉角的旋转。  

## Outputs

Abc转换设置。Abc转换设置结构。
