# MakeDatasmithReimportOptions

Adds a node that create a 'DatasmithReimportOptions' from its members

## 图示

![]($-20221218-14500841.png)

## Inputs

Datasmith Scene Actors: Boolean. Update Actors:. Specifies whether or not to update Datasmith Scene Actors in the current Level.

Re- Spawn Deleted Actors: Boolean. Respawn Deleted Actors:. Specifies whether or not to add back Actors you've deleted from the current Level.  

## Outputs

Datasmith Reimport Options: Datasmith Reimport Options Structure.
