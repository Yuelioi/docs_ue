# MakeColorGradingSettings

添加一个节点，从其成员中创建一个'ColorGradingSettings'。

## 图示

![]($-20221218-14484567.png)

## Inputs

全局：颜色等级每范围设置结构。全局。

阴影。颜色等级每范围设置结构。阴影。

中调。每个范围内的颜色等级设置结构。中调。

亮点。颜色等级每范围设置结构。亮点。

阴影最大值：浮点数（单精度）。阴影最大值。

突出显示 Min:Float（单精度）。亮点 Min.

亮点最大：浮点数（单精度）。亮点 最大。  

## Outputs

颜色分级设置。颜色分级设置结构。
