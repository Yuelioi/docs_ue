# BreakCanvasUVTri

Adds a node that breaks a 'CanvasUVTri' into its member fields

## 图示

![]($-20221218-14321579.png)

## Inputs

Canvas UVTri: Canvas UVTri Structure (by ref).  

## Outputs

V0 Pos: Vector 2D Structure. V0 Pos:. Position of first vertex.

V0 UV: Vector 2D Structure. V0 UV:. UV of first vertex.

V0 Color: Linear Color Structure. V0 Color:. Color of first vertex.

V1 Pos: Vector 2D Structure. V1 Pos:. Position of second vertex.

V1 UV: Vector 2D Structure. V1 UV:. UV of second vertex.

V1 Color: Linear Color Structure. V1 Color:. Color of second vertex.

V2 Pos: Vector 2D Structure. V2 Pos:. Position of third vertex.

V2 UV: Vector 2D Structure. V2 UV:. UV of third vertex.

V2 Color: Linear Color Structure. V2 Color:. Color of third vertex.
