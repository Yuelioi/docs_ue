# BreakCanvasUVTri

添加一个节点，将'CanvasUVTri'分解成其成员字段

## 图示

![]($-20221218-14321579.png)

## Inputs

Canvas UVTri: Canvas UVTri结构（通过引用）。 

## Outputs

V0 Pos: Vector 2D结构。V0 Pos:. 第一个顶点的位置。

V0 UV: 矢量二维结构。V0 UV:. 第一个顶点的UV。

V0 Color：线性颜色结构。V0 Color:. 第一个顶点的颜色。

V1 Pos: 矢量2D结构。V1 Pos:. 第二个顶点的位置。

V1 UV: 向量2D结构。V1 UV:. 第二个顶点的UV。

V1 Color: 线性颜色结构。V1 Color:. 第二个顶点的颜色。

V2 Pos: 矢量2D结构。V2 Pos:. 第三个顶点的位置。

V2 UV：向量二维结构。V2 UV:. 第三个顶点的UV。

V2 Color：线性颜色结构。V2 Color：。第三个顶点的颜色。
