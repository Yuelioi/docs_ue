# MakeInterpCurveLinearColor

Adds a node that create a 'InterpCurveLinearColor' from its members

## Inputs

Points: Array of Interp Curve Point Linear Color Structures. Points:. Holds the collection of interpolation points..

Is Looped: Boolean. Is Looped:. Specify whether the curve is looped or not.

Loop Key Offset: Float (single-precision). Loop Key Offset:. Specify the offset from the last point's input key corresponding to the loop point.  

## Outputs

Interp Curve Linear Color: Interp Curve Linear Color Structure.
