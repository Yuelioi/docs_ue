# BreakAbcStaticMeshSettings

添加一个打破'AbcStaticMeshSettings'的节点到其成员字段中。

## Inputs

Abc Static Mesh Settings。Abc静态网格设置结构（通过引用）。 

## Outputs

合并网格。布尔值。Merge Meshes:（合并网格）。是否在导入时合并静态网格（记住这可能会导致UV集重叠的问题）。

Propagate Matrix Transformations（传播矩阵变换）。Boolean（布尔）。Propagate Matrix Transformations（传播矩阵变换）：。如果适用，这将把矩阵变换应用到网格上。

Generate Lightmap UVs: 布尔值。Generate Lightmap UVs（生成光照图UV）：。是否应该生成光照图UV的标志。
