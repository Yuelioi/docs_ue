# BreakAudioSliderStyle

添加一个节点，将'AudioSliderStyle'分解成其成员字段

## 图示

![]($-20221218-14313436.png)

## Inputs

音频滑块样式。音频滑块风格结构（通过引用）。 

## Outputs

滑块风格。Slider Style: 滑块风格结构。Slider Style:.滑块风格。用于底层SSlider的风格。

Text Box Style: 音频文本框风格结构。文本框风格：。用于音频文本框小部件的样式。

Widget Background Image: Slate Brush结构。小工具背景图像：。小工具背景的图像。

Slider Background Color: Slate Color Structure. 滑块背景颜色:. 用于绘制滑块背景的颜色。

滑块背景尺寸。矢量2D结构。Slider Background Size:.滑块背景尺寸。滑块背景的大小（滑块默认为垂直）。

Label Padding（标签填充）。Float（单精度）。Label Padding:.标签填充。标签和滑块之间的填充物的大小。

Slider Bar Color: 石板颜色结构。Slider Bar Color:.滑块条的颜色。用于绘制滑块条的颜色。

Slider Thumb Color: 石板颜色结构。滑块拇指的颜色：。用于绘制滑块拇指（手柄）的颜色。

小工具背景色：石板色结构。小工具背景色：。用来绘制小工具背景的颜色。
