# MakeCanvasUVTri

添加一个节点，从其成员中创建一个 "CanvasUVTri"。

## 图示

![]($-20221218-14482017.png)

## Inputs

V0位置。矢量2D结构。V0 Pos:.第一个顶点的位置。

V0 UV: 矢量2D结构。V0 UV:.第一个顶点的UV。

V0色：线性颜色结构。V0颜色：。第一个顶点的颜色。

V1位置。矢量2D结构。V1 Pos:.第二个顶点的位置。

V1 UV: 矢量2D结构。V1 UV:.第二个顶点的UV。

V1颜色：线性颜色结构。V1颜色：。第二个顶点的颜色。

V2位置。矢量2D结构。V2 Pos:。第三个顶点的位置。

V2 UV: 矢量2D结构。V2 UV:。第三个顶点的UV。

V2颜色：线性颜色结构。V2颜色：。第三个顶点的颜色。  

## Outputs

Canvas UVTri: Canvas UVTri结构。
