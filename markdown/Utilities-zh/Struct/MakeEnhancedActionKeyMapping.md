# MakeEnhancedActionKeyMapping

添加一个节点，从其成员中创建一个'EnhancedActionKeyMapping'。

![]($-20221218-14522798.png)

## Inputs

触发器。输入触发器对象参考数组。触发器：。触发器限定词。如果存在任何触发器限定词，映射将不会触发，除非：.如果这个列表中有任何显式触发器，必须至少满足其中一个。此列表中的所有隐式触发器都必须满足。

修改器。输入修改器对象参考数组。Modifiers:.应用于原始键值的修改器。这些是按数组顺序应用的。

## Outputs

增强的动作键映射。增强的动作键映射结构。
