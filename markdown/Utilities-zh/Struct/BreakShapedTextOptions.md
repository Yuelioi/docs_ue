# BreakShapedTextOptions

添加一个节点，将'ShapedTextOptions'分解成其成员字段

## 图示

![]($-20221218-14423256.png)

## Inputs

形状的文本选项。形状文本选项结构（按参考）。  

## Outputs

文本整形方法。ETextShapingMethod Enum.文本整形方法：。这个小组件中的文本应该使用哪种文本塑造方法？(不设置则使用GetDefaultTextShapingMethod返回的默认值）。

文本流向。ETextFlowDirection Enum.文本流向：。这个小组件内的文本应该使用哪个文本流向？(不设置，使用GetDefaultTextFlowDirection返回的默认值)。
