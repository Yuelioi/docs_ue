# MakeLiveLinkLightBlueprintData

Adds a node that create a 'LiveLinkLightBlueprintData' from its members

## 图示

![]($-20221218-14565720.png)

## Inputs

Static Data: Live Link Light Static Data Structure. Static Data:. Static data that should not change every frame.

Frame Data: Live Link Light Frame Data Structure. Frame Data:. Dynamic data that can change every frame.  

## Outputs

Live Link Light Blueprint Data: Live Link Light Blueprint Data Structure.
