# MakeMovieSceneCameraShakeSourceT-

Adds a node that create a 'MovieSceneCameraShakeSourceTrigger' from its members

## Inputs

Shake Class: Camera Shake Base Class Reference. Shake Class:. Class of the camera shake to play.

Play Scale: Float (single-precision). Play Scale:. Scalar that affects shake intensity.

Play Space: ECameraShakePlaySpace Enum. Play Space.

User Defined Play Space: Rotator. User Defined Play Space.  

## Outputs

Movie Scene Camera Shake Source Trigger: Movie Scene Camera Shake Source Trigger Structure.
