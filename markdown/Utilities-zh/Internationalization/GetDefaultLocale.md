# GetDefaultLocale

获取该平台使用的默认语言（用于国际化）。注意：这应该以IETF语言标签的形式返回。

  * 一个两个字母的ISO 639-1语言代码（例如，"zh"）。

  * 一个可选的四个字母的ISO 15924脚本代码（例如，"Hans"）。

  * 一个可选的两个字母的ISO 3166-1国家代码（例如，"CN"）。





目标是Kismet系统库
