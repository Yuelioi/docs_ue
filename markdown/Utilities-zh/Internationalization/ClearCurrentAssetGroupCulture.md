# ClearCurrentAssetGroupCulture

Clear the given asset group category culture.

Target is Kismet Internationalization Library

## 图示

![]($-20221218-14260748.png)

## Inputs

In: Exec.

Asset Group: Name. The asset group to clear the culture for..

Save to Config: Boolean. If true, save the new setting to the users' "GameUserSettings" config so that it persists after a reload..  

## Outputs

Out: Exec.
