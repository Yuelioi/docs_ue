# StateMachineGetLocalWeight

Returns the weight of a state, relative to its state machine (specified in the provided FCachedAnimStateData)

Target is Cached Anim Data Library

## Inputs

In Anim Instance: Anim Instance Object Reference.

Cached Anim State Data: Cached Anim State Data Structure (by ref).  

## Outputs

Return Value: Float (single-precision). Returns the weight of a state, relative to its state machine (specified in the provided FCachedAnimStateData).
