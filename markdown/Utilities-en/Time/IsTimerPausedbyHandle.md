# IsTimerPausedbyHandle

Returns true if a timer exists and is paused for the given handle, false otherwise.

Target is Kismet System Library

## 图示

![]($-20221218-15245250.png)

## Inputs

Handle: Timer Handle Structure. The handle of the timer to check whether it is paused..  

## Outputs

Return Value: Boolean. True if the timer exists and is paused..
