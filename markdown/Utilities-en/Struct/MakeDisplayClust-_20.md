# MakeDisplayClust-_20

Adds a node that create a 'DisplayClusterConfigurationViewport_CustomPostprocessSettings' from its members

## 图示

![]($-20221218-14503691.png)

## Inputs

Is Enabled: Boolean. Is Enabled:. Enable custom postprocess.

Is One Frame: Boolean. Is One Frame:. Apply postprocess for one frame.

Post Process Settings: Post Process Settings Structure. Post Process Settings:. Custom postprocess settings.

Blend Weight: Float (single-precision). Blend Weight:. Override blend weight.  

## Outputs

Display Cluster Configuration Viewport Custom Postprocess Settings: Display Cluster Configuration Viewport Custom Postprocess Settings Structure.
