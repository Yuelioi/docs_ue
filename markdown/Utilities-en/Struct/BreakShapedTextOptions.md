# BreakShapedTextOptions

Adds a node that breaks a 'ShapedTextOptions' into its member fields

## 图示

![]($-20221218-14423256.png)

## Inputs

Shaped Text Options: Shaped Text Options Structure (by ref).  

## Outputs

Text Shaping Method: ETextShapingMethod Enum. Text Shaping Method:. Which text shaping method should the text within this widget use? (unset to use the default returned by GetDefaultTextShapingMethod).

Text Flow Direction: ETextFlowDirection Enum. Text Flow Direction:. Which text flow direction should the text within this widget use? (unset to use the default returned by GetDefaultTextFlowDirection).
