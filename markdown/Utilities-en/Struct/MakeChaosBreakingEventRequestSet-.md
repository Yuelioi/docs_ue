# MakeChaosBreakingEventRequestSet-

Adds a node that create a 'ChaosBreakingEventRequestSettings' from its members

## 图示

![]($-20221218-14482787.png)

## Inputs

Max Number Of Results: Integer. Max Number Of Results:. The maximum number of results to return..

Min Radius: Float (single-precision). Min Radius:. The minimum breaking radius threshold for the results..

Min Speed: Float (single-precision). Min Speed:. The minimum speed threshold for the results..

Min Mass: Float (single-precision). Min Mass:. The minimum mass threshold for the results..

Max Distance: Float (single-precision). Max Distance:. The maximum distance threshold for the results..

Sort Method: EChaosBreakingSortMethod Enum. Sort Method:. The method used to sort the breaking events..  

## Outputs

Chaos Breaking Event Request Settings: Chaos Breaking Event Request Settings Structure.
