# MakeInterpCurveVector

Adds a node that create a 'InterpCurveVector' from its members

## Inputs

Points: Array of Interp Curve Point Vector Structures. Points:. Holds the collection of interpolation points..

Is Looped: Boolean. Is Looped:. Specify whether the curve is looped or not.

Loop Key Offset: Float (single-precision). Loop Key Offset:. Specify the offset from the last point's input key corresponding to the loop point.  

## Outputs

Interp Curve Vector: Interp Curve Vector Structure.
