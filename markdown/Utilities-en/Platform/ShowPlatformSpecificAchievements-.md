# ShowPlatformSpecificAchievements-

Displays the built-in achievements GUI (iOS and Android only; this function may be renamed or moved in a future release)

Target is Kismet System Library

## 图示

![]($-20221218-14285708.png)

## Inputs

In: Exec.

Specific Player: Player Controller Object Reference. Specific player's achievements to show. May not be supported on all platforms. If null, defaults to the player with ControllerId 0.  

## Outputs

Out: Exec.
