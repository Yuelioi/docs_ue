# IsValidIndex

Target is Kismet Array Library

## Inputs

Target Array: Array of Wildcards. Array to use for the IsValidIndex test *.

Index to Test: Integer. The Index, that we want to test for being valid *.  

## Outputs

Return Value: Boolean. True if the Index is Valid, i.e. greater than or equal to zero, and less than the number of elements in TargetArray..
