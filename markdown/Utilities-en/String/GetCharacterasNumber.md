# GetCharacterasNumber

Gets a single character from the string (as an integer)

Target is Kismet String Library

## 图示

![]($-20221218-14293356.png)

## Inputs

Source String: String. The string to convert.

Index: Integer. Location of the character whose value is required.  

## Outputs

Return Value: Integer. The integer value of the character or 0 if index is out of range.
