# ParseIntoArray

Gets an array of strings from a source string divided up by a separator and empty strings can optionally be culled.

Target is Kismet String Library

## 图示

![]($-20221218-14294661.png)

## Inputs

Source String: String. The string to chop up.

Delimiter: String. The string to delimit on.

Cull Empty Strings: Boolean. = true - Cull (true) empty strings or add them to the array (false).  

## Outputs

Return Value: Array of Strings. The array of string that have been separated.
