# AsTime_fromUTC

Converts a passed in date & time to a text, formatted as a time using the given timezone (default is the local timezone). This will convert the given date & time from UTC to the given timezone (taking into account DST).

Target is Kismet Text Library

## Inputs

In Date Time: Date Time Structure (by ref).

In Time Zone: String.  

## Outputs

Return Value: Text. Converts a passed in date & time to a text, formatted as a time using the given timezone (default is the local timezone). This will convert the given date & time from UTC to the given timezone (taking into account DST)..
