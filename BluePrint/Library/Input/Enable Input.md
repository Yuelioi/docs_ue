## 说明

启用输入。从玩家控制器所处理输入的堆栈中移除此 actor。

## 参数

### 输入

Target : Actor 引用 

Play Control : Player Controller 玩家控制器对象引用需要接收此玩家控制器的输入事件。

如为空，此 actor 将停止从所有玩家控制器接收输入。



## 示例

禁用之后，我的 input（比如按键）无法使用了。过了 3 秒重新打开，E 键又可以使用了
