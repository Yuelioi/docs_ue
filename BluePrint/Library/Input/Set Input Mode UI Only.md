## 说明

设置一个只允许 UI 响应用户输入的输入模式。

目标是控件蓝图库


## 参数



### 输入

Play Control : Player Controller 玩家控制器对象。 

In Widget to Focus：控件对象引用

In Mouse Lock Mode：EMouseLockMode 枚举值
