# AddEnumNode

Adds an enum node to the graph. Enum nodes can be used to represent constant enum values within the graph

Target is Rig VMController

## Inputs

In: Exec.

Target: Rig VMController Object Reference.

In CPPType Object Path: Name (by ref).

In Position: Vector 2D Structure (by ref).

In Node Name: String.

Setup Undo Redo: Boolean.

Print Python Command: Boolean.  

## Outputs

Out: Exec.

Return Value: Rig VMEnum Node Object Reference. Adds an enum node to the graph. Enum nodes can be used to represent constant enum values within the graph.
