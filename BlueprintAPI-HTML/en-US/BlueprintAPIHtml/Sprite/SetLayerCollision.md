# SetLayerCollision

Sets the collision thickness for a specific layer. Note: This will only work on components that own their own tile map (OwnsTileMap returns true), you cannot modify standalone tile map assets

Target is Paper Tile Map Component

## Inputs

In: Exec.

Target: Paper Tile Map Component Object Reference.

Layer: Integer.

Has Collision: Boolean.

Override Thickness: Boolean.

Custom Thickness: Float (single-precision).

Override Offset: Boolean.

Custom Offset: Float (single-precision).

Rebuild Collision: Boolean.  

## Outputs

Out: Exec.
