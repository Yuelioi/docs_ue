# SetPhysicsAsset

Override the Physics Asset of the mesh. It uses SkeletalMesh.PhysicsAsset, but if you'd like to override use this function

Target is Skinned Mesh Component

## Inputs

In: Exec.

Force Re Init: Boolean. Force reinitialize.

Out: Exec.
