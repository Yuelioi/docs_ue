# CreateSession

Creates a session with the default online subsystem

Target is Create Session Callback Proxy

## 图示

![]($-20221218-20162931.png)

## Inputs

In: Exec.

Player Controller: Player Controller Object Reference.

Public Connections: Integer.

Use LAN: Boolean.  

## Outputs

Out: Exec.

On Success: Exec. On Success:. Called when the session was created successfully.

On Failure: Exec. On Failure:. Called when there was an error creating the session.
